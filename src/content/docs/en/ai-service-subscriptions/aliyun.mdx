## Positioning

Alibaba Cloud DashScope is one of the providers currently recommended by the docs preset set for Claude Code. A practical choice for teams already operating in the Alibaba Cloud ecosystem and wanting unified console management.

## Good fit for

- You already use Alibaba Cloud accounts or resources
- You want subscription and key management to stay in one cloud console
- You prefer a fixed mapped model for fast onboarding

## Key things to notice

- The current preset metadata marks Alibaba Cloud DashScope as recommended.
- It maps all Claude tiers to `glm-4.7` for a straightforward setup path.
- It is especially convenient if your team already works inside Alibaba Cloud workflows.

## Suggested next steps

1. Confirm which subscription tier or API activation path you want to use.
2. Return to the Claude Code setup guide and place the base URL plus token into `settings.json` or environment variables.
3. Validate the connection locally after setup.
