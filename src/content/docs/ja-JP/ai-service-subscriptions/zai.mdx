## サービスの位置づけ

智譜AIは現在のドキュメントpresetで推奨されているClaude Code購読エントリの一つです。GLM Codingプランとマルチモデルマッピングを備え、Claude Codeを日常の主力ツールとして使う開発者に適しています。

## 対象ユーザー

- プログラミングシーンに特化したプランを希望する方
- Claudeの階層別モデルマッピングを網羅的に必要とする方
- 一つのサービスでClaude Code、Clineなど複数のツールをカバーしたい方

## まず注目すべきポイント

- 現在のpresetで智譜AIが推奨サービスプロバイダーとしてマークされています。
- Anthropic互換インターフェースを提供しており、Claude Codeへの直接接続が容易です。
- presetには`glm-4.7`、`glm-5`、`glm-4.5-air`などのデフォルトモデルマッピングが含まれています。

## 主な接続情報

| 項目 | 内容 |
|------|------|
| 購読エントリ | [購読ページへ](https://www.bigmodel.cn/claude-code?ic=14BY54APZA) |
| 公式ドキュメント / コンソール | [公式ページを見る](https://open.bigmodel.cn/dev/api) |
| Anthropic互換Base URL | `https://open.bigmodel.cn/api/anthropic` |
| 推奨認証環境変数 | `ANTHROPIC_AUTH_TOKEN` |

## デフォルトモデルマッピング

| Claude階層 | デフォルトモデル |
|-------------|----------|
| Sonnet | `glm-4.7` |
| Opus | `glm-5` |
| Haiku | `glm-4.5-air` |

:::tip
プラン、割引、特典は随時変更される可能性があります。最新価格を確認したい場合は、サービスプロバイダーの購読ページを直接ご覧ください。
:::

## 設定のアドバイス

1. まず利用したいプランまたはAPI Keyの開通方法を確認します。
2. 次に統合AI Agent CLIページのClaude Code項目を開き、Anthropic公式のインストールおよびリファレンスドキュメントに進みます。
3. 完了後、ローカルで`claude --version`、設定ファイルの確認、または実際のリクエストを実行して接続が成功したかを検証できます。
