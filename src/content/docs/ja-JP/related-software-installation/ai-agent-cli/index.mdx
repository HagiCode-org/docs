:::tip[このページの使い方]
まず現在のサポート状態とセットアップ モードの表を確認してから、ベンダーが管理するインストール、認証、ACP、権限、およびトラブルシューティングに関するドキュメントに進みます。サポート範囲とセットアップ モードは別の問題です。
:::

## 州の用語集

- **現在サポートされている** は、CLI が現在アクティブなHagiCode サポート範囲に属していることを意味します。
- **ワンクリック インストール** は、HagiCode がガイド付きインストール エントリ ポイントを提供することを意味します。一部の CLI では、インストール後もベンダー ログインまたは認証手順が必要です。
- **手動インストール/認証** は、CLI が現在サポートされていることを意味しますが、HagiCode が CLI を検出して使用できるようにするには、ローカル インストールまたは認証フローを完了する必要があります。
- **レガシー互換性のみ** は、次のような廃止された値を意味します。 `IFlowCli` 履歴データの互換性のためにまだ存在していますが、現在のアクティブなサポート範囲には含まれていません。

## 推奨される次のステップ

- 続けて [Node.jsをインストールする](/ja-JP/related-software-installation/nodejs/installation) そして [OpenSpec をインストールする](/ja-JP/related-software-installation/openspec/setup-openspec) まだ前提条件が必要な場合。
- 複数の OpenAI 互換ツールに 1 つのローカル ゲートウェイが必要な場合は、次の手順に進みます。 [OmniRoute のインストールと使用法](/ja-JP/related-software-installation/omniroute/).
- に戻る [インストールガイド](/ja-JP/installation) より広範なセットアップ パスに対応します。
- 歴史的なブログ投稿からここにたどり着いた場合は、このページとインストールの概要を最新の信頼できる情報源として扱ってください。
