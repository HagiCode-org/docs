HagiCode は複数のインストール パスをサポートしています。シナリオに合った展開方法から始めて、必要な場合にのみサポート ソフトウェアを追加します。

セットアップに最も関連する現在アクティブな Agent CLI サポート範囲は、Claude Code、Codex、GitHub Copilot、OpenCode、Hermes、QoderCLI、Kiro、Kimi、Gemini、DeepAgents、および Codebuddy です。現在、Hermes、Gemini、および DeepAgents は手動インストールまたは認証フローを使用しています。他の CLI は、ワンクリック インストールのエントリ ポイントまたはガイド付きインストール フローを公開します。
