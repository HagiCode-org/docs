## このセクションの内容

FAQ セクションは次の場所です。

- 起動時と実行時の注意事項
- プラットフォーム、流通チャネル、パッケージ化、またはバンドルされたランタイムに関連する互換性に関する注意事項
- 安定したスタンドアロンの説明ページが必要なトラブルシューティングのトピック

現在 Steam for Linux の起動動作のトラブルシューティングを行っている場合は、 [Steam Linux スタートアップノート](/ja-JP/faq/steam-linux-startup)。どのノード デスクトップが使用されているか、または診断で言及されている理由を確認する必要がある場合 `portable-fixed/toolchain`、開く [デスクトップノード環境](/ja-JP/faq/desktop-node-environment).
