HagiCode는 다양한 설치 경로를 지원합니다. 시나리오에 맞는 배포 방법으로 시작한 다음 필요할 때만 지원 소프트웨어를 추가하세요.

설정과 가장 관련이 있는 현재 활성 에이전트 CLI 지원 범위는 Claude Code, Codex, GitHub Copilot, OpenCode, Hermes, QoderCLI, Kiro, Kimi, Gemini, DeepAgents 및 Codebuddy입니다. Hermes, Gemini 및 DeepAgents는 현재 수동 설치 또는 인증 흐름을 사용합니다. 다른 CLI는 원클릭 설치 진입점이나 안내 설치 흐름을 제공합니다.
