이 섹션에서는 `repos/docs/public/presets/claude-code/providers/*.json`에 현재 추천으로 표시된 AI 서비스 구독 사이트를 정리합니다. 먼저 여기서 구독 진입점을 확인한 후, 설정 문서로 이동하여 연동을 완료하세요.

**2026-03-09** 기준으로, 이 섹션에서는 다음 서비스를 다룹니다: **Zhipu AI, MiniMax, 알리윈 DashScope, 화산엔진 Coding Plan**.

:::tip[권장 읽기 순서]
아직 어느 서비스를 선택할지 결정하지 못했다면 먼저 이 페이지에서 비교하세요. 이미 서비스 제공자를 선택했다면 해당 상세 페이지로 바로 이동한 후, 통합 AI Agent CLI 페이지의 Claude Code 항목을 참고하세요.
:::

## 현재 추천 서비스 비교

| 서비스 | 특징 | 적합 대상 | 상세 페이지 |
|------|------|--------|--------|
| Zhipu AI | 코딩 전용 플랜, 다단계 모델 매핑 지원 | Claude Code를 일상 주력 도구로 사용하려는 개발자 | [상세 보기](./zai/) |
| MiniMax | 통합 모델 매핑, 낮은 설정 이해 비용 | 연동 원활성과 유지보수 비용을 우선 고려하는 팀 | [상세 보기](./minimax/) |
| 알리윈 DashScope | 높은 클라우드 플랫폼 통합, 콘솔 통합 관리 | 알리윈 생태계에서 이미 작업 중인 팀 | [상세 보기](./aliyun/) |
| 화산엔진 Coding Plan | 강력한 생태계 확장성, 풍부한 프로모션 정보 | 화산엔진 생태계 및 다중 모델 플랫폼 기능에 관심 있는 개발자 | [상세 보기](./volcengine/) |

:::note
`Anthropic Official`은 현재도 사용 가능하지만, 현재 preset 메타데이터에서 추천 항목으로 표시되지 않으므로 이 섹션의 첫 번째 독립 페이지 범위에는 포함되지 않습니다.
:::

## 빠른 선택 가이드

- **코딩 전용 플랜과 더 세밀한 기본 모델 매핑을 원한다면**: [Zhipu AI](./zai/)를 우선 확인하세요
- **최대한 빨리 워크플로우를 구축하고 설정 이해 비용을 줄이고 싶다면**: [MiniMax](./minimax/)를 우선 확인하세요
- **알리윈 리소스를 이미 깊이 사용 중이라면**: [알리윈 DashScope](./aliyun/)를 우선 확인하세요
- **화산엔진 생태계나 다중 모델 플랫폼 기능을 결합하고 싶다면**: [화산엔진 Coding Plan](./volcengine/)를 우선 확인하세요
