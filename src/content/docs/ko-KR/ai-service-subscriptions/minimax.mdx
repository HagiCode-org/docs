## 서비스 포지셔닝

MiniMax는 현재 문서 preset에서 추천하는 Claude Code 구독 진입점 중 하나입니다. 통합 모델, 간단한 설정으로 연동 원활성과 유지보수 비용을 우선 고려하는 팀에 적합합니다.

## 적합 대상

- 세 단계 모델 모두 동일한 기본 매핑을 사용하고 싶은 사용자
- 설정의 단순성과 안정적인 연동을 더 중요하게 생각하는 사용자
- Claude Code 워크플로우를 빠르게 구축해보고 싶은 사용자

## 먼저 확인해야 할 사항

- 현재 preset에서 MiniMax를 추천 서비스 제공자로 표시합니다.
- `MiniMax-M2.5`를 기본 모델로 통일 사용하여 설정 이해 비용이 낮습니다.
- 워크플로우를 먼저 구축하고, 이후 모델 전략을 세분화하려는 팀에 적합합니다.

## 핵심 연동 정보

| 항목 | 내용 |
|------|------|
| 구독 진입점 | [구독 페이지로 이동](https://platform.minimaxi.com/subscribe/coding-plan?code=8wNck4etDM&source=link) |
| 공식 문서 / 콘솔 | [공식 페이지 보기](https://www.minimaxi.com) |
| Anthropic 호환 Base URL | `https://api.minimaxi.com/anthropic` |
| 권장 인증 환경 변수 | `ANTHROPIC_AUTH_TOKEN` |

## 기본 모델 매핑

| Claude 계층 | 기본 모델 |
|-------------|----------|
| Sonnet | `MiniMax-M2.5` |
| Opus | `MiniMax-M2.5` |
| Haiku | `MiniMax-M2.5` |

:::tip
플랜, 할인 및 혜택은 시간에 따라 변동될 수 있습니다. 실시간 가격 정보가 필요하면 서비스 제공자의 구독 페이지를 직접 확인하세요.
:::

## 설정 권장 사항

1. 먼저 사용할 플랜 또는 API Key 개통 방식을 확인합니다.
2. 그다음 통합 AI Agent CLI 페이지의 Claude Code 항목을 열어 Anthropic 공식 설치 및 참고 문서로 이동합니다.
3. 완료 후 로컬에서 `claude --version`, 설정 파일 확인 또는 실제 요청을 실행하여 연동이 성공했는지 검증합니다.
