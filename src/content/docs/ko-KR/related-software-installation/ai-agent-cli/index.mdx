이 페이지는 통합 로컬 AI Agent CLI 설치 항목입니다. `repos/docs`. HagiCode는 현재 백엔드에 이어 11개의 활성 에이전트 CLI를 지원합니다. `PrimaryProfessions` 순서: Claude Code, Codex, GitHub Copilot, OpenCode, Hermes, QoderCLI, Kiro, Kimi, Gemini, DeepAgents 및 Codebuddy.

:::tip[페이지 이용방법]
현재 지원 상태 및 설정 모드에 대한 표를 먼저 확인한 다음 공급업체에서 유지 관리하는 설치, 인증, ACP, 권한 및 문제 해결 문서를 계속 진행하세요. 지원 범위와 설정 모드는 별도의 문제입니다.
:::

## 주 용어집

- **현재 지원됨**은 CLI가 현재 활성화된 HagiCode 지원 범위에 속한다는 것을 의미합니다.
- **원클릭 설치**는 HagiCode가 안내식 설치 진입점을 제공한다는 의미입니다. 일부 CLI에는 설치 후에도 여전히 공급업체 로그인 또는 인증 단계가 필요합니다.
- **수동 설치/인증**은 현재 CLI가 지원되지만 HagiCode가 이를 감지하고 사용하기 전에 로컬 설치 또는 인증 흐름을 완료해야 함을 의미합니다.
- **기존 호환성만**은 다음과 같은 폐기된 값을 의미합니다. `IFlowCli` 기록 데이터 호환성을 위해 여전히 존재하지만 현재 활성 지원 범위에 속하지 않습니다.
