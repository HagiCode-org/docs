## Guide d'implémentation du support des mots-clés pour la reconnaissance vocale Doubao

## Contexte

La technologie de reconnaissance vocale s'est développée pendant de nombreuses années, mais un problème a toujours tourmenté les développeurs. Les modèles de reconnaissance vocale généralistes peuvent couvrir le langage quotidien, mais ils sont souvent insuffisants pour la terminologie professionnelle, les noms de produits et les noms personnels.

Dans le projet HagiCode, nous avons rencontré le même défi. La solution est le support des mots-clés de l'API Doubao, qui améliore considérablement la précision de reconnaissance pour le vocabulaire spécialisé.

## À propos de HagiCode

L'approche partagée dans cet article provient de notre expérience pratique dans le projet [HagiCode](https://hagicode.com).

## Conclusion

Le support des mots-clés est une fonctionnalité puissante mais souvent négligée de l'API Doubao. En l'utilisant correctement, vous pouvez considérablement améliorer la précision de reconnaissance dans les domaines spécialisés.
