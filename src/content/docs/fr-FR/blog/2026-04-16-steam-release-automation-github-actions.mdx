## Comment automatiser les publications Steam avec GitHub Actions

> Cet article partage la solution complète que nous avons implémentée pour les publications automatisées sur Steam dans le projet HagiCode Desktop, couvrant le flux d'automatisation de bout en bout depuis le déclenchement du build jusqu'à la mise en ligne sur Steam.

## Contexte

Publier une application sur Steam est un processus manuel fastidieux : build, empaquetage, téléchargement, remplissage des notes de version, et attente de la validation. Pour le projet HagiCode Desktop, nous avons automatisé l'ensemble de ce processus avec GitHub Actions.

## À propos de HagiCode

L'approche partagée dans cet article provient de notre expérience pratique dans le projet [HagiCode](https://hagicode.com).

## Configuration du workflow GitHub Actions

Le workflow de publication Steam se déclenche lors de la création d'une nouvelle release GitHub. Il automatise le build, l'empaquetage, et le téléchargement vers Steam via l'API Steamworks.

## Conclusion

L'automatisation des publications Steam avec GitHub Actions élimine les erreurs humaines et accélère considérablement le processus de mise en ligne.
