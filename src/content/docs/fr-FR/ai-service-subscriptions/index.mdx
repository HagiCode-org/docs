Cette section regroupe les sites d'abonnement aux services IA actuellement marques comme recommandes dans `repos/docs/public/presets/claude-code/providers/*.json`. Determinez d'abord votre point d'abonnement ici, puis rendez-vous dans la documentation de configuration pour finaliser l'integration.

Au **2026-03-09**, cette section couvre les services suivants : **Zhipu AI, MiniMax, Alibaba Cloud DashScope, Volcengine Coding Plan**.

:::tip[Ordre de lecture recommande]
Si vous n'avez pas encore decide quel fournisseur choisir, consultez d'abord cette page pour comparer ; si vous avez deja choisi un fournisseur, rendez-vous directement sur la page de details correspondante, puis accedez a la section Claude Code de la page unifiee des CLI d'agents IA.
:::

## Comparaison des services actuellement recommandes

| Service | Caracteristiques | Public cible | Page de details |
|---------|-----------------|--------------|-----------------|
| Zhipu AI | Forfaits dedies au developpement, support de mappage multi-niveaux de modeles | Developpeurs souhaitant utiliser Claude Code comme outil principal au quotidien | [Voir les details](./zai/) |
| MiniMax | Mappage de modeles unifie, faible cout de comprehension de la configuration | Equipes privilegiant la fluidite d'integration et la facilite de maintenance | [Voir les details](./minimax/) |
| Alibaba Cloud DashScope | Forte integration avec la plateforme cloud, gestion unifiee via la console | Equipes travaillant deja dans l'ecosysteme Alibaba Cloud | [Voir les details](./aliyun/) |
| Volcengine Coding Plan | Forte extensibilite de l'ecosysteme, informations riches sur les promotions | Developpeurs interesses par l'ecosysteme Volcengine et les capacites multi-modeles | [Voir les details](./volcengine/) |

:::note
`Anthropic Official` est toujours utilisable, mais comme il n'est pas marque comme recommande dans les metadonnees du preset actuel, il n'est pas inclus dans la premiere serie de pages independantes de cette section.
:::

## Comment choisir rapidement

- **Vous voulez un forfait dedie au developpement et un mappage de modeles par defaut plus precis** : consultez d'abord [Zhipu AI](./zai/)
- **Vous voulez mettre en place rapidement votre flux de travail et reduire le cout de comprehension de la configuration** : consultez d'abord [MiniMax](./minimax/)
- **Vous utilisez deja intensivement les ressources Alibaba Cloud** : consultez d'abord [Alibaba Cloud DashScope](./aliyun/)
- **Vous souhaitez combiner l'ecosysteme Volcengine ou les capacites de plateformes multi-modeles** : consultez d'abord [Volcengine Coding Plan](./volcengine/)
