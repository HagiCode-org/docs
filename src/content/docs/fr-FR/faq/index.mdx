## Ce que couvre cette section

La section FAQ est l'endroit idéal pour :

- mises en garde concernant le démarrage et l'exécution
- notes de compatibilité liées aux plates-formes, aux canaux de distribution, aux emballages ou aux environnements d'exécution groupés
- sujets de dépannage qui méritent une page d'explication autonome et stable

Si vous dépannez actuellement le comportement de démarrage de Steam pour Linux, ouvrez [Notes de démarrage de Steam Linux](/fr-FR/faq/steam-linux-startup). Si vous devez confirmer quel Node Desktop utilise ou pourquoi les diagnostics mentionnent `portable-fixed/toolchain`, ouvert [Environnement de nœud de bureau](/fr-FR/faq/desktop-node-environment).
