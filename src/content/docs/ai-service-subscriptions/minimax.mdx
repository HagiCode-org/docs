## 服务定位

MiniMax 是当前文档 preset 中推荐的 Claude Code 订阅入口之一。统一模型、配置简单，适合优先考虑接入平滑度和维护成本的团队。

## 适合谁

- 希望三档模型都使用同一套默认映射
- 更关注配置简单和稳定接入
- 想先快速跑通 Claude Code 工作流

## 你可以先关注这些点

- 当前 preset 将 MiniMax 标记为推荐服务商。
- 统一使用 `MiniMax-M2.5` 作为默认模型，配置理解成本较低。
- 适合希望先跑通工作流、后续再细化模型策略的团队。

## 关键接入信息

| 项目 | 内容 |
|------|------|
| 订阅入口 | [前往订阅页面](https://platform.minimaxi.com/subscribe/coding-plan?code=8wNck4etDM&source=link) |
| 官方文档 / 控制台 | [查看官方页面](https://www.minimaxi.com) |
| Anthropic 兼容 Base URL | `https://api.minimaxi.com/anthropic` |
| 推荐认证环境变量 | `ANTHROPIC_AUTH_TOKEN` |

## 默认模型映射

| Claude 层级 | 默认模型 |
|-------------|----------|
| Sonnet | `MiniMax-M2.5` |
| Opus | `MiniMax-M2.5` |
| Haiku | `MiniMax-M2.5` |

:::tip
套餐、优惠和权益会随时间变化；如果你更关注实时价格，请直接查看服务商订阅页面。
:::

## 配置建议

1. 先确认你想使用的套餐或 API Key 开通方式。
2. 再打开统一 AI Agent CLI 页面中的 Claude Code 条目，继续前往 Anthropic 官方安装与参考文档。
3. 完成后可在本地运行 `claude --version`、配置文件检查或一次实际请求来验证接入是否成功。
