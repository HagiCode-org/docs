## 服务定位

智谱 AI 是当前文档 preset 中推荐的 Claude Code 订阅入口之一。GLM Coding 套餐与多模型映射，适合把 Claude Code 作为日常主力工具的开发者。

## 适合谁

- 希望获得专门面向编程场景的套餐
- 需要较完整的 Claude 层级模型映射
- 想通过一个服务覆盖 Claude Code、Cline 等多种工具

## 你可以先关注这些点

- 当前 preset 中将智谱 AI 标记为推荐服务商。
- 提供 Anthropic 兼容接口，便于直接对接 Claude Code。
- 预设中包含 `glm-4.7`、`glm-5` 与 `glm-4.5-air` 等默认模型映射。

## 关键接入信息

| 项目 | 内容 |
|------|------|
| 订阅入口 | [前往订阅页面](https://www.bigmodel.cn/claude-code?ic=14BY54APZA) |
| 官方文档 / 控制台 | [查看官方页面](https://open.bigmodel.cn/dev/api) |
| Anthropic 兼容 Base URL | `https://open.bigmodel.cn/api/anthropic` |
| 推荐认证环境变量 | `ANTHROPIC_AUTH_TOKEN` |

## 默认模型映射

| Claude 层级 | 默认模型 |
|-------------|----------|
| Sonnet | `glm-4.7` |
| Opus | `glm-5` |
| Haiku | `glm-4.5-air` |

:::tip
套餐、优惠和权益会随时间变化；如果你更关注实时价格，请直接查看服务商订阅页面。
:::

## 配置建议

1. 先确认你想使用的套餐或 API Key 开通方式。
2. 再打开统一 AI Agent CLI 页面中的 Claude Code 条目，继续前往 Anthropic 官方安装与参考文档。
3. 完成后可在本地运行 `claude --version`、配置文件检查或一次实际请求来验证接入是否成功。
