## 服务定位

火山引擎 Coding Plan 是当前文档 preset 中推荐的 Claude Code 订阅入口之一。适合关注平台活动、希望在火山引擎生态中使用多模型能力的开发者。

## 适合谁

- 希望在火山引擎生态中继续扩展 AI 工具链
- 关注多模型平台能力与活动入口
- 需要 Anthropic 兼容入口来接入 Claude Code

## 你可以先关注这些点

- 当前 preset 将火山引擎 Coding Plan 标记为推荐服务商。
- 预设说明中提到其支持 Doubao、GLM、DeepSeek、Kimi 等模型生态。
- 活动与价格变化较快，具体权益请以服务商订阅页面实时说明为准。

## 默认模型映射

| Claude 层级 | 默认模型 |
|-------------|----------|
| Sonnet | `glm-4.7` |
| Opus | `glm-4.7` |
| Haiku | `glm-4.7` |

:::tip
套餐、优惠和权益会随时间变化；如果你更关注实时价格，请直接查看服务商订阅页面。
:::

## 配置建议

1. 先确认你想使用的套餐或 API Key 开通方式。
2. 再回到 Claude Code 配置指南，把 Base URL 和 Token 写入 `settings.json` 或环境变量。
3. 完成后可在本地运行 `claude --version`、配置文件检查或一次实际请求来验证接入是否成功。
