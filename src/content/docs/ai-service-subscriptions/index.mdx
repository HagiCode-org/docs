本章节汇总 `repos/docs/public/presets/claude-code/providers/*.json` 中当前标记为推荐的 AI 服务订阅网站，方便你先确定订阅入口，再进入配置文档完成接入。

截至 **2026-03-09**，本章节覆盖以下服务：**智谱 AI、MiniMax、阿里云 DashScope、火山引擎 Coding Plan**。

:::tip[推荐阅读顺序]
如果你还没确定买哪一家，先看本页做对比；如果你已经选好了服务商，直接进入对应详情页，然后再跳转到 Claude Code 配置指南。
:::

## 当前推荐服务对比

| 服务 | 特点 | 适合谁 | 详情页 |
|------|------|--------|--------|
| 智谱 AI | 编程专项套餐，支持多档模型映射 | 希望把 Claude Code 作为日常主力工具的开发者 | [查看详情](./zai/) |
| MiniMax | 统一模型映射，配置理解成本低 | 优先考虑接入平滑度和维护成本的团队 | [查看详情](./minimax/) |
| 阿里云 DashScope | 云平台整合度高，适合控制台统一管理 | 已经在阿里云生态中工作的团队 | [查看详情](./aliyun/) |
| 火山引擎 Coding Plan | 生态延展性强，活动信息丰富 | 关注火山引擎生态和多模型平台能力的开发者 | [查看详情](./volcengine/) |

:::note
`Anthropic Official` 目前仍可使用，但因为它在当前 preset 元数据中没有标记为推荐项，所以暂不纳入本章节的首批独立页面范围。
:::

## 如何快速选择

- **想要编程专项套餐与更细的默认模型映射**：优先看 [智谱 AI](./zai/)
- **想要尽快跑通工作流、减少配置理解成本**：优先看 [MiniMax](./minimax/)
- **已经深度使用阿里云资源**：优先看 [阿里云 DashScope](./aliyun/)
- **希望结合火山引擎生态或多模型平台能力**：优先看 [火山引擎 Coding Plan](./volcengine/)
