本页是 `repos/docs` 中统一的 AI Agent CLI 本地安装入口。HagiCode 当前活跃支持 11 个 Agent CLI，顺序以后端 `PrimaryProfessions` 为准：Claude Code、Codex、GitHub Copilot、OpenCode、Hermes、QoderCLI、Kiro、Kimi、Gemini、DeepAgents、Codebuddy。

:::tip[使用方式]
先看下表确认某个 CLI 的**当前状态**与**安装方式**，再跳到对应官方文档完成安装、登录、ACP、权限与排错。重点是：**是否支持**，与**是否支持一键安装**，是两件事。
:::

## 状态说明

- **当前支持**：属于 HagiCode 当前活跃支持范围，可在系统中作为现行 Agent CLI 使用。
- **一键安装**：HagiCode 提供引导式安装入口；部分 CLI 安装后仍要按厂商流程继续登录或认证。
- **手动安装 / 认证**：该 CLI 已在当前支持范围内，但需要你先完成本地安装、登录或认证，再回到 HagiCode 刷新可用状态。
- **历史兼容保留**：`IFlowCli` 仅为旧数据兼容保留值，不属于当前活跃支持列表，也不应视为现行推荐项。
