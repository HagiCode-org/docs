本页是 `repos/docs` 中唯一的 AI Agent CLI 本地安装入口。HagiCode 当前支持的 Claude Code、Codex、GitHub Copilot、CodeBuddy、OpenCode、Kimi、QoderCLI 和 Kiro 都集中在这里。

:::tip[使用方式]
先从下表选择工具，再跳转到对应官方文档完成安装、登录、ACP、权限和排错。本站只保留统一入口与相关前置依赖链接，不重复维护第三方 CLI 的本地安装步骤。
:::

<h2 id="claude-code">Claude Code</h2>

Anthropic 官方的 Claude Code CLI，适合希望直接使用 Claude Code 工作流的用户。

- 官方安装入口：[Getting Started](https://code.claude.com/docs/en/getting-started)
- 官方参考入口：[Overview](https://code.claude.com/docs/en/overview)

<h2 id="codex">Codex</h2>

OpenAI 官方的 Codex CLI，适合希望使用 OpenAI 本地编码代理工作流的用户。

- 官方安装入口：[OpenAI Codex CLI repository](https://github.com/openai/codex)
- 官方参考入口：[OpenAI Codex docs](https://developers.openai.com/codex)

<h2 id="github-copilot">GitHub Copilot</h2>

GitHub 官方的 Copilot CLI，适合已经在 GitHub Copilot 生态中工作的用户。

- 官方安装入口：[Install Copilot CLI](https://docs.github.com/en/copilot/how-tos/copilot-cli/set-up-copilot-cli/install-copilot-cli)
- 官方参考入口：[Copilot CLI overview](https://docs.github.com/en/copilot/how-tos/copilot-cli/use-copilot-cli-agents/overview)

<h2 id="codebuddy">CodeBuddy</h2>

腾讯 CodeBuddy 官方 CLI，适合需要官方 ACP 文档和安装说明配套阅读的用户。

- 官方安装入口：[CodeBuddy CLI Installation](https://www.codebuddy.ai/docs/cli/installation)
- 官方参考入口：[CodeBuddy ACP](https://www.codebuddy.ai/docs/cli/acp)

<h2 id="opencode">OpenCode</h2>

OpenCode 官方 CLI，适合希望使用本地运行时和独立模型提供商工作流的用户。

- 官方安装入口：[OpenCode CLI docs](https://opencode.ai/docs/cli)
- 官方参考入口：[OpenCode docs](https://opencode.ai/docs/)

<h2 id="kimi">Kimi</h2>

Kimi Code CLI 官方工具，适合希望按 Moonshot 官方流程完成安装和 ACP 配置的用户。

- 官方安装入口：[Kimi Getting Started](https://moonshotai.github.io/kimi-cli/en/guides/getting-started.html)
- 官方参考入口：[kimi acp Reference](https://moonshotai.github.io/kimi-cli/en/reference/kimi-acp.html)

<h2 id="qodercli">QoderCLI</h2>

Qoder 官方 CLI，适合需要 Quick Start 与 ACP 文档配套使用的用户。

- 官方安装入口：[Qoder Quick Start](https://docs.qoder.com/cli/quick-start)
- 官方参考入口：[Qoder ACP](https://docs.qoder.com/cli/acp)

<h2 id="kiro">Kiro</h2>

Kiro 官方 CLI，适合需要先完成安装、认证，再继续 ACP 接入的用户。

- 官方安装入口：[Kiro Installation](https://kiro.dev/docs/cli/installation/)
- 官方参考入口：[Kiro Authentication](https://kiro.dev/docs/cli/authentication/)
- 官方 ACP 文档：[Kiro ACP](https://kiro.dev/docs/cli/acp/)
