## GitHub Copilot CLI 简介

GitHub Copilot CLI 是 GitHub 提供的终端原生 AI 助手，可直接在命令行中与 Copilot 交互并处理开发任务。

> 注意：GitHub Copilot CLI 当前为 Public Preview，能力和命令可能随官方版本更新调整。

## 适用版本与验证日期

- **适用对象**：GitHub Copilot CLI（命令行版本，不是 IDE 插件）
- **最后验证日期**：2026-03-09
- **官方安装文档**：[Installing GitHub Copilot CLI](https://docs.github.com/en/copilot/how-tos/copilot-cli/install-copilot-cli)

## 安装前系统要求

| 检查项 | 最低要求 | 如何验证 |
|---|---|---|
| Copilot 权限 | 账号有 Copilot 订阅或组织分配席位 | 登录 GitHub 后检查 Copilot 可用状态 |
| Windows 环境 | PowerShell 6 或更高（如使用 Windows） | `pwsh --version` |
| Node.js（npm 安装方式） | Node.js 22 或更高 | `node --version` |
| npm（npm 安装方式） | npm 10 或更高 | `npm --version` |
| 网络连接 | 可访问 GitHub 与 Copilot 服务 | 浏览器访问 [github.com](https://github.com) |

:::warning
如果你通过组织/企业使用 Copilot，管理员可以在策略中禁用 Copilot CLI。安装前请先确认组织策略已开启。
:::

## 详细安装步骤

你可以选择一种安装方式即可。

### 方式 A：npm（跨平台，推荐）

```bash
npm install -g @github/copilot
```

安装完成后验证：

```bash
copilot --version
```

### 方式 B：WinGet（Windows）

```powershell
winget install GitHub.Copilot
```

### 方式 C：Homebrew（macOS / Linux）

```bash
brew install copilot-cli
```

### 方式 D：安装脚本（macOS / Linux）

```bash
curl -fsSL https://gh.io/copilot-install | bash
```

## 安装后配置方法

### 1) 首次登录

安装后在终端启动：

```bash
copilot
```

首次使用时，在 CLI 中执行 `/login`，按提示完成 GitHub 登录与授权。

### 2) 可选：使用令牌认证

你也可以使用带 **Copilot Requests** 权限的 fine-grained token，并通过环境变量传入：

- `GH_TOKEN`
- `GITHUB_TOKEN`

### 3) 首次可用性验证

```bash
copilot
```

进入会话后输入一个简单任务，例如“解释当前目录项目结构”，确认 CLI 能正常响应。

:::tip
Copilot CLI 会提示是否信任当前目录。仅在你信任该目录文件时选择继续。
:::

## 常见问题排查

| 问题现象 | 可能原因 | 处理步骤 |
|---|---|---|
| `copilot` 命令不存在 | 全局安装路径未加入 PATH | 重新打开终端；检查 npm 全局 bin 路径是否在 PATH |
| 登录失败或卡住 | 浏览器登录态异常 / 网络受限 | 重新执行 `/login`；检查代理与网络；必要时切换网络 |
| 提示无权限使用 Copilot CLI | 订阅未开通或组织策略禁用 | 检查账号订阅状态；联系组织管理员开启 Copilot CLI 策略 |
| npm 安装报脚本相关错误 | 本机 npm 配置 `ignore-scripts=true` | 使用 `npm_config_ignore_scripts=false npm install -g @github/copilot` |

### 快速恢复流程（推荐）

1. 运行 `copilot --version` 确认二进制可执行。
2. 运行 `copilot` 并重新执行 `/login`。
3. 在可访问 GitHub 的网络下重试。
4. 如仍失败，改用另一种安装方式（如 npm ↔ brew / winget）重装。
