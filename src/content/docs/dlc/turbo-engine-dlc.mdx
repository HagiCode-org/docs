Turbo Engine 是 HagiCode Desktop 的核心增强插件，面向已经进入高频使用阶段的用户。它解决的是一件事：**把 HagiCode 从基础态推向高强度工作态。**

<SteamProductArtwork product={steamProducts['turbo-engine']} />

## 购买方式

Turbo Engine 是 Microsoft Store 中的一次性买断许可证（Store ID: `9NSD809W18Z6`），购买后永久生效。你可以在 HagiCode Desktop 的订阅工作区中直接完成购买和状态刷新。

## 解锁特性

购买 Turbo Engine 后，以下能力会被永久解锁：

| 特性 | 说明 |
| --- | --- |
| **提案并发上限 3 → 32** | 生成、执行、归档中的提案共享此上限。32 并发让你可以稳定并行推进多条任务线。 |
| **扩展文档主题** | 在冒险团管理界面中切换文档主题，实时预览不同主题下的术语与文案表现。 |
| **自定义 Git 提交署名** | 设置默认 `Co-Authored-By` 模式，或在单次 AI Compose Commit 中临时切换署名。 |
| **品牌自定义** | 上传头部 Logo，设置应用标题、浏览器标题和品牌点击跳转链接。 |
| **自定义头像上传** | 为当前 Hero 上传自定义头像，或从共享头像库中复用到新建和批量创建流程。 |
| **DLC 头像资源** | Turbo Engine 专属头像包接入 Hero Avatar Gallery 与批量创建流程，直接参与工作流选择。 |

## 谁需要它

- 经常同时推进多个提案或任务的用户。
- 默认 3 并发已经不够用的重度使用者。
- 需要自定义 Git 署名、文档主题、头像和品牌标识的用户。
