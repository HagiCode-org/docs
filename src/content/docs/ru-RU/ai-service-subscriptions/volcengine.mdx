## Позиционирование сервиса

Volcengine Coding Plan — один из рекомендуемых пунктов подписки для Claude Code в текущем preset документации. Подходит для разработчиков, интересующихся платформенными акциями и желающих использовать мульти-модельные возможности в экосистеме Volcengine.

## Для кого подходит

- Хотят расширить цепочку ИИ-инструментов в экосистеме Volcengine
- Интересуются возможностями мульти-модельных платформ и акциями
- Нужен Anthropic-совместимый API для подключения Claude Code

## На что обратить внимание

- В текущем preset Volcengine Coding Plan отмечен как рекомендуемый поставщик.
- В описании preset упоминается поддержка модельных экосистем Doubao, GLM, DeepSeek, Kimi и других.
- Акции и цены могут часто меняться — подробную информацию о правах смотрите на странице подписки поставщика.

:::tip
Тарифы, скидки и права могут изменяться с течением времени. Если вас интересуют актуальные цены, перейдите непосредственно на страницу подписки поставщика.
:::

## Рекомендации по настройке

1. Сначала определитесь с тарифом или способом получения API Key.
2. Затем откройте раздел Claude Code на странице AI Agent CLI и перейдите к официальной документации Anthropic по установке.
3. После завершения настройку можно проверить, выполнив `claude --version`, проверив конфигурационные файлы или сделав тестовый запрос.
