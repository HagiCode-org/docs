В этом разделе собраны сайты подписок на ИИ-сервисы, отмеченные как рекомендуемые в `repos/docs/public/presets/claude-code/providers/*.json`. Сначала определите здесь подходящий пункт подписки, затем перейдите к документации по настройке для завершения подключения.

По состоянию на **2026-03-09** этот раздел охватывает следующие сервисы: **Zhipu AI, MiniMax, Alibaba Cloud DashScope, Volcengine Coding Plan**.

:::tip[Рекомендуемый порядок чтения]
Если вы ещё не определились с выбором поставщика, сначала прочитайте эту страницу для сравнения. Если вы уже выбрали сервис, перейдите непосредственно на соответствующую страницу с подробностями, а затем — в раздел Claude Code на странице AI Agent CLI.
:::

## Сравнение рекомендуемых сервисов

| Сервис | Особенности | Для кого | Страница подробностей |
|--------|-------------|----------|----------------------|
| Zhipu AI | Специализированные тарифы для программирования, поддержка нескольких уровней моделей | Разработчики, желающие использовать Claude Code как основной ежедневный инструмент | [Подробнее](/ru-RU/ai-service-subscriptions/zai/) |
| MiniMax | Единое сопоставление моделей, низкие затраты на понимание конфигурации | Команды, которым важны плавность интеграции и низкие затраты на обслуживание | [Подробнее](/ru-RU/ai-service-subscriptions/minimax/) |
| Alibaba Cloud DashScope | Высокая степень интеграции с облачной платформой, единое управление через консоль | Команды, уже работающие в экосистеме Alibaba Cloud | [Подробнее](/ru-RU/ai-service-subscriptions/aliyun/) |
| Volcengine Coding Plan | Сильная расширяемость экосистемы, богатая информация об акциях | Разработчики, интересующиеся экосистемой Volcengine и мульти-модельными платформами | [Подробнее](/ru-RU/ai-service-subscriptions/volcengine/) |

:::note
`Anthropic Official` по-прежнему доступен, но поскольку в текущих метаданных preset он не отмечен как рекомендуемый, он пока не включён в首批 отдельных страниц этого раздела.
:::

## Как быстро выбрать

- **Хотите специализированные тарифы для программирования и более детальное сопоставление моделей**: сначала посмотрите [Zhipu AI](/ru-RU/ai-service-subscriptions/zai/)
- **Хотите быстрее запустить рабочий процесс и снизить затраты на понимание конфигурации**: сначала посмотрите [MiniMax](/ru-RU/ai-service-subscriptions/minimax/)
- **Активно используете ресурсы Alibaba Cloud**: сначала посмотрите [Alibaba Cloud DashScope](/ru-RU/ai-service-subscriptions/aliyun/)
- **Хотите сочетать экосистему Volcengine или возможности мульти-модельной платформы**: сначала посмотрите [Volcengine Coding Plan](/ru-RU/ai-service-subscriptions/volcengine/)
