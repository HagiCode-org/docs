## 这个分区会放什么

FAQ 分区用于集中收纳以下内容：

- 启动和运行时的特殊说明
- 来自 Index 并适合沉淀为长期文章的使用 tips
- 与平台、发行渠道、打包方式或内置运行时相关的兼容性提示
- 适合做成独立说明页的常见排障入口

如果你想系统浏览 HagiCode 的日常使用建议，请阅读 [HagiCode 使用 Tips](/faq/tips)。如果你想确认 Desktop 使用哪个 Node，或为什么诊断里出现 `portable-fixed/toolchain`，请阅读 [Desktop Node 环境说明](/faq/desktop-node-environment)。如果你想了解 HagiCode 为什么没有上架 Steam，请阅读 [为什么 HagiCode 没有上架 Steam](/faq/steam-distribution-status)。
