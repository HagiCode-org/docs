## 这个分区会放什么

- 启动和运行时的特殊说明
- 来自 Index 并适合沉淀为长期文章的使用 tips
- 与平台、发行渠道、打包方式或内置运行时相关的兼容性提示
- 适合做成独立说明页的常见排障入口
