Esta página es la entrada de instalación unificada de la CLI del AI Agent local en `repos/docs`. HagiCode actualmente admite 11 CLI de agentes activos, siguiendo el backend `PrimaryProfessions` orden: Claude Code, Codex, GitHub Copilot, OpenCode, Hermes, QoderCLI, Kiro, Kimi, Gemini, DeepAgents y Codebuddy.

:::consejo[Cómo utilizar esta página]
Primero consulte la tabla para conocer el estado de soporte actual y el modo de configuración, luego continúe con los documentos de instalación, autenticación, ACP, permisos y solución de problemas mantenidos por el proveedor. El alcance del soporte y el modo de configuración son preocupaciones separadas.
:::

## Glosario estatal

- **Actualmente compatible** significa que la CLI pertenece al rango de soporte activo actual de HagiCode.
- **Instalación con un clic** significa que HagiCode proporciona un punto de entrada de instalación guiada; Algunas CLI aún requieren un paso de autenticación o inicio de sesión del proveedor después de la instalación.
- **Instalación/autenticación manual** significa que la CLI ahora es compatible, pero debe finalizar la instalación local o el flujo de autenticación antes de que HagiCode pueda detectarlo y usarlo.
- **Solo compatibilidad heredada** significa un valor retirado como `IFlowCli` todavía existe por compatibilidad de datos históricos, pero no forma parte del rango de soporte activo actual.
