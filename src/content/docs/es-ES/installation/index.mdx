HagiCode admite múltiples rutas de instalación. Comience con el método de implementación que se ajuste a su escenario y luego agregue software de soporte solo cuando lo necesite.

El rango de compatibilidad de CLI del agente activo actual más relevante para la configuración es: Claude Code, Codex, GitHub Copilot, OpenCode, Hermes, QoderCLI, Kiro, Kimi, Gemini, DeepAgents y Codebuddy. Hermes, Gemini y DeepAgents utilizan actualmente flujos de autenticación o instalación manual; las otras CLI exponen puntos de entrada de instalación con un solo clic o flujos de instalación guiados.
