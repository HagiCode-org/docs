## Dienstpositionierung

Zhipu AI ist einer der empfohlenen Claude Code-Abonnementeinstiegspunkte im aktuellen Dokumentations-Preset. Das GLM Coding Paket mit Multi-Modell-Zuordnung eignet sich für Entwickler, die Claude Code als tägliches Hauptwerkzeug nutzen.

## Für wen geeignet

- Entwickler, die ein speziell auf Programmierung zugeschnittenes Paket wünschen
- Die eine möglichst vollständige Claude-Hierarchie-Modellzuordnung benötigen
- Die Claude Code, Cline und andere Werkzeuge über einen einzigen Dienst abdecken möchten

## Wichtige Punkte

- Im aktuellen Preset ist Zhipu AI als empfohlener Anbieter markiert.
- Es bietet eine Anthropic-kompatible Schnittstelle für die direkte Anbindung an Claude Code.
- Das Preset enthält Standardmodellzuordnungen wie `glm-4.7`, `glm-5` und `glm-4.5-air`.

## Standardmodellzuordnung

| Claude-Stufe | Standardmodell |
|-------------|----------------|
| Sonnet | `glm-4.7` |
| Opus | `glm-5` |
| Haiku | `glm-4.5-air` |

:::tip
Pakete, Angebote und Berechtigungen ändern sich im Laufe der Zeit. Wenn Sie sich für aktuelle Preise interessieren, besuchen Sie bitte die Abonnementseite des Anbieters.
:::

## Konfigurationsempfehlung

1. Bestätigen Sie zuerst, welches Paket oder welche API Key-Aktivierungsmethode Sie verwenden möchten.
2. Öffnen Sie dann den Claude Code-Eintrag auf der einheitlichen AI Agent CLI-Seite und fahren Sie mit der offiziellen Anthropic-Installationsdokumentation fort.
3. Führen Sie nach Abschluss lokal `claude --version`, eine Konfigurationsdateiprüfung oder eine tatsächliche Anfrage aus, um die Verbindung zu überprüfen.
