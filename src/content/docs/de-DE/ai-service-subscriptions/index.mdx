Dieses Kapitel fasst die aktuell als empfohlen markierten KI-Dienst-Abonnementseiten aus `repos/docs/public/presets/claude-code/providers/*.json` zusammen. Bestimmen Sie hier zuerst den Abonnementeinstieg, bevor Sie mit der Konfigurationsdokumentation fortfahren.

Stand **2026-03-09** deckt dieses Kapitel die folgenden Dienste ab: **Zhipu AI, MiniMax, Alibaba Cloud DashScope, Volcengine Coding Plan**.

:::tip[Empfohlene Lesereihenfolge]
Wenn Sie sich noch nicht für einen Anbieter entschieden haben, lesen Sie zuerst diese Seite für einen Vergleich. Wenn Sie bereits einen Anbieter gewählt haben, gehen Sie direkt zur entsprechenden Detailseite und dann zum Claude Code-Eintrag auf der Seite für einheitliche AI Agent CLI.
:::

## Vergleich der empfohlenen Dienste

| Dienst | Merkmale | Geeignet für | Detailseite |
|--------|----------|--------------|-------------|
| Zhipu AI | Programmierungsspezifisches Paket, Unterstützung mehrerer Modellstufen | Entwickler, die Claude Code als tägliches Hauptwerkzeug nutzen möchten | [Details anzeigen](./zai/) |
| MiniMax | Einheitliche Modellzuordnung, geringe Konfigurationskomplexität | Teams, denen reibungslose Integration und geringe Wartungskosten wichtig sind | [Details anzeigen](./minimax/) |
| Alibaba Cloud DashScope | Hohe Cloud-Plattform-Integration, einheitliche Konsolverwaltung | Teams, die bereits im Alibaba Cloud-Ökosystem arbeiten | [Details anzeigen](./aliyun/) |
| Volcengine Coding Plan | Starkes Ökosystem-Wachstum, umfangreiche Aktionsinformationen | Entwickler mit Interesse am Volcengine-Ökosystem und Multi-Modell-Plattformen | [Details anzeigen](./volcengine/) |

:::note
`Anthropic Official` ist weiterhin nutzbar, wird aber in den aktuellen Preset-Metadaten nicht als empfohlen markiert und daher vorerst nicht in die ersten eigenständigen Seiten dieses Kapitels aufgenommen.
:::

## Schnellauswahl

- **Programmierungsspezifisches Paket mit detaillierterer Standardmodellzuordnung**: Zuerst [Zhipu AI](./zai/) ansehen
- **Workflow schnell in Betrieb nehmen, Konfigurationsaufwand minimieren**: Zuerst [MiniMax](./minimax/) ansehen
- **Bereits intensiver Alibaba Cloud-Nutzer**: Zuerst [Alibaba Cloud DashScope](./aliyun/) ansehen
- **Volcengine-Ökosystem oder Multi-Modell-Plattform nutzen möchten**: Zuerst [Volcengine Coding Plan](./volcengine/) ansehen
