HagiCode unterstützt mehrere Installationspfade. Beginnen Sie mit der Bereitstellungsmethode, die zu Ihrem Szenario passt, und fügen Sie dann unterstützende Software nur dann hinzu, wenn Sie sie benötigen.

Der aktuell für die Einrichtung relevanteste Bereich der aktiven Agent-CLI-Unterstützung ist: Claude Code, Codex, GitHub Copilot, OpenCode, Hermes, QoderCLI, Kiro, Kimi, Gemini, DeepAgents und Codebuddy. Hermes, Gemini und DeepAgents verwenden derzeit manuelle Installations- oder Authentifizierungsabläufe; Die anderen CLIs stellen Ein-Klick-Einstiegspunkte für die Installation oder geführte Installationsabläufe bereit.

## Empfohlene nächste Schritte

- Wenn Sie neu bei HagiCode sind, beginnen Sie mit [Desktop-Installation](/de-DE/installation/desktop).
- Wenn Sie die Bereitstellung für eine Server- oder Teamumgebung durchführen, verwenden Sie [Docker Compose-Installation](/de-DE/installation/docker-compose).
- Wenn Sie den Supportstatus, den Setup-Modus oder historische Kompatibilitätshinweise für eine CLI bestätigen müssen, öffnen Sie [AI Agent CLI-Installation](/de-DE/related-software-installation/ai-agent-cli/).
