Diese Seite ist der einheitliche lokale AI Agent CLI-Installationseintrag in `repos/docs`. HagiCode unterstützt derzeit 11 aktive Agent-CLIs, die dem Backend folgen `PrimaryProfessions` Reihenfolge: Claude Code, Codex, GitHub Copilot, OpenCode, Hermes, QoderCLI, Kiro, Kimi, Gemini, DeepAgents und Codebuddy.

:::Tipp[So verwenden Sie diese Seite]
Überprüfen Sie zunächst die Tabelle auf den aktuellen Supportstatus und den Setup-Modus und fahren Sie dann mit den vom Anbieter verwalteten Installations-, Authentifizierungs-, ACP-, Berechtigungs- und Fehlerbehebungsdokumenten fort. Supportumfang und Setup-Modus sind separate Anliegen.
:::

## Staatsglossar

- **Derzeit unterstützt** bedeutet, dass die CLI zum aktuell aktiven HagiCode-Unterstützungsbereich gehört.
- **Ein-Klick-Installation** bedeutet, dass HagiCode einen Einstiegspunkt für die geführte Installation bietet; Einige CLIs erfordern nach der Installation immer noch eine Herstelleranmeldung oder einen Authentifizierungsschritt.
- **Manuelle Installation/Authentifizierung** bedeutet, dass die CLI jetzt unterstützt wird, Sie müssen jedoch die lokale Installation oder den Authentifizierungsvorgang abschließen, bevor HagiCode sie erkennen und verwenden kann.
- **Nur Legacy-Kompatibilität** bedeutet einen veralteten Wert wie z `IFlowCli` Aus Kompatibilitätsgründen mit historischen Daten existiert es weiterhin, es ist jedoch nicht Teil des derzeit aktiven Supportbereichs.
