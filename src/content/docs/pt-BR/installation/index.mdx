HagiCode oferece suporte a vários caminhos de instalação. Comece com o método de implantação adequado ao seu cenário e, em seguida, adicione software de suporte somente quando precisar dele.

A faixa de suporte atual do Agent CLI ativo mais relevante para a configuração é: Claude Code, Codex, GitHub Copilot, OpenCode, Hermes, QoderCLI, Kiro, Kimi, Gemini, DeepAgents e Codebuddy. Hermes, Gemini e DeepAgents atualmente usam instalação manual ou fluxos de autenticação; as outras CLIs expõem pontos de entrada de instalação com um clique ou fluxos de instalação guiados.

## Próximas etapas recomendadas

- Se você é novo no HagiCode, comece com [Instalação na área de trabalho](/pt-BR/installation/desktop).
