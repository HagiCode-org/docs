Esta secao reune os sites de assinatura de servicos de IA atualmente marcados como recomendados em `repos/docs/public/presets/claude-code/providers/*.json`. Determine primeiro o ponto de entrada da assinatura aqui e depois acesse a documentacao de configuracao para completar a integracao.

Ate **2026-03-09**, esta secao cobre os seguintes servicos: **Zhipu AI, MiniMax, Alibaba Cloud DashScope, Volcengine Coding Plan**.

:::tip[Ordem de leitura recomendada]
Se voce ainda nao decidiu qual provedor escolher, leia esta pagina primeiro para comparar; se voce ja escolheu o provedor, acesse diretamente a pagina de detalhes correspondente e depois va ate o item Claude Code na pagina unificada de AI Agent CLI.
:::

## Comparativo dos servicos recomendados

| Servico | Caracteristicas | Indicado para | Pagina de detalhes |
|---------|----------------|---------------|-------------------|
| Zhipu AI | Planos dedicados a programacao, suporte a mapeamento de modelos em multiplas faixas | Desenvolvedores que desejam usar o Claude Code como ferramenta principal do dia a dia | [Ver detalhes](./zai/) |
| MiniMax | Mapeamento unificado de modelos, baixo custo de compreensao de configuracao | Equipes que priorizam suavidade de integracao e custo de manutencao | [Ver detalhes](./minimax/) |
| Alibaba Cloud DashScope | Alta integracao com plataforma cloud, adequado para gerenciamento unificado no console | Equipes que ja trabalham no ecossistema Alibaba Cloud | [Ver detalhes](./aliyun/) |
| Volcengine Coding Plan | Forte extensibilidade de ecossistema, informacoes ricas de promocoes | Desenvolvedores interessados no ecossistema Volcengine e capacidades de plataformas multimodelo | [Ver detalhes](./volcengine/) |

:::note
`Anthropic Official` ainda pode ser utilizado, mas como nao esta marcado como recomendado nos metadados atuais do preset, nao e incluido no escopo inicial desta secao.
:::

## Como escolher rapidamente

- **Quer um plano dedicado a programacao com mapeamento padrao mais detalhado**: priorize [Zhipu AI](./zai/)
- **Quer colocar o fluxo de trabalho em funcionamento o mais rapido possivel com menor custo de configuracao**: priorize [MiniMax](./minimax/)
- **Ja usa intensivamente recursos Alibaba Cloud**: priorize [Alibaba Cloud DashScope](./aliyun/)
- **Deseja combinar o ecossistema Volcengine ou capacidades de plataforma multimodelo**: priorize [Volcengine Coding Plan](./volcengine/)
