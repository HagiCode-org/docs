## Posicionamento do servico

O Zhipu AI e um dos pontos de entrada de assinatura do Claude Code recomendados no preset atual da documentacao. Com planos GLM Coding e mapeamento multimodelo, e adequado para desenvolvedores que desejam usar o Claude Code como ferramenta principal do dia a dia.

## Indicado para quem

- Deseja obter planos voltados para cenarios de programacao
- Precisa de mapeamento relativamente completo dos niveis do Claude
- Quer cobrir Claude Code, Cline e outras ferramentas com um unico servico

## Pontos a observar

- O preset atual marca o Zhipu AI como provedor recomendado.
- Oferece interface compativel com Anthropic para integracao direta com Claude Code.
- O preset inclui mapeamentos padrao como `glm-4.7`, `glm-5` e `glm-4.5-air`.

## Informacoes-chave de integracao

| Item | Conteudo |
|------|----------|
| Ponto de assinatura | [Ir para a pagina de assinatura](https://www.bigmodel.cn/claude-code?ic=14BY54APZA) |
| Documentacao oficial / Console | [Ver pagina oficial](https://open.bigmodel.cn/dev/api) |
| Base URL compativel com Anthropic | `https://open.bigmodel.cn/api/anthropic` |
| Variavel de ambiente de autenticacao recomendada | `ANTHROPIC_AUTH_TOKEN` |

## Mapeamento padrao de modelos

| Nivel do Claude | Modelo padrao |
|-----------------|---------------|
| Sonnet | `glm-4.7` |
| Opus | `glm-5` |
| Haiku | `glm-4.5-air` |

:::tip
Planos, promocoes e beneficios podem mudar ao longo do tempo; se voce esta mais preocupado com precos em tempo real, consulte diretamente a pagina de assinatura do provedor.
:::

## Recomendacoes de configuracao

1. Primeiro, confirme o plano ou metodo de ativacao de API Key que deseja usar.
2. Depois, acesse o item Claude Code na pagina unificada de AI Agent CLI e continue para a instalacao e documentacao oficial da Anthropic.
3. Apos concluir, voce pode verificar se a integracao foi bem-sucedida executando `claude --version` localmente, verificando o arquivo de configuracao ou fazendo uma solicitacao real.
