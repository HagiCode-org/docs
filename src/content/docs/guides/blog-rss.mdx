HagiCode 文档博客提供三个 RSS 订阅源。你可以继续使用原有的全量订阅，也可以只订阅中文或英文文章。

| 订阅范围 | RSS 地址 |
|----------|----------|
| 全部语言 | `https://docs.hagicode.com/blog/rss.xml` |
| 中文文章 | `https://docs.hagicode.com/blog/rss.zh-CN.xml` |
| 英文文章 | `https://docs.hagicode.com/blog/rss.en.xml` |

## 兼容性

`https://docs.hagicode.com/blog/rss.xml` 是现有的兼容订阅源，会继续输出中文和英文文章。已经订阅这个地址的阅读器、自动化脚本和 README 同步流程不需要修改。

## 语言规则

RSS 生成时会先为每篇博客文章归一化语言：

- 如果文章 frontmatter 显式设置了 `language`，会将 `zh` / `zh-CN` 归一化为 `zh-CN`，将 `en` / `en-US` 归一化为 `en`。
- 如果没有显式语言字段，则根据内容路径判断：`src/content/docs/blog/` 视为中文，`src/content/docs/en/blog/` 视为英文。

每个 RSS 条目都会带有对应的 `<language>` 标记，便于订阅器或下游自动化流程识别来源语言。
