## 服務定位

火山引擎 Coding Plan 是目前文件 preset 中推薦的 Claude Code 訂閱入口之一。適合關注平台活動、希望在火山引擎生態中使用多模型能力的開發者。

## 適合誰

- 希望在火山引擎生態中繼續擴展 AI 工具鏈
- 關注多模型平台能力與活動入口
- 需要 Anthropic 相容入口來接入 Claude Code

## 你可以先關注這些點

- 目前 preset 將火山引擎 Coding Plan 標記為推薦服務商。
- 預設說明中提到其支援 Doubao、GLM、DeepSeek、Kimi 等模型生態。
- 活動與價格變化較快，具體權益請以服務商訂閱頁面即時說明為準。

## 配置建議

1. 先確認你想使用的套餐或 API Key 開通方式。
2. 再打開統一 AI Agent CLI 頁面中的 Claude Code 條目，繼續前往 Anthropic 官方安裝與參考文件。
3. 完成後可在本機執行 `claude --version`、配置檔案檢查或一次實際請求來驗證接入是否成功。
