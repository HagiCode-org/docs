本章節彙整 `repos/docs/public/presets/claude-code/providers/*.json` 中目前標記為推薦的 AI 服務訂閱網站。先在這裡確定訂閱入口，再進入配置文件完成接入。

截至 **2026-03-09**，本章節覆蓋以下服務：**智譜 AI、MiniMax、阿里雲 DashScope、火山引擎 Coding Plan**。

:::tip[推薦閱讀順序]
若你還沒確定買哪一家，先看本頁做對比；若你已經選好服務商，直接進入對應詳情頁，再去統一 AI Agent CLI 頁面裡的 Claude Code 條目。
:::

## 目前推薦服務對比

| 服務 | 特點 | 適合誰 | 詳情頁 |
|------|------|--------|--------|
| 智譜 AI | 程式設計專項套餐，支援多檔模型對應 | 希望把 Claude Code 作為日常主力工具的開發者 | [查看詳情](/zh-Hant/ai-service-subscriptions/zai/) |
| MiniMax | 統一模型對應，配置理解成本低 | 優先考慮接入平滑度和維護成本的團隊 | [查看詳情](/zh-Hant/ai-service-subscriptions/minimax/) |
| 阿里雲 DashScope | 雲平台整合度高，適合控制台統一管理 | 已經在阿里雲生態中工作的團隊 | [查看詳情](/zh-Hant/ai-service-subscriptions/aliyun/) |
| 火山引擎 Coding Plan | 生態延展性強，活動資訊豐富 | 關注火山引擎生態和多模型平台能力的開發者 | [查看詳情](/zh-Hant/ai-service-subscriptions/volcengine/) |

:::note
`Anthropic Official` 目前仍可使用，但因為它在當前 preset 元資料中沒有標記為推薦項，所以暫不納入本章節的首批獨立頁面範圍。
:::

## 如何快速選擇

- **想要程式設計專項套餐與更細的預設模型對應**：優先看 [智譜 AI](/zh-Hant/ai-service-subscriptions/zai/)
- **想要盡快跑通工作流、減少配置理解成本**：優先看 [MiniMax](/zh-Hant/ai-service-subscriptions/minimax/)
- **已經深度使用阿里雲資源**：優先看 [阿里雲 DashScope](/zh-Hant/ai-service-subscriptions/aliyun/)
- **希望結合火山引擎生態或多模型平台能力**：優先看 [火山引擎 Coding Plan](/zh-Hant/ai-service-subscriptions/volcengine/)
