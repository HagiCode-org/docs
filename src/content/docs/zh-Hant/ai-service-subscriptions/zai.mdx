## 服務定位

智譜 AI 是目前文件 preset 中推薦的 Claude Code 訂閱入口之一。GLM Coding 套餐與多模型對應，適合把 Claude Code 作為日常主力工具的開發者。

## 適合誰

- 希望獲得專門面向程式設計場景的套餐
- 需要較完整的 Claude 層級模型對應
- 想透過一個服務覆蓋 Claude Code、Cline 等多種工具

## 你可以先關注這些點

- 目前 preset 中將智譜 AI 標記為推薦服務商。
- 提供 Anthropic 相容介面，便於直接對接 Claude Code。
- 預設中包含 `glm-4.7`、`glm-5` 與 `glm-4.5-air` 等預設模型對應。

## 關鍵接入資訊

| 項目 | 內容 |
|------|------|
| 訂閱入口 | [前往訂閱頁面](https://www.bigmodel.cn/claude-code?ic=14BY54APZA) |
| 官方文件 / 控制台 | [查看官方頁面](https://open.bigmodel.cn/dev/api) |
| Anthropic 相容 Base URL | `https://open.bigmodel.cn/api/anthropic` |
| 推薦認證環境變數 | `ANTHROPIC_AUTH_TOKEN` |

## 預設模型對應

| Claude 層級 | 預設模型 |
|-------------|----------|
| Sonnet | `glm-4.7` |
| Opus | `glm-5` |
| Haiku | `glm-4.5-air` |

:::tip
套餐、優惠和權益會隨時間變化；如果你更關注即時價格，請直接查看服務商訂閱頁面。
:::

## 配置建議

1. 先確認你想使用的套餐或 API Key 開通方式。
2. 再打開統一 AI Agent CLI 頁面中的 Claude Code 條目，繼續前往 Anthropic 官方安裝與參考文件。
3. 完成後可在本機執行 `claude --version`、配置檔案檢查或一次實際請求來驗證接入是否成功。
