## 服務定位

MiniMax 是目前文件 preset 中推薦的 Claude Code 訂閱入口之一。統一模型、配置簡單，適合優先考慮接入平滑度和維護成本的團隊。

## 適合誰

- 希望三檔模型都使用同一套預設對應
- 更關注配置簡單和穩定接入
- 想先快速跑通 Claude Code 工作流

## 你可以先關注這些點

- 目前 preset 將 MiniMax 標記為推薦服務商。
- 統一使用 `MiniMax-M2.5` 作為預設模型，配置理解成本較低。
- 適合希望先跑通工作流、後續再細化模型策略的團隊。

## 關鍵接入資訊

| 項目 | 內容 |
|------|------|
| 訂閱入口 | [前往訂閱頁面](https://platform.minimaxi.com/subscribe/coding-plan?code=8wNck4etDM&source=link) |
| 官方文件 / 控制台 | [查看官方頁面](https://www.minimaxi.com) |
| Anthropic 相容 Base URL | `https://api.minimaxi.com/anthropic` |
| 推薦認證環境變數 | `ANTHROPIC_AUTH_TOKEN` |

## 預設模型對應

| Claude 層級 | 預設模型 |
|-------------|----------|
| Sonnet | `MiniMax-M2.5` |
| Opus | `MiniMax-M2.5` |
| Haiku | `MiniMax-M2.5` |

:::tip
套餐、優惠和權益會隨時間變化；如果你更關注即時價格，請直接查看服務商訂閱頁面。
:::

## 配置建議

1. 先確認你想使用的套餐或 API Key 開通方式。
2. 再打開統一 AI Agent CLI 頁面中的 Claude Code 條目，繼續前往 Anthropic 官方安裝與參考文件。
3. 完成後可在本機執行 `claude --version`、配置檔案檢查或一次實際請求來驗證接入是否成功。
