## 服務定位

阿里雲 DashScope 是目前文件 preset 中推薦的 Claude Code 訂閱入口之一。適合已經在阿里雲生態中工作的團隊，希望把訂閱、控制台與金鑰管理統一在雲平台裡。

## 適合誰

- 已經在使用阿里雲控制台或雲資源
- 希望從統一雲平台管理服務與 API Key
- 希望透過固定模型對應快速接入 Claude Code

## 你可以先關注這些點

- 目前 preset 將阿里雲 DashScope 標記為推薦服務商。
- 使用 `glm-4.7` 作為預設對應模型，配置路徑清晰。
- 如果團隊已有阿里雲帳號體系，接入與權限管理會更順手。

## 關鍵接入資訊

| 項目 | 內容 |
|------|------|
| 訂閱入口 | [前往訂閱頁面](https://www.aliyun.com/benefit/ai/aistar?userCode=vmx5szbq&clubBiz=subTask..12384055..10263..) |
| 官方文件 / 控制台 | [查看官方頁面](https://dashscope.aliyun.com) |
| Anthropic 相容 Base URL | `https://coding.dashscope.aliyuncs.com/apps/anthropic` |
| 推薦認證環境變數 | `ANTHROPIC_AUTH_TOKEN` |

## 預設模型對應

| Claude 層級 | 預設模型 |
|-------------|----------|
| Sonnet | `glm-4.7` |
| Opus | `glm-4.7` |
| Haiku | `glm-4.7` |

:::tip
套餐、優惠和權益會隨時間變化；如果你更關注即時價格，請直接查看服務商訂閱頁面。
:::

## 配置建議

1. 先確認你想使用的套餐或 API Key 開通方式。
2. 再打開統一 AI Agent CLI 頁面中的 Claude Code 條目，繼續前往 Anthropic 官方安裝與參考文件。
3. 完成後可在本機執行 `claude --version`、配置檔案檢查或一次實際請求來驗證接入是否成功。
