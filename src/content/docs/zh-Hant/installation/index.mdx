HagiCode 支援多種安裝路徑。從適合您場景的部署方法開始，然後僅在需要時新增支援軟體。

目前與設定最相關的有效代理 CLI 支援範圍是：Claude Code、Codex、GitHub Copilot、OpenCode、Hermes、QoderCLI、Kiro、Kimi、Gemini、DeepAgents 和 Codebuddy。 Hermes、Gemini 和 DeepAgents 目前使用手動安裝或驗證流程；其他 CLI 公開一鍵安裝入口點或引導安裝流程。

## 建議的後續步驟

- 如果您是 HagiCode 的新手，請從 [桌面安裝](/zh-Hant/installation/desktop).
- 如果您正在為伺服器或團隊環境進行部署，請使用 [Docker Compose 安裝](/zh-Hant/installation/docker-compose).
- 如果您需要確認 CLI 的支援狀態、設定模式或歷史相容性說明，請開啟 [AI代理CLI安裝](/zh-Hant/related-software-installation/ai-agent-cli/).
