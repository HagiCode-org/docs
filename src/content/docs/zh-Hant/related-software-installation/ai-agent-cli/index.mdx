該頁面是統一的本地AI Agent CLI安裝入口 `repos/docs`。 HagiCode 目前支援 11 個活動代理 CLI，遵循後端 `PrimaryProfessions` 順序：Claude Code、Codex、GitHub Copilot、OpenCode、Hermes、QoderCLI、Kiro、Kimi、Gemini、DeepAgents 和 Codebuddy。

:::提示[如何使用此頁面]
首先檢查目前支援狀態和設定模式的表，然後繼續執行供應商維護的安裝、驗證、ACP、權限和故障排除文件。支援範圍和設定模式是不同的問題。
:::

## 國家術語表

- **目前支援** 表示 CLI 屬於目前有效的 HagiCode 支援範圍。
- **一鍵安裝**表示HagiCode提供了引導式安裝入口點；某些 CLI 在安裝後仍需要供應商登入或驗證步驟。
- **手動安裝/身份驗證**表示現在支援 CLI，但您必須完成本機安裝或驗證流程，HagiCode 才能偵測到並使用它。
- **僅舊版相容性**表示已退役的值，例如 `IFlowCli` 為了歷史資料相容性仍然存在，但它不屬於目前的活躍支撐範圍。
