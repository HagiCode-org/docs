Hero Feature Showcase Assets
============================

This directory stores a docs-owned subset of hero avatar visuals for the
product overview pages.

Non-interactive default:
- mirror the current homepage "Interesting" gallery vocabulary
- keep the docs build independent from repos/web runtime assets
- cover captain, support, and battle-report moods with a compact subset
