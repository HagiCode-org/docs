HagiCode birden fazla kurulum yolunu destekler. Senaryonuza uygun dağıtım yöntemiyle başlayın, ardından yalnızca ihtiyaç duyduğunuzda destekleyici yazılımı ekleyin.

Kurulumla en alakalı mevcut aktif Agent CLI destek aralığı şunlardır: Claude Code, Codex, GitHub Copilot, OpenCode, Hermes, QoderCLI, Kiro, Kimi, Gemini, DeepAgents ve Codebuddy. Hermes, Gemini ve DeepAgents şu anda manuel kurulum veya kimlik doğrulama akışlarını kullanıyor; diğer CLI'ler tek tıklamayla kurulum giriş noktalarını veya yönlendirmeli kurulum akışlarını ortaya çıkarır.
