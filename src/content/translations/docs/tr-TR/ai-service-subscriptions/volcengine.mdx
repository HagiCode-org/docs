## Konumlandırma

Volcengine Kodlama Planı, şu anda Claude Code için ayarlanmış dokümanlar tarafından önerilen sağlayıcılardan biridir. Volcengine ekosistem uyumu ve geniş çoklu model platform seçenekleri isteyen geliştiriciler için iyi bir seçim.

## için iyi uyum

- Volcengine ekosistemi içinde kalmak istiyorsunuz
- Çok modelli platform seçeneklerine ve kampanya odaklı planlara önem veriyorsunuz
- Claude Kodu için Antropik uyumlu bir uç noktaya ihtiyacınız var

## Dikkat edilmesi gereken önemli noktalar

- Mevcut ön ayarlı meta veriler, Volcengine Kodlama Planını tavsiye edildiği gibi işaretler.
- Ön ayar notlarında Doubao, GLM, DeepSeek ve Kimi gibi ekosistemlere yönelik destekten bahsediliyor.
- Promosyonlar ve fiyatlar hızla değişebileceğinden, sağlayıcı sayfasındaki en son paket ayrıntılarını doğrulayın.

## Entegrasyon anlık görüntüsü

| Öğe | Değer |
|------|-------|
| Abonelik girişi | [Abonelik sayfasını aç](https://volcengine.com/L/Irib4rkFUHk/?inviteCode=KZQBPSE5) |
| Resmi belgeler / konsol | [Resmi sayfayı aç](https://www.volcengine.com) |
| Antropik uyumlu temel URL | `https://ark.cn-beijing.volces.com/api/coding` |
| Önerilen kimlik doğrulama ortamı var | `ANTHROPIC_AUTH_TOKEN` |

| Claude katmanı | Varsayılan model |
|-------------|---------------|
| Sone | `glm-4.7` |
| Eser | `glm-4.7` |

:::ipucu
Fiyatlandırma, promosyonlar ve paket ayrıntıları zamanla değişebilir. Eğer asıl endişeniz maliyetse, en son bilgileri doğrudan sağlayıcı sayfasından doğrulayın.
:::

## Önerilen sonraki adımlar

1. Hangi abonelik katmanını veya API etkinleştirme yolunu kullanmak istediğinizi onaylayın.
2. Birleşik AI Agent CLI merkezinde Claude Code girişini açın ve Anthropic'in resmi kurulum ve referans belgelerine devam edin.
3. Kurulumdan sonra bağlantıyı yerel olarak doğrulayın.
