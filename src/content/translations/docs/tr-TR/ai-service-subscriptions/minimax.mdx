## Konumlandırma

MiniMax, Claude Code için belirlenen dokümanlar ön ayarı tarafından şu anda önerilen sağlayıcılardan biridir. Basit yapılandırmaya ve öngörülebilir katılıma değer veren ekipler için uygun, katmanlar arasında tek bir varsayılan model.

## için iyi uyum

- Tüm Claude katmanlarında aynı eşlenmiş modeli istiyorsunuz
- Düşük sürtünmeli kurulum ve bakımı tercih ediyorsunuz
- Model stratejisini ayarlamadan önce Claude Code'un hızlı bir şekilde çalışmasını sağlamak istiyorsunuz

## Dikkat edilmesi gereken önemli noktalar

- Geçerli ön ayarlı meta veriler MiniMax'ı önerilen olarak işaretler.
- Kullanır `MiniMax-M2.5` tüm katmanlar için varsayılan eşlenen model olarak.
- Operasyonel basitliğin seviyeye özgü ayarlardan daha önemli olduğu durumlarda bu iyi bir seçimdir.

## Entegrasyon anlık görüntüsü

| Öğe | Değer |
|------|-------|
| Abonelik girişi | [Abonelik sayfasını aç](https://platform.minimaxi.com/subscribe/coding-plan?code=8wNck4etDM&source=link) |
| Resmi belgeler / konsol | [Resmi sayfayı aç](https://www.minimaxi.com) |
| Antropik uyumlu temel URL | `https://api.minimaxi.com/anthropic` |
| Önerilen kimlik doğrulama ortamı var | `ANTHROPIC_AUTH_TOKEN` |

## Varsayılan model eşleme

| Claude katmanı | Varsayılan model |
|-------------|---------------|
| Sone | `MiniMax-M2.5` |
| Eser | `MiniMax-M2.5` |
| Haiku | `MiniMax-M2.5` |

## Önerilen sonraki adımlar

1. Hangi abonelik katmanını veya API etkinleştirme yolunu kullanmak istediğinizi onaylayın.
2. Birleşik AI Agent CLI merkezinde Claude Code girişini açın ve Anthropic'in resmi kurulum ve referans belgelerine devam edin.
3. Kurulumdan sonra bağlantıyı yerel olarak doğrulayın.
