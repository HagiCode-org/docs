## Konumlandırma

Alibaba Cloud DashScope, Claude Code için belirlenen dokümanlar ön ayarı tarafından şu anda önerilen sağlayıcılardan biridir. Halihazırda Alibaba Bulut ekosisteminde faaliyet gösteren ve birleşik konsol yönetimi isteyen ekipler için pratik bir seçim.

## için iyi uyum

- Alibaba Cloud hesaplarını veya kaynaklarını zaten kullanıyorsunuz
- Abonelik ve anahtar yönetiminin tek bir bulut konsolunda kalmasını istiyorsunuz
- Hızlı katılım için sabit eşlenmiş bir modeli tercih ediyorsunuz

## Dikkat edilmesi gereken önemli noktalar

- Mevcut ön ayarlı meta veriler, Alibaba Cloud DashScope'u önerildiği şekilde işaretler.
- Tüm Claude katmanlarını eşler `glm-4.7` basit bir kurulum yolu için.
- Ekibiniz zaten Alibaba Cloud iş akışlarında çalışıyorsa bu özellikle kullanışlıdır.

## Entegrasyon anlık görüntüsü

| Öğe | Değer |
|------|-------|
| Abonelik girişi | [Abonelik sayfasını aç](https://www.aliyun.com/benefit/ai/aistar?userCode=vmx5szbq&clubBiz=subTask..12384055..10263..) |
| Resmi belgeler / konsol | [Resmi sayfayı aç](https://dashscope.aliyun.com) |
| Antropik uyumlu temel URL | `https://coding.dashscope.aliyuncs.com/apps/anthropic` |
| Önerilen kimlik doğrulama ortamı var | `ANTHROPIC_AUTH_TOKEN` |

## Varsayılan model eşleme

| Claude katmanı | Varsayılan model |
|-------------|---------------|
| Sone | `glm-4.7` |
| Eser | `glm-4.7` |
| Haiku | `glm-4.7` |

:::ipucu
Fiyatlandırma, promosyonlar ve paket ayrıntıları zamanla değişebilir. Eğer asıl endişeniz maliyetse, en son bilgileri doğrudan sağlayıcı sayfasından doğrulayın.
:::

## Önerilen sonraki adımlar

1. Hangi abonelik katmanını veya API etkinleştirme yolunu kullanmak istediğinizi onaylayın.
2. Birleşik AI Agent CLI merkezinde Claude Code girişini açın ve Anthropic'in resmi kurulum ve referans belgelerine devam edin.
3. Kurulumdan sonra bağlantıyı yerel olarak doğrulayın.
