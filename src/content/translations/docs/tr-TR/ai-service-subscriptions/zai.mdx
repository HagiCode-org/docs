## Konumlandırma

ZAI, şu anda Claude Code için ayarlanmış dokümanlar tarafından önerilen sağlayıcılardan biridir. GLM Kodlama, Claude Code'u günlük bir sürücü olarak isteyen geliştiriciler için katman eşlemeli planlar yapar.

## için iyi uyum

- Kodlama odaklı bir abonelik planı istiyorsunuz
- Eşlenmiş Sonnet/Opus/Haiku varsayılanlarına ihtiyacınız var
- Tek bir sağlayıcının Claude Code ve benzeri kodlama araçlarıyla çalışmasını beklersiniz

## Dikkat edilmesi gereken önemli noktalar

- Geçerli ön ayarlı meta veriler ZAI'yi önerildiği şekilde işaretler.
- Doğrudan Claude Code tarafından kullanılabilecek Antropik uyumlu bir uç noktayı ortaya çıkarır.
- Ön ayar, aşağıdakiler için varsayılan eşlemeleri gönderir: `glm-4.7`, `glm-5`ve `glm-4.5-air`.

## Entegrasyon anlık görüntüsü

| Öğe | Değer |
|------|-------|
| Abonelik girişi | [Abonelik sayfasını aç](https://www.bigmodel.cn/claude-code?ic=14BY54APZA) |
| Resmi belgeler / konsol | [Resmi sayfayı aç](https://open.bigmodel.cn/dev/api) |
| Antropik uyumlu temel URL | `https://open.bigmodel.cn/api/anthropic` |
| Önerilen kimlik doğrulama ortamı var | `ANTHROPIC_AUTH_TOKEN` |

## Varsayılan model eşleme

| Claude katmanı | Varsayılan model |
|-------------|---------------|
| Sone | `glm-4.7` |
| Eser | `glm-5` |
| Haiku | `glm-4.5-air` |

:::ipucu
Fiyatlandırma, promosyonlar ve paket ayrıntıları zamanla değişebilir. Eğer asıl endişeniz maliyetse, en son bilgileri doğrudan sağlayıcı sayfasından doğrulayın.
:::

## Önerilen sonraki adımlar

1. Hangi abonelik katmanını veya API etkinleştirme yolunu kullanmak istediğinizi onaylayın.
2. Birleşik AI Agent CLI merkezinde Claude Code girişini açın ve Anthropic'in resmi kurulum ve referans belgelerine devam edin.
3. Kurulumdan sonra bağlantıyı yerel olarak doğrulayın.
