Den här sidan är den förenade lokala AI Agent CLI-installationsposten i `repos/docs`. HagiCode stöder för närvarande 11 aktiva agent-CLI:er, efter backend `PrimaryProfessions` ordning: Claude Code, Codex, GitHub Copilot, OpenCode, Hermes, QoderCLI, Kiro, Kimi, Gemini, DeepAgents och Codebuddy.

:::tips[Hur man använder den här sidan]
Kontrollera först i tabellen för aktuellt supportläge och inställningsläge, fortsätt sedan med de leverantörsunderhållna installations-, autentiserings-, ACP-, behörighets- och felsökningsdokumenten. Supportomfattning och inställningsläge är separata problem.
:::

## Statens ordlista

- **Stöds för närvarande** betyder att CLI tillhör det nuvarande aktiva HagiCode-stödområdet.
- **Installation med ett klick** betyder att HagiCode tillhandahåller en ingångspunkt för guidad installation; vissa CLI kräver fortfarande en leverantörsinloggning eller autentiseringssteg efter installationen.
- **Manuell installation/autentisering** betyder att CLI stöds nu, men du måste slutföra den lokala installationen eller autentiseringsflödet innan HagiCode kan upptäcka och använda det.
- **Endast äldre kompatibilitet** betyder ett pensionerat värde som t.ex `IFlowCli` existerar fortfarande för historisk datakompatibilitet, men det är inte en del av det nuvarande aktiva supportintervallet.
