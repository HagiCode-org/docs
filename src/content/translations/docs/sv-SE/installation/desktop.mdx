Den här guiden täcker installation och användning av Hagicode Desktop. Det är den officiella skrivbordsklienten och håller standardvägen för lokala installationer plus dagliga kontroller på ett ställe.

:::tips[Rekommenderad metod]
Desktop är den lokala standardsökvägen för enskilda användare och utvecklare eftersom:
- installationsprogrammet håller standardflödet i en vägledd väg
- beroenden och tjänster hanteras i appen
- UI gör tillstånd och drift lättare att inspektera
- inbyggd versionshantering gör bytet enklare
- data förblir lokal som standard
:::

## Vad är Hagicode Desktop?

Hagicode Desktop är Hagicodes officiella skrivbordsapplikation, designad för Windows-, macOS- och Linux-system. Den kombinerar Hagicodes kärnfunktioner, och installationsprogrammet behåller standardinställningsvägen i ett guidat flöde.

### Huvudsakliga fördelar

- **Installation med ett klick**: Slutför alla konfigurationssteg automatiskt via installationsguiden
- **Automatisk körtidsförberedelse**: Standardinstallationsprogrammet innehåller redan den körtid det behöver, så du behöver inte installera en separat `.NET` körtid först
- **Service Management**: Inbyggd Hagicode Server-hantering, enkelt starta och stoppa tjänster
- **Versionsbyte**: Stöd för multiversionshantering, växla fritt mellan olika versioner
- **Lokal drift**: All data lagras lokalt, vilket skyddar din integritet
- **Automatiska uppdateringar**: Upptäcker och laddar ned den senaste versionen automatiskt

### Tillämpliga scenarier

Desktop passar bra när:
- Enskilda utvecklare som vill använda Hagicode lokalt
- Användare som behöver använda Hagicode offline
- Icke-tekniska användare som vill förenkla installationsprocessen
- Utvecklare som ofta behöver byta mellan olika versioner

### Andra nedladdningsmetoder

Du kan också hämta Hagicode Desktop-installationsprogrammet från följande kanaler:

1. **Officiell releasesida**: Besök [Hagicode GitHub släpps](https://github.com/Hagicode-org/releases) för att ladda ner den senaste versionen
2. **Officiell webbplats**: Besök [Hagicode webbplats](https://hagicode.com/) för att få nedladdningslänkar

### Välj rätt version

Välj motsvarande installationsprogram för ditt operativsystem:
- **Windows**: Ladda ner `.exe` installatör
- **macOS**: Ladda ner `.dmg` installatör
- **Linux**: Ladda ner `.AppImage` eller `.tar.gz` installationspaket

:::note[Systemkrav]
- **Windows**: Windows 10 eller senare version (64-bitars)
- **macOS**: macOS 10.15 eller senare
- **Linux**: Mainstream Linux-distributioner (Ubuntu 20.04+, Fedora 33+, etc.)
:::

:::tip[Standardförutsättningar]
För det senaste HagiCode lokala installationsflödet behöver du vanligtvis inte förbereda en separat `.NET` körtid. Desktop paketerar redan den körtid som krävs för standardflödet och initierar den lokala SQLite-sökvägen som stöds som standard.
:::

## Installations- och startanteckningar

:::varning[macOS Gatekeeper Notice]
Om du redan har installerat appen till standardsökvägen `/Applications/Hagicode.app` men macOS visar "appen är skadad", "kan inte öppnas" eller en liknande Gatekeeper-varning vid första lansering, den vanliga orsaken är `com.apple.quarantine` attribut som macOS lägger till appar som laddas ner från internet, inte ett fel i själva Hagicode Desktop-installationsprogrammet.

Kör följande kommando i Terminal. Det kräver administratörsbehörigheter och tar rekursivt bort karantänattributet från appen i standardinstallationssökvägen:

```bash
sudo xattr -dr com.apple.quarantine /Applications/Hagicode.app
```

När kommandot är klart, starta om Hagicode Desktop.
:::

## Kör Desktop

Efter nedladdning öppnar du installationsprogrammet eller applikationspaketet och följer instruktionerna på skärmen. Desktop förbereder körtiden i den lokala sökvägen som stöds för närvarande, hanterar Hagicode Server och visar framsteg och status när nätverksnedladdningar krävs.

:::note[Nätverkskrav]
Se till att din nätverksanslutning är tillgänglig. Desktop kan behöva hämta versionsinformation eller komponentpaket från internet; Om en nedladdning misslyckas, kontrollera dina nätverks-, proxy- eller brandväggsinställningar och försök igen.
:::

## Versionshantering

Hagicode Desktop stöder hantering av flera versioner, så att du fritt kan växla mellan olika versioner.

![Versionshantering](../../img/screenshots/installation/install-desktop/12-version-management/original.png)

### Byt version

1. Öppna Hagicode Desktop
2. Gå till sidan "Versionshantering".
3. Välj den version du vill byta
4. Klicka på knappen "Byt version".
5. Vänta tills bytet är klart

:::note[Version Switching Notes]
- Om du byter version startar Hagicode Server automatiskt om
- Se till att du sparar allt arbete innan du byter
- Projektkonfigurationer kan skilja sig åt mellan olika versioner
:::

## Starta och stoppa service

Hagicode Desktop erbjuder ett enkelt sätt att hantera Hagicode Server-tjänsten.

### Starta service

På skrivbordsprogrammets startsida klickar du på knappen "Starta tjänst" för att starta Hagicode Server.

![Starta service](../../img/screenshots/installation/install-desktop/13-start-service-homepage/original.png)

### Stoppa service

För att stoppa Hagicode Server, klicka på knappen "Stoppa tjänst" på startsidan.

:::tips[Servicestatus]
Skrivbordsapplikationen visar Hagicode Servers körstatus i realtid, inklusive:
- Servicestatus (kör/stoppad)
- Serviceportnummer
- Information om aktuell version
:::

## Felsökning

### Installationsfel

Om du stöter på problem under installationen:

1. **Kontrollera nätverksanslutning**: Se till att du har tillgång till internet
2. **Kontrollera diskutrymme**: Se till att du har tillräckligt med diskutrymme
3. **Kör som administratör**: I Windows högerklickar du på installationsprogrammet och väljer "Kör som administratör"
4. **Visa loggfiler**: Installationsprogrammet genererar detaljerade loggfiler som kan hjälpa till att diagnostisera problem

### Beroendeinstallationsfel

Om beroendeinstallationen misslyckas:

1. **Kontrollera systemkompatibilitet**: Se till att ditt operativsystem uppfyller minimikraven
2. **Kontrollera nätverks- och brandväggsinställningar**: Se till att installationsprogrammet kan nå nödvändiga nedladdningar och inte blockeras av din brandvägg
3. **Bekräfta installationsbehörigheter**: I Windows, försök att köra installationsprogrammet som administratör så att det har tillräckliga behörigheter
4. **Starta om appen och försök igen**: Stäng och öppna Hagicode Desktop igen, försök sedan installera igen
5. **Visa loggar**: Granska detaljerade felloggar i skrivbordsprogrammet

### Tjänstens startfel

Om Hagicode Server inte startar:

1. **Kontrollera portbeläggning**: Se till att port 45000 inte är upptagen av andra program
2. **Kontrollera brandväggsinställningarna**: Se till att brandväggen tillåter Hagicode Server-nätverksåtkomst
3. **Starta om programmet**: Testa att starta om Hagicode Desktop
4. **Visa loggar**: Visa detaljerade felloggar i skrivbordsprogrammet

### Behöver du mer hjälp?

Om du stöter på problem som inte täcks här:

1. Kontrollera [GitHub-problem](https://github.com/HagiCode-org/site/issues) för liknande problem
2. Besök vår [Samhällsforum](https://github.com/HagiCode-org/site/discussions) för hjälp
3. Skicka in ett nytt nummer och beskriv ditt problem i detalj
