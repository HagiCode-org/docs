HagiCode stöder flera installationsvägar. Börja med den distributionsmetod som passar ditt scenario och lägg sedan till stödprogramvara endast när du behöver den.

Det nuvarande aktiva Agent CLI-stödintervallet som är mest relevant för installationen är: Claude Code, Codex, GitHub Copilot, OpenCode, Hermes, QoderCLI, Kiro, Kimi, Gemini, DeepAgents och Codebuddy. Hermes, Gemini och DeepAgents använder för närvarande manuella installations- eller autentiseringsflöden; de andra CLI:erna exponerar installationsingångspunkter med ett klick eller guidade installationsflöden.

## Rekommenderade nästa steg

- Om du planerar att köpa och installera HagiCode genom Steam, börja med [Steam installation](/sv-SE/installation/steam).
- Om du är ny på HagiCode, börja med [Desktopinstallation](/sv-SE/installation/desktop).
- Om du distribuerar för en server- eller teammiljö, använd [Installation av Docker Compose](/sv-SE/installation/docker-compose).
- Om du behöver bekräfta supportstatus, inställningsläge eller historiska kompatibilitetsanmärkningar för en CLI, öppna [AI Agent CLI-installation](/sv-SE/related-software-installation/ai-agent-cli/).
