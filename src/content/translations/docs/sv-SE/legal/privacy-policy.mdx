Ikraftträdandedatum: 2026-04-15

Denna integritetspolicy förklarar vilka kategorier av data HagiCode kan behandla när den tillhandahåller `hagicode.com` och alla dess underdomänwebbplatser, vägledning för nedladdning av programvara, supportkanaler och relaterade produktupplevelser, tillsammans med syftena, bevarandemetoden och rättigheterna som är tillgängliga för dig.

## Kategorier av information vi samlar in

Beroende på hur du interagerar med HagiCode kan vi behandla:

- **Åtkomst- och enhetsinformation**, såsom webbläsartyp, operativsystem, besökta sidor, hänvisningslänkar, tidsstämplar, grov region och enhetsidentifierare;
- **Användningsbeteendedata**, såsom sidvisningar, tid på sidan, klick, rullningsaktivitet, sessionsinspelningar och samlad analys som används för att förstå `hagicode.com` och alla dess underdomänwebbplatser tillsammans med relaterade produktsidor;
- **Support- och kommunikationsinformation**, såsom ditt namn, e-postadress, problembeskrivning, loggutdrag eller bilagor som du frivilligt skickar via e-post, problem, community-kanaler eller formulär; och
- **Nedladdnings- och konfigurationskontext**, såsom installationsguider, nedladdningsposter, förinställda filer eller versionskommentarer som du får tillgång till för att hjälpa oss att förstå dokumentation och releasekvalitet.

## Användningsändamål

Vi behandlar den informationen för att:

- tillhandahålla och underhålla `hagicode.com` och alla dess underdomänwebbplatser, nedladdningspunkter och supportkanaler;
- förstå vilka sidor, installationssteg eller produktförklaringar som behöver förbättras;
- felsöka webbplatsfel, nedladdningsproblem, kompatibilitetsfel och säkerhetsrisker;
- svara på dina frågor, supportförfrågningar, feedback eller klagomål; och
- uppfylla tillämpliga krav på lagar, regler, tillsyn eller plattformssäkerhet när det är nödvändigt.

## Kontaktpunkter för webbplats och produktdata

Den nuvarande HagiCode `hagicode.com` egendom och alla dess underdomänwebbplatser kan röra relevant data genom:

- loggar för webbplatsåtkomst och grundläggande metadata för begäran;
- Microsoft Clarity eller jämförbar sidanalys som används för att förstå läsflöden och interaktionshotspots;
- 51LA eller jämförbar trafikanalys som används för att förstå trafiktrender och hänvisare; och
- supportinformation som du frivilligt skickar till `support@hagicode.com` eller GitHub-problem.

Om HagiCode-skrivbordsklienten, backend-tjänsterna eller tredjeparts AI-integrationer behandlar ytterligare data i separata produktsammanhang, dokumenteras dessa detaljer i relevant produktmaterial, inställningar eller tredjepartsvillkor.

## Förvaring och retention

Vi behåller data under rimliga perioder baserat på operativa behov, juridiska skyldigheter och säkerhetskrav:

- analysdata hålls i allmänhet i aggregerad form eller enligt standardlagringsreglerna för analysleverantören;
- supporte-postmeddelanden, problemrapporter och felsökningsregister sparas när de behövs för att lösa problem, granska kommunikation eller uppfylla juridiska skyldigheter; och
- vissa loggar och supportposter kan sparas längre när det är nödvändigt för säkerhet, förebyggande av missbruk, bibehållande av efterlevnad eller tvistlösning.
- vissa data kan lagras, bearbetas, överföras, säkerhetskopieras eller på annat sätt hanteras via Microsoft Azure, Alibaba Cloud eller relaterade infrastrukturfunktioner, vilket innebär att relevant data kan flyttas mellan olika länder eller regioner.

När data inte längre behövs för dessa ändamål raderar vi den, anonymiserar den eller tillåter att den löper ut enligt plattformens standardlagringsregler.

## Tredjepartstjänster

HagiCode kan förlita sig på tredje parts infrastruktur, analysleverantörer, kodvärdplattformar, e-posttjänster eller AI-tjänsteleverantörer. Dessa parter behandlar data endast efter behov för att leverera sina tjänster och förblir föremål för sina egna villkor och integritetsmeddelanden.

I nuvarande eller framtida produkt- och webbplatsverksamhet kan vi få vissa data bearbetade eller på annat sätt hanterade genom molninfrastruktur som Microsoft Azure och Alibaba Cloud. Det betyder att data kan överföras mellan olika länder eller regioner för lagring, säkerhetskopiering, logganalys, felsökning, innehållsleverans eller tjänsteorkestrering.

Där det är rimligt praktiskt använder vi relevant data i anonymiserad, aggregerad eller avidentifierad form för operationsanalys, tjänsteförbättring, stabilitetsarbete och säkerhetsskydd. Du förstår och samtycker dock till att vi inte helt kan förhindra, kontrollera eller garantera att molnleverantörer, infrastrukturvägar eller överföringar över regioner alltid kommer att förbli inom ett enda land eller en region.

När du använder tredjepartsintegrationer bör du också granska relevanta leverantörspolicyer för att förstå deras egen behandling och gränsöverskridande överföringsmetoder.

Om du inte kan acceptera denna molnbaserade bearbetning, gränsöverskridande överföring eller relaterad anonymiserad användning av data, ska du inte ladda ner, installera, komma åt eller fortsätta använda HagiCodes produkter, tjänster eller `hagicode.com` och någon av dess underdomänwebbplatser.

## Användarrättigheter

I den utsträckning som följer av tillämplig lag kan du begära att:

- veta om vi har personlig information relaterad till dig;
- korrigera felaktig eller ofullständig information;
- radera information som inte längre behövs eller som du på annat sätt har rätt att få raderad;
- begränsa eller invända mot vissa bearbetningsaktiviteter; och
- få ytterligare förklaring angående en supportkommunikation eller ett klagomål.

Om en begäran påverkar kontosäkerhet, juridiska skyldigheter, systemsäkerhet, förhindrande av missbruk eller gränser för tredjepartsplattformar, kan vi behöva verifiera identiteten först eller behålla begränsad data enligt lag.

## Policyuppdateringar

Om HagiCode-webbplatsens funktioner, analysverktyg, kontaktkanaler eller efterlevnadskrav ändras, kan vi uppdatera denna integritetspolicy. Uppdaterade versioner kommer att publiceras på denna sida med ett reviderat ikraftträdandedatum.
