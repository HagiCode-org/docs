Det här kapitlet samlar AI-tjänstprenumerationswebbplatser som för närvarande är markerade som rekommenderade i `repos/docs/public/presets/claude-code/providers/*.json`. Välj en leverantör här först, fortsätt sedan till Claude Code-konfiguration.

Från och med **9 mars 2026** omfattar detta kapitel: **ZAI, MiniMax, Alibaba Cloud DashScope och Volcengine Coding Plan**.

:::tips[Föreslagen läsordning]
Om du inte har valt en leverantör ännu, börja med den här sidan för jämförelse. Om du redan vet vilken leverantör du vill ha, hoppa till dess detaljsida först och fortsätt sedan till Claude Code-posten i den förenade AI Agent CLI-hubben.
:::

## Aktuella rekommenderade leverantörer

| Leverantör | Styrka | Bra passform för | Detaljsida |
|----------|----------|--------------|-------------|
| ZAI | Kodningsfokuserade planer med tiermapping | Utvecklare som vill ha Claude Code som daglig förare | [Visa detaljer](./zai/) |
| MiniMax | En standardmodell över nivåer, enkel introduktion | Team som optimerar för lågfriktionsinställning | [Visa detaljer](./minimax/) |
| Alibaba Cloud DashScope | Molnkonsolanpassning och enhetlig hantering | Team som redan arbetar i Alibaba Cloud | [Visa detaljer](./aliyun/) |
| Volcengine kodningsplan | Starka ekosystemförlängningar och multimodellplattformssignaler | Utvecklare som vill ha Volcengine ekosystemanpassning | [Visa detaljer](./volcengine/) |

:::obs
`Anthropic Official` förblir användbar, men den ingår inte i den första omgången av dedikerade sidor eftersom den aktuella förinställda metadatan inte markerar den som rekommenderad.
:::

## Snabbvalstips

- **Behöver kodningsfokuserade planer och tydligare nivåkartläggning**: börja med [ZAI](./zai/)
- **Behöver den enklaste installationsvägen först**: börja med [MiniMax](./minimax/)
- **Kör redan i Alibaba Cloud**: börja med [Alibaba Cloud DashScope](./aliyun/)
- **Vill ha Volcengine-ekosystemanpassning eller räckvidd för flera modeller av plattformar**: börja med [Volcengine kodningsplan](./volcengine/)
