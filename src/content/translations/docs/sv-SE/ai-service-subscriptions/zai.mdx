## Positionering

ZAI är en av de leverantörer som för närvarande rekommenderas av dokumentförinställningen för Claude Code. GLM Coding planer med tier mapping för utvecklare som vill ha Claude Code som en daglig förare.

## Bra passform för

- Du vill ha en kodningsfokuserad prenumerationsplan
- Du behöver mappade Sonnet/Opus/Haiku-standardvärden
- Du förväntar dig att en leverantör fungerar med Claude Code och liknande kodningsverktyg

## Viktiga saker att lägga märke till

- Den nuvarande förinställda metadatan markerar ZAI som rekommenderad.
- Den avslöjar en antropisk-kompatibel slutpunkt som kan användas direkt av Claude Code.
- Förinställningen levererar standardmappningar för `glm-4.7`, `glm-5`, och `glm-4.5-air`.

## Integration ögonblicksbild

| Objekt | Värde |
|------|-------|
| Prenumerationsinträde | [Öppna prenumerationssidan](https://www.bigmodel.cn/claude-code?ic=14BY54APZA) |
| Officiella dokument/konsol | [Öppna den officiella sidan](https://open.bigmodel.cn/dev/api) |
| Antropisk-kompatibel bas-URL | `https://open.bigmodel.cn/api/anthropic` |
| Rekommenderad auth env var | `ANTHROPIC_AUTH_TOKEN` |

## Standardmodellmappning

| Claude tier | Standardmodell |
|-------------|---------------|
| Sonett | `glm-4.7` |
| Opus | `glm-5` |
| Haiku | `glm-4.5-air` |

:::tips
Priser, kampanjer och paketdetaljer kan ändras över tiden. Om kostnaden är ditt största problem, bekräfta den senaste informationen direkt på leverantörssidan.
:::

## Föreslagna nästa steg

1. Bekräfta vilken prenumerationsnivå eller API-aktiveringsväg du vill använda.
2. Öppna Claude Code-posten i den förenade AI Agent CLI-hubben och fortsätt med Anthropics officiella installations- och referensdokument.
3. Validera anslutningen lokalt efter installationen.
