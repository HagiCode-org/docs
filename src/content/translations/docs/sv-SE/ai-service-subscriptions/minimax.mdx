## Positionering

MiniMax är en av de leverantörer som för närvarande rekommenderas av dokumentförinställningen för Claude Code. En enda standardmodell över nivåer, lämplig för team som värdesätter enkel konfiguration och förutsägbar onboarding.

## Bra passform för

- Du vill ha samma mappade modell över alla Claude-nivåer
- Du föredrar installation och underhåll med låg friktion
- Du vill få Claude Code igång snabbt innan du ställer in modellstrategin

## Viktiga saker att lägga märke till

- Den aktuella förinställda metadatan markerar MiniMax som rekommenderad.
- Den använder `MiniMax-M2.5` som standardmappad modell för alla nivåer.
- Detta passar bra när operativ enkelhet är viktigare än nivåspecifik inställning.

## Integration ögonblicksbild

| Objekt | Värde |
|------|-------|
| Prenumerationsinträde | [Öppna prenumerationssidan](https://platform.minimaxi.com/subscribe/coding-plan?code=8wNck4etDM&source=link) |
| Officiella dokument/konsol | [Öppna den officiella sidan](https://www.minimaxi.com) |
| Antropisk-kompatibel bas-URL | `https://api.minimaxi.com/anthropic` |
| Rekommenderad auth env var | `ANTHROPIC_AUTH_TOKEN` |

## Standardmodellmappning

| Claude tier | Standardmodell |
|-------------|---------------|
| Sonett | `MiniMax-M2.5` |
| Opus | `MiniMax-M2.5` |
| Haiku | `MiniMax-M2.5` |

:::tips
Priser, kampanjer och paketdetaljer kan ändras över tiden. Om kostnaden är ditt största problem, bekräfta den senaste informationen direkt på leverantörssidan.
:::

## Föreslagna nästa steg

1. Bekräfta vilken prenumerationsnivå eller API-aktiveringsväg du vill använda.
2. Öppna Claude Code-posten i den förenade AI Agent CLI-hubben och fortsätt med Anthropics officiella installations- och referensdokument.
3. Validera anslutningen lokalt efter installationen.
