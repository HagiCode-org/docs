HagiCode รองรับเส้นทางการติดตั้งหลายเส้นทาง เริ่มต้นด้วยวิธีการปรับใช้ที่เหมาะกับสถานการณ์ของคุณ จากนั้นเพิ่มซอฟต์แวร์ที่รองรับเมื่อคุณต้องการเท่านั้น

ช่วงการสนับสนุน Agent CLI ที่ใช้งานอยู่ในปัจจุบันที่เกี่ยวข้องกับการตั้งค่ามากที่สุดคือ: Claude Code, Codex, GitHub Copilot, OpenCode, Hermes, QoderCLI, Kiro, Kimi, Gemini, DeepAgents และ Codebuddy ปัจจุบัน Hermes, Gemini และ DeepAgents ใช้ขั้นตอนการติดตั้งหรือการรับรองความถูกต้องด้วยตนเอง CLI อื่นๆ เปิดเผยจุดเข้าติดตั้งในคลิกเดียวหรือขั้นตอนการติดตั้งที่แนะนำ

## แนะนำขั้นตอนถัดไป

- หากคุณวางแผนที่จะซื้อและติดตั้ง HagiCode ผ่าน Steam ให้เริ่มต้นด้วย [การติดตั้งไอน้ำ](/th-TH/installation/steam).
- หากคุณยังใหม่กับ HagiCode ให้เริ่มต้นด้วย [การติดตั้งเดสก์ท็อป](/th-TH/installation/desktop).
- หากคุณกำลังปรับใช้สำหรับสภาพแวดล้อมเซิร์ฟเวอร์หรือทีม ให้ใช้ [การติดตั้งนักเทียบท่าเขียน](/th-TH/installation/docker-compose).
- หากคุณต้องการยืนยันสถานะการสนับสนุน โหมดการตั้งค่า หรือบันทึกความเข้ากันได้ในอดีตสำหรับ CLI ให้เปิด [การติดตั้ง AI Agent CLI](/th-TH/related-software-installation/ai-agent-cli/).
