## การวางตำแหน่ง

MiniMax เป็นหนึ่งในผู้ให้บริการที่แนะนำในปัจจุบันโดยเอกสารที่ตั้งไว้ล่วงหน้าสำหรับรหัส Claude โมเดลเริ่มต้นเดียวสำหรับทุกระดับ เหมาะสำหรับทีมที่ให้ความสำคัญกับการกำหนดค่าที่เรียบง่ายและการเริ่มต้นใช้งานที่คาดการณ์ได้

## เหมาะสำหรับ

- คุณต้องการโมเดลที่แมปเดียวกันในทุกระดับของ Claude
- คุณชอบการตั้งค่าและการบำรุงรักษาที่มีแรงเสียดทานต่ำ
- คุณต้องการให้ Claude Code ทำงานอย่างรวดเร็วก่อนที่จะปรับแต่งกลยุทธ์โมเดล

## สิ่งสำคัญที่ควรสังเกต

- ข้อมูลเมตาที่ตั้งไว้ล่วงหน้าปัจจุบันทำเครื่องหมาย MiniMax ตามที่แนะนำ
- มันใช้ `MiniMax-M2.5` เป็นโมเดลแมปเริ่มต้นสำหรับทุกระดับ
- ซึ่งเหมาะอย่างยิ่งเมื่อความเรียบง่ายในการปฏิบัติงานมีความสำคัญมากกว่าการปรับแต่งเฉพาะระดับ

## ภาพรวมการบูรณาการ

| รายการ | ความคุ้มค่า |
|------|-------|
| รายการสมัครสมาชิก | [เปิดหน้าสมัครสมาชิก](https://platform.minimaxi.com/subscribe/coding-plan?code=8wNck4etDM&source=link) |
| เอกสาร / คอนโซลอย่างเป็นทางการ | [เปิดเพจอย่างเป็นทางการ](https://www.minimaxi.com) |
| URL พื้นฐานที่เข้ากันได้กับมานุษยวิทยา | `https://api.minimaxi.com/anthropic` |
| การรับรองความถูกต้อง env var ที่แนะนำ | `ANTHROPIC_AUTH_TOKEN` |

| ระดับโคลด์ | โมเดลเริ่มต้น |
|-------------|---------------|
| โคลง | `MiniMax-M2.5` |
| บทประพันธ์ | `MiniMax-M2.5` |
| ไฮกุ | `MiniMax-M2.5` |

:::คำแนะนำ
ราคา โปรโมชั่น และรายละเอียดแพ็กเกจสามารถเปลี่ยนแปลงได้เมื่อเวลาผ่านไป หากค่าใช้จ่ายคือข้อกังวลหลักของคุณ ให้ยืนยันข้อมูลล่าสุดได้โดยตรงจากหน้าผู้ให้บริการ
:::

## แนะนำขั้นตอนต่อไป

1. ยืนยันว่าระดับการสมัครสมาชิกหรือเส้นทางการเปิดใช้งาน API ใดที่คุณต้องการใช้
2. เปิดรายการรหัส Claude ในฮับ AI Agent CLI แบบรวม และดำเนินการต่อด้วยเอกสารการติดตั้งและอ้างอิงอย่างเป็นทางการของ Anthropic
3. ตรวจสอบการเชื่อมต่อภายในเครื่องหลังการตั้งค่า
