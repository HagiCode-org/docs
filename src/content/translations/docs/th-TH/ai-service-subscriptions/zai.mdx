## การวางตำแหน่ง

ZAI เป็นหนึ่งในผู้ให้บริการที่แนะนำโดยเอกสารที่ตั้งไว้ล่วงหน้าสำหรับ Claude Code แผน GLM Coding พร้อมการแมประดับสำหรับนักพัฒนาที่ต้องการให้ Claude Code เป็นตัวขับเคลื่อนรายวัน

## เหมาะสำหรับ

- คุณต้องการแผนการสมัครสมาชิกที่เน้นการเขียนโค้ด
- คุณต้องมีค่าเริ่มต้น Sonnet/Opus/Haiku ที่แมปไว้
- คุณคาดหวังว่าผู้ให้บริการรายหนึ่งจะทำงานกับ Claude Code และเครื่องมือการเขียนโค้ดที่คล้ายกัน

## สิ่งสำคัญที่ควรสังเกต

- ข้อมูลเมตาที่ตั้งไว้ล่วงหน้าปัจจุบันทำเครื่องหมาย ZAI ตามที่แนะนำ
- โดยจะเปิดเผยอุปกรณ์ปลายทางที่เข้ากันได้กับ Anthropic ซึ่ง Claude Code สามารถใช้ได้โดยตรง
- ค่าที่ตั้งล่วงหน้าจัดส่งการแมปเริ่มต้นสำหรับ `glm-4.7`, `glm-5`และ `glm-4.5-air`.

## ภาพรวมการบูรณาการ

| รายการ | ความคุ้มค่า |
|------|-------|
| รายการสมัครสมาชิก | [เปิดหน้าสมัครสมาชิก](https://www.bigmodel.cn/claude-code?ic=14BY54APZA) |
| เอกสาร / คอนโซลอย่างเป็นทางการ | [เปิดเพจอย่างเป็นทางการ](https://open.bigmodel.cn/dev/api) |
| URL พื้นฐานที่เข้ากันได้กับมานุษยวิทยา | `https://open.bigmodel.cn/api/anthropic` |
| การรับรองความถูกต้อง env var ที่แนะนำ | `ANTHROPIC_AUTH_TOKEN` |

## การแม็ปแบบจำลองเริ่มต้น

| ระดับโคลด์ | โมเดลเริ่มต้น |
|-------------|---------------|
| โคลง | `glm-4.7` |
| บทประพันธ์ | `glm-5` |
| ไฮกุ | `glm-4.5-air` |

:::คำแนะนำ
ราคา โปรโมชั่น และรายละเอียดแพ็กเกจสามารถเปลี่ยนแปลงได้เมื่อเวลาผ่านไป หากค่าใช้จ่ายคือข้อกังวลหลักของคุณ ให้ยืนยันข้อมูลล่าสุดได้โดยตรงจากหน้าผู้ให้บริการ
:::

## แนะนำขั้นตอนต่อไป

1. ยืนยันว่าระดับการสมัครสมาชิกหรือเส้นทางการเปิดใช้งาน API ใดที่คุณต้องการใช้
2. เปิดรายการรหัส Claude ในฮับ AI Agent CLI แบบรวม และดำเนินการต่อด้วยเอกสารการติดตั้งและอ้างอิงอย่างเป็นทางการของ Anthropic
3. ตรวจสอบการเชื่อมต่อภายในเครื่องหลังการตั้งค่า
