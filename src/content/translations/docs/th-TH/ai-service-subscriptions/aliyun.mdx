## การวางตำแหน่ง

Alibaba Cloud DashScope เป็นหนึ่งในผู้ให้บริการที่แนะนำโดยเอกสารที่กำหนดไว้ล่วงหน้าสำหรับ Claude Code ตัวเลือกที่เป็นประโยชน์สำหรับทีมที่ดำเนินงานอยู่ในระบบนิเวศของ Alibaba Cloud และต้องการการจัดการคอนโซลแบบรวมเป็นหนึ่ง

## เหมาะสำหรับ

- คุณใช้บัญชีหรือทรัพยากรของ Alibaba Cloud อยู่แล้ว
- คุณต้องการให้การสมัครสมาชิกและการจัดการคีย์อยู่ในคอนโซลคลาวด์เดียว
- คุณชอบโมเดลที่แมปแบบตายตัวเพื่อการออนบอร์ดอย่างรวดเร็ว

## สิ่งสำคัญที่ควรสังเกต

- เมตาดาต้าที่ตั้งไว้ล่วงหน้าปัจจุบันทำเครื่องหมาย Alibaba Cloud DashScope ตามที่แนะนำ
- โดยจะแมปทุกระดับของ Claude `glm-4.7` เพื่อเส้นทางการตั้งค่าที่ตรงไปตรงมา
- จะสะดวกเป็นพิเศษหากทีมของคุณทำงานในเวิร์กโฟลว์ของ Alibaba Cloud อยู่แล้ว

## ภาพรวมการบูรณาการ

| รายการ | ความคุ้มค่า |
|------|-------|
| รายการสมัครสมาชิก | [เปิดหน้าสมัครสมาชิก](https://www.aliyun.com/benefit/ai/aistar?userCode=vmx5szbq&clubBiz=subTask..12384055..10263..) |
| เอกสาร / คอนโซลอย่างเป็นทางการ | [เปิดเพจอย่างเป็นทางการ](https://dashscope.aliyun.com) |
| URL พื้นฐานที่เข้ากันได้กับมานุษยวิทยา | `https://coding.dashscope.aliyuncs.com/apps/anthropic` |
| การรับรองความถูกต้อง env var ที่แนะนำ | `ANTHROPIC_AUTH_TOKEN` |

|-------------|---------------|
| โคลง | `glm-4.7` |
| บทประพันธ์ | `glm-4.7` |
| ไฮกุ | `glm-4.7` |

:::คำแนะนำ
ราคา โปรโมชั่น และรายละเอียดแพ็กเกจสามารถเปลี่ยนแปลงได้เมื่อเวลาผ่านไป หากค่าใช้จ่ายคือข้อกังวลหลักของคุณ ให้ยืนยันข้อมูลล่าสุดได้โดยตรงจากหน้าผู้ให้บริการ
:::

## แนะนำขั้นตอนต่อไป

1. ยืนยันว่าระดับการสมัครสมาชิกหรือเส้นทางการเปิดใช้งาน API ใดที่คุณต้องการใช้
2. เปิดรายการรหัส Claude ในฮับ AI Agent CLI แบบรวม และดำเนินการต่อด้วยเอกสารการติดตั้งและอ้างอิงอย่างเป็นทางการของ Anthropic
3. ตรวจสอบการเชื่อมต่อภายในเครื่องหลังการตั้งค่า
