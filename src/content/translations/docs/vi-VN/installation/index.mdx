HagiCode hỗ trợ nhiều đường dẫn cài đặt. Bắt đầu với phương pháp triển khai phù hợp với kịch bản của bạn, sau đó chỉ thêm phần mềm hỗ trợ khi bạn cần.

Phạm vi hỗ trợ CLI của Agent đang hoạt động hiện tại phù hợp nhất để thiết lập là: Claude Code, Codex, GitHub Copilot, OpenCode, Hermes, QoderCLI, Kiro, Kimi, Gemini, DeepAgents và Codebuddy. Hermes, Gemini và DeepAgents hiện sử dụng các quy trình xác thực hoặc cài đặt thủ công; các CLI khác hiển thị các điểm nhập cài đặt bằng một cú nhấp chuột hoặc các luồng cài đặt được hướng dẫn.

## Các bước tiếp theo được đề xuất

- Nếu bạn dự định mua và cài đặt HagiCode thông qua Steam, hãy bắt đầu với [Cài đặt hơi nước](/vi-VN/installation/steam).
- Nếu bạn chưa quen với HagiCode, hãy bắt đầu với [Cài đặt máy tính để bàn](/vi-VN/installation/desktop).
- Nếu bạn đang triển khai cho môi trường máy chủ hoặc nhóm, hãy sử dụng [Cài đặt soạn thảo Docker](/vi-VN/installation/docker-compose).
- Nếu bạn cần xác nhận trạng thái hỗ trợ, chế độ thiết lập hoặc ghi chú về khả năng tương thích lịch sử cho CLI, hãy mở [Cài đặt AI Agent CLI](/vi-VN/related-software-installation/ai-agent-cli/).
