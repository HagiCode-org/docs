## Posizionamento

ZAI è uno dei fornitori attualmente consigliati dal set di documenti preimpostato per Claude Code. Piani di GLM Coding con mappatura dei livelli per gli sviluppatori che desiderano Claude Code come driver quotidiano.

## Buona vestibilità per

- Desideri un piano di abbonamento incentrato sulla codifica
- Sono necessarie le impostazioni predefinite di Sonnet/Opus/Haiku mappate
- Ti aspetti che un fornitore funzioni con Claude Code e strumenti di codifica simili

## Cose fondamentali da notare

- I metadati preimpostati correnti contrassegnano ZAI come consigliato.
- Espone un endpoint compatibile con l'ambiente antropico che può essere utilizzato direttamente da Claude Code.
- La preimpostazione fornisce mappature predefinite per `glm-4.7`, `glm-5`, e `glm-4.5-air`.

## Istantanea dell'integrazione

| Articolo | Valore |
|------|-------|
| Voce di abbonamento | [Apri la pagina di iscrizione](https://www.bigmodel.cn/claude-code?ic=14BY54APZA) |
| Documenti/console ufficiali | [Apri la pagina ufficiale](https://open.bigmodel.cn/dev/api) |
| URL di base compatibile con gli antropici | `https://open.bigmodel.cn/api/anthropic` |
| Var. ambiente autenticazione consigliata | `ANTHROPIC_AUTH_TOKEN` |

## Mappatura del modello predefinita

| Livello Claude | Modello predefinito |
|-------------|---------------|
| Sonetto | `glm-4.7` |
| Opus | `glm-5` |
| Haiku | `glm-4.5-air` |

:::suggerimento
Prezzi, promozioni e dettagli del pacchetto possono cambiare nel tempo. Se il costo è la tua preoccupazione principale, conferma le informazioni più recenti direttamente sulla pagina del fornitore.
:::

## Passaggi successivi suggeriti

1. Conferma quale livello di abbonamento o percorso di attivazione API desideri utilizzare.
2. Apri la voce Codice Claude nell'hub CLI dell'agente AI unificato e continua con l'installazione ufficiale e i documenti di riferimento di Anthropic.
3. Convalidare la connessione localmente dopo la configurazione.
