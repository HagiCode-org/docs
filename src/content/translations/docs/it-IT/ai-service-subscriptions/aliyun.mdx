## Posizionamento

Alibaba Cloud DashScope è uno dei fornitori attualmente consigliati dal set di documenti preimpostato per Claude Code. Una scelta pratica per i team che già operano nell'ecosistema Alibaba Cloud e desiderano una gestione unificata della console.

## Buona vestibilità per

- Utilizzi già account o risorse Alibaba Cloud
- Desideri che l'abbonamento e la gestione delle chiavi rimangano in un'unica console cloud
- Preferisci un modello a mappatura fissa per un onboarding rapido

## Cose fondamentali da notare

- Gli attuali metadati preimpostati contrassegnano Alibaba Cloud DashScope come consigliato.
- Mappa tutti i livelli Claude su `glm-4.7` per un percorso di installazione semplice.
- È particolarmente conveniente se il tuo team lavora già all'interno dei flussi di lavoro Alibaba Cloud.

## Passaggi successivi suggeriti

1. Conferma quale livello di abbonamento o percorso di attivazione API desideri utilizzare.
2. Apri la voce Codice Claude nell'hub CLI dell'agente AI unificato e continua con l'installazione ufficiale e i documenti di riferimento di Anthropic.
3. Convalidare la connessione localmente dopo la configurazione.
