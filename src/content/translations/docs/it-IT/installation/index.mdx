HagiCode supporta più percorsi di installazione. Inizia con il metodo di distribuzione adatto al tuo scenario, quindi aggiungi il software di supporto solo quando ne hai bisogno.

L'attuale gamma di supporto CLI dell'agente attivo più rilevante per la configurazione è: Claude Code, Codex, GitHub Copilot, OpenCode, Hermes, QoderCLI, Kiro, Kimi, Gemini, DeepAgents e Codebuddy. Hermes, Gemini e DeepAgents attualmente utilizzano flussi di installazione o autenticazione manuali; le altre CLI espongono punti di ingresso per l'installazione con un clic o flussi di installazione guidati.

## Passaggi successivi consigliati

- Se prevedi di acquistare e installare HagiCode tramite Steam, inizia con [Installazione di Steam](/it-IT/installation/steam).
- Se sei nuovo a HagiCode, inizia con [Installazione sul desktop](/it-IT/installation/desktop).
- Se stai distribuendo per un ambiente server o team, utilizza [Installazione di Docker Compose](/it-IT/installation/docker-compose).
- Se devi confermare lo stato del supporto, la modalità di configurazione o le note sulla compatibilità storica per una CLI, apri [Installazione della CLI dell'agente AI](/it-IT/related-software-installation/ai-agent-cli/).
