## Cosa tratta questa sezione

La sezione FAQ è il posto dove:

- avvertenze sull'avvio e sul runtime
- note di compatibilità legate a piattaforme, canali di distribuzione, pacchetti o runtime in bundle
- argomenti per la risoluzione dei problemi che meritano una pagina di spiegazione autonoma e stabile

Se è necessario confermare quale Node Desktop utilizza o perché viene menzionato dalla diagnostica `portable-fixed/toolchain`, aperto [Ambiente del nodo desktop](/it-IT/faq/desktop-node-environment).

Se vuoi sapere perché HagiCode non è su Steam, apri [Perché HagiCode non è disponibile su Steam](/it-IT/faq/steam-distribution-status).
