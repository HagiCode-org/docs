Questa pagina è la voce di installazione della CLI AI Agent locale unificata in `repos/docs`. HagiCode attualmente supporta 11 CLI dell'agente attive, seguendo il backend `PrimaryProfessions` ordine: Claude Code, Codex, GitHub Copilot, OpenCode, Hermes, QoderCLI, Kiro, Kimi, Gemini, DeepAgents e Codebuddy.

:::suggerimento[Come utilizzare questa pagina]
Controlla prima la tabella per lo stato attuale del supporto e la modalità di configurazione, quindi continua con i documenti di installazione, autenticazione, ACP, autorizzazione e risoluzione dei problemi gestiti dal fornitore. L'ambito del supporto e la modalità di configurazione sono questioni separate.
:::

## Glossario di stato

- **Attualmente supportato** significa che la CLI appartiene all'attuale gamma di supporto HagiCode attivo.
- **Installazione con un clic** significa che HagiCode fornisce un punto di ingresso guidato per l'installazione; alcune CLI richiedono ancora l'accesso del fornitore o una fase di autenticazione dopo l'installazione.
- **Installazione/autenticazione manuale** significa che la CLI è ora supportata, ma è necessario completare l'installazione locale o il flusso di autenticazione prima che HagiCode possa rilevarla e utilizzarla.
- **Solo compatibilità legacy** indica un valore ritirato come `IFlowCli` esiste ancora per la compatibilità dei dati storici, ma non fa parte dell'attuale intervallo di supporto attivo.
