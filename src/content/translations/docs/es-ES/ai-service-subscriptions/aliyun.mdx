## Posicionamiento

Alibaba Cloud DashScope es uno de los proveedores recomendados actualmente por el conjunto de preajustes de la documentación para Claude Code. Una opción práctica para los equipos que ya operan en el ecosistema de Alibaba Cloud y desean una gestión unificada desde la consola.

## Adecuado para

- Ya utilizas cuentas o recursos de Alibaba Cloud
- Quieres que la gestión de suscripciones y claves se mantenga en una única consola en la nube
- Prefieres un modelo asignado fijo para una incorporación rápida

## Aspectos clave a tener en cuenta

- Los metadatos del preajuste actual marcan Alibaba Cloud DashScope como recomendado.
- Asigna todos los niveles de Claude a `glm-4.7` para una ruta de configuración sencilla.
- Es especialmente conveniente si tu equipo ya trabaja dentro de los flujos de trabajo de Alibaba Cloud.

## Instantánea de la integración

| Elemento | Valor |
|------|-------|
| Entrada de suscripción | [Abrir página de suscripción](https://www.aliyun.com/benefit/ai/aistar?userCode=vmx5szbq&clubBiz=subTask..12384055..10263..) |
| Documentación oficial / consola | [Abrir página oficial](https://dashscope.aliyun.com) |
| URL base compatible con Anthropic | `https://coding.dashscope.aliyuncs.com/apps/anthropic` |
| Variable de entorno de autenticación recomendada | `ANTHROPIC_AUTH_TOKEN` |

## Asignación de modelo predeterminada

| Nivel de Claude | Modelo predeterminado |
|-------------|---------------|
| Sonnet | `glm-4.7` |
| Opus | `glm-4.7` |
| Haiku | `glm-4.7` |

:::tip
Los precios, las promociones y los detalles de los paquetes pueden cambiar con el tiempo. Si el coste es tu principal preocupación, confirma la información más reciente directamente en la página del proveedor.
:::

## Pasos siguientes sugeridos

1. Confirma qué nivel de suscripción o ruta de activación de API deseas utilizar.
2. Abre la entrada de Claude Code en el centro unificado de CLI de agentes de IA y continúa con la documentación oficial de instalación y referencia de Anthropic.
3. Valida la conexión localmente después de la configuración.
