## Penentuan posisi

ZAI adalah salah satu penyedia yang saat ini direkomendasikan oleh preset dokumen yang ditetapkan untuk Claude Code. GLM Coding berencana dengan pemetaan tingkatan untuk pengembang yang menginginkan Claude Code sebagai driver harian.

## Cocok untuk

- Anda menginginkan paket berlangganan yang berfokus pada pengkodean
- Anda perlu memetakan default Soneta/Opus/Haiku
- Anda mengharapkan satu penyedia untuk bekerja di Claude Code dan alat pengkodean serupa

## Hal-hal penting yang perlu diperhatikan

- Metadata prasetel saat ini menandai ZAI sebagai yang direkomendasikan.
- Ini memperlihatkan titik akhir yang kompatibel dengan Antropik yang dapat digunakan langsung oleh Claude Code.
- Prasetel mengirimkan pemetaan default untuk `glm-4.7`, `glm-5`, dan `glm-4.5-air`.

## Cuplikan integrasi

| Barang | Nilai |
|------|-------|
| Entri berlangganan | [Buka halaman berlangganan](https://www.bigmodel.cn/claude-code?ic=14BY54APZA) |
| Dokumen/konsol resmi | [Buka halaman resmi](https://open.bigmodel.cn/dev/api) |
| URL dasar yang kompatibel dengan antropik | `https://open.bigmodel.cn/api/anthropic` |
| Direkomendasikan auth env var | `ANTHROPIC_AUTH_TOKEN` |

## Pemetaan model default

| Tingkat Claude | Model bawaan |
|-------------|---------------|
| Soneta | `glm-4.7` |
| Karya | `glm-5` |
| Haiku | `glm-4.5-air` |

:::tip
Detail harga, promosi, dan paket dapat berubah seiring waktu. Jika biaya menjadi perhatian utama Anda, konfirmasikan informasi terbaru langsung di halaman penyedia.
:::

## Disarankan langkah selanjutnya

1. Konfirmasikan tingkat langganan atau jalur aktivasi API mana yang ingin Anda gunakan.
2. Buka entri Kode Claude di hub AI Agen CLI terpadu dan lanjutkan dengan dokumen referensi dan instalasi resmi Anthropic.
3. Validasi koneksi secara lokal setelah pengaturan.
