Bab ini mengumpulkan situs web berlangganan layanan AI yang saat ini ditandai sebagai direkomendasikan `repos/docs/public/presets/claude-code/providers/*.json`. Pilih dulu penyedianya di sini, lalu lanjutkan ke konfigurasi Claude Code.

Mulai **9 Maret 2026**, bab ini mencakup: **ZAI, MiniMax, Alibaba Cloud DashScope, dan Volcengine Coding Plan**.

:::tip[Urutan bacaan yang disarankan]
Jika Anda belum memilih penyedia layanan, mulailah dengan halaman ini untuk perbandingan. Jika Anda sudah mengetahui penyedia mana yang Anda inginkan, buka halaman detailnya terlebih dahulu lalu lanjutkan ke entri Claude Code di hub AI Agent CLI terpadu.
:::

## Penyedia yang direkomendasikan saat ini

| Penyedia | Kekuatan | Cocok untuk | Halaman detail |
|----------|----------|--------------|-------------|
| ZAI | Rencana yang berfokus pada pengkodean dengan pemetaan tingkat | Developer yang menginginkan Claude Code sebagai daily driver | [Lihat detailnya](./zai/) |
| Mini Maks | Satu model default di seluruh tingkatan, orientasi sederhana | Tim mengoptimalkan pengaturan dengan gesekan rendah | [Lihat detailnya](./minimax/) |
| Lingkup Dasbor Alibaba Cloud | Penyelarasan konsol cloud dan manajemen terpadu | Tim sudah bekerja di Alibaba Cloud | [Lihat detailnya](./aliyun/) |
| Rencana Pengkodean Volcengine | Perluasan ekosistem yang kuat dan sinyal platform multi-model | Pengembang yang menginginkan penyelarasan ekosistem Volcengine | [Lihat detailnya](./volcengine/) |

:::catatan
`Anthropic Official` tetap dapat digunakan, tetapi tidak disertakan dalam kumpulan halaman khusus pertama karena metadata preset saat ini tidak menandainya sebagai direkomendasikan.
:::

## Kiat pemilihan cepat

- **Membutuhkan rencana yang berfokus pada pengkodean dan pemetaan tingkat yang lebih jelas**: mulailah dengan [ZAI](./zai/)
- **Perlu jalur penyiapan termudah terlebih dahulu**: mulai dengan [Mini Maks](./minimax/)
- **Sudah berjalan di dalam Alibaba Cloud**: mulai dengan [Lingkup Dasbor Alibaba Cloud](./aliyun/)
- **Ingin penyelarasan ekosistem Volcengine atau jangkauan platform multi-model**: mulai dengan [Rencana Pengkodean Volcengine](./volcengine/)
