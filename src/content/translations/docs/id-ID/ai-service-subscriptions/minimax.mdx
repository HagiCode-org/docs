## Penentuan posisi

MiniMax adalah salah satu penyedia yang saat ini direkomendasikan oleh preset dokumen yang disetel untuk Claude Code. Model default tunggal di seluruh tingkatan, cocok untuk tim yang menghargai konfigurasi sederhana dan orientasi yang dapat diprediksi.

## Cocok untuk

- Anda ingin model yang dipetakan sama di semua tingkatan Claude
- Anda lebih memilih pengaturan dan pemeliharaan dengan gesekan rendah
- Anda ingin Kode Claude berjalan cepat sebelum menyetel strategi model

## Hal-hal penting yang perlu diperhatikan

- Metadata prasetel saat ini menandai MiniMax sebagai yang direkomendasikan.
- Ini menggunakan `MiniMax-M2.5` sebagai model yang dipetakan secara default untuk semua tingkatan.
- Ini sangat cocok ketika kesederhanaan operasional lebih penting daripada penyetelan tingkat tertentu.

## Cuplikan integrasi

| Barang | Nilai |
|------|-------|
| Entri berlangganan | [Buka halaman berlangganan](https://platform.minimaxi.com/subscribe/coding-plan?code=8wNck4etDM&source=link) |
| Dokumen/konsol resmi | [Buka halaman resmi](https://www.minimaxi.com) |
| URL dasar yang kompatibel dengan antropik | `https://api.minimaxi.com/anthropic` |
| Direkomendasikan auth env var | `ANTHROPIC_AUTH_TOKEN` |

## Pemetaan model default

| Tingkat Claude | Model bawaan |
|-------------|---------------|
| Soneta | `MiniMax-M2.5` |
| Karya | `MiniMax-M2.5` |
| Haiku | `MiniMax-M2.5` |

## Disarankan langkah selanjutnya

1. Konfirmasikan tingkat langganan atau jalur aktivasi API mana yang ingin Anda gunakan.
2. Buka entri Kode Claude di hub AI Agen CLI terpadu dan lanjutkan dengan dokumen referensi dan instalasi resmi Anthropic.
3. Validasi koneksi secara lokal setelah pengaturan.
