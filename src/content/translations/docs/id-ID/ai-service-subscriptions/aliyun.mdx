## Penentuan posisi

Alibaba Cloud DashScope adalah salah satu penyedia yang saat ini direkomendasikan oleh preset dokumen yang disetel untuk Claude Code. Pilihan praktis bagi tim yang sudah beroperasi di ekosistem Alibaba Cloud dan menginginkan manajemen konsol terpadu.

## Cocok untuk

- Anda sudah menggunakan akun atau sumber daya Alibaba Cloud
- Anda ingin langganan dan manajemen kunci tetap berada di satu konsol cloud
- Anda lebih memilih model yang dipetakan tetap untuk orientasi cepat

## Hal-hal penting yang perlu diperhatikan

- Metadata preset saat ini menandai Alibaba Cloud DashScope sebagai yang direkomendasikan.
- Ini memetakan semua tingkatan Claude `glm-4.7` untuk jalur pengaturan yang mudah.
- Hal ini sangat berguna jika tim Anda sudah bekerja dalam alur kerja Alibaba Cloud.

## Cuplikan integrasi

| Barang | Nilai |
|------|-------|
| Entri berlangganan | [Buka halaman berlangganan](https://www.aliyun.com/benefit/ai/aistar?userCode=vmx5szbq&clubBiz=subTask..12384055..10263..) |
| Dokumen/konsol resmi | [Buka halaman resmi](https://dashscope.aliyun.com) |
| URL dasar yang kompatibel dengan antropik | `https://coding.dashscope.aliyuncs.com/apps/anthropic` |
| Direkomendasikan auth env var | `ANTHROPIC_AUTH_TOKEN` |

## Pemetaan model default

| Tingkat Claude | Model bawaan |
|-------------|---------------|
| Soneta | `glm-4.7` |
| Karya | `glm-4.7` |
| Haiku | `glm-4.7` |

:::tip
Detail harga, promosi, dan paket dapat berubah seiring waktu. Jika biaya menjadi perhatian utama Anda, konfirmasikan informasi terbaru langsung di halaman penyedia.
:::

## Disarankan langkah selanjutnya

1. Konfirmasikan tingkat langganan atau jalur aktivasi API mana yang ingin Anda gunakan.
2. Buka entri Kode Claude di hub AI Agen CLI terpadu dan lanjutkan dengan dokumen referensi dan instalasi resmi Anthropic.
3. Validasi koneksi secara lokal setelah pengaturan.
