HagiCode mendukung beberapa jalur instalasi. Mulailah dengan metode penerapan yang sesuai dengan skenario Anda, lalu tambahkan perangkat lunak pendukung hanya saat Anda membutuhkannya.

Rentang dukungan Agen CLI aktif saat ini yang paling relevan untuk penyiapan adalah: Claude Code, Codex, GitHub Copilot, OpenCode, Hermes, QoderCLI, Kiro, Kimi, Gemini, DeepAgents, dan Codebuddy. Hermes, Gemini, dan DeepAgents saat ini menggunakan instalasi manual atau alur otentikasi; CLI lainnya menampilkan titik masuk instalasi sekali klik atau alur instalasi yang dipandu.

## Langkah Berikutnya yang Direkomendasikan

- Jika Anda berencana membeli dan menginstal HagiCode melalui Steam, mulailah dengan [Instalasi Uap](/id-ID/installation/steam).
- Jika Anda baru mengenal HagiCode, mulailah dengan [Instalasi Desktop](/id-ID/installation/desktop).
- Jika Anda menerapkan untuk server atau lingkungan tim, gunakan [Instalasi Penulisan Docker](/id-ID/installation/docker-compose).
- Jika Anda perlu mengonfirmasi status dukungan, mode pengaturan, atau catatan kompatibilitas historis untuk CLI, buka [Instalasi CLI Agen AI](/id-ID/related-software-installation/ai-agent-cli/).
