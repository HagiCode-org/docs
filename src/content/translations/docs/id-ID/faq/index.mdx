## Apa yang Dicakup Bagian Ini

Bagian FAQ adalah tempat untuk:

- peringatan startup dan runtime
- catatan kompatibilitas yang terkait dengan platform, saluran distribusi, pengemasan, atau runtime yang dibundel
- topik pemecahan masalah yang layak mendapatkan halaman penjelasan mandiri yang stabil

Jika Anda perlu mengonfirmasi Node Desktop mana yang digunakan atau mengapa diagnostik disebutkan `portable-fixed/toolchain`, buka [Lingkungan Node Desktop](/id-ID/faq/desktop-node-environment).

Jika Anda ingin mengetahui mengapa HagiCode tidak ada di Steam, buka [Mengapa HagiCode tidak tersedia di Steam](/id-ID/faq/steam-distribution-status).
