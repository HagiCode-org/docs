## Ikhtisar

### Apa itu Masukan Suara Doubao

Doubao Voice Input adalah fitur pengenalan ucapan terintegrasi HagiCode, yang didukung oleh ByteDance Doubao Open Platform. Anda berbicara, dan sistem mengubah ucapan menjadi teks. HagiCode juga menggunakan konteks proyek saat ini untuk membantu mengenali kosakata domain dan istilah teknis dengan lebih andal.

### Kasus Penggunaan Utama

Di HagiCode, masukan suara berfungsi paling baik untuk:

- **Memasukkan proposal dengan cepat**: Membuat dan mengirimkan proposal hanya dengan berbicara, meningkatkan efisiensi secara signifikan tanpa mengetik
- **Memberikan komentar**: Tambahkan komentar dan masukan selama peninjauan kode atau peninjauan dokumen dengan masukan suara
- **Membalas pesan**: Membalas pesan dengan cepat dalam percakapan dan diskusi tanpa mengetik
- **Pembuatan konten berdurasi panjang**: Membuat dokumentasi proyek, spesifikasi teknis, catatan rapat, dan konten berdurasi panjang lainnya dengan cepat

### Keuntungan praktis

- **Integrasi Mendalam dengan Kesadaran Konteks**: HagiCode terintegrasi secara mendalam dengan pengenalan ucapan Doubao, secara otomatis memanfaatkan konteks proyek saat ini (seperti struktur kode, tumpukan teknologi, terminologi domain) untuk memberikan pengenalan kosakata domain yang tepat, secara signifikan meningkatkan akurasi pengenalan istilah teknis
- **Jam Uji Coba Gratis**: Platform Doubao memberikan 20 jam waktu pengenalan gratis bagi pengguna baru untuk merasakan input suara tanpa hambatan
- **Sangat Cepat**: Pengenalan waktu nyata, lihat teks saat Anda berbicara tanpa perlu menunggu
- **Akurasi Tinggi**: Didukung oleh model pengenalan suara canggih Doubao yang dikombinasikan dengan konteks proyek untuk hasil yang tepat dan andal
- **Integrasi yang Mulus**: Langsung terintegrasi ke dalam kotak input pesan HagiCode, tidak perlu berpindah aplikasi
- **Mudah Digunakan**: Cukup klik mikrofon untuk memulai, intuitif dan lugas

## Mulai Cepat

### Prasyarat

Sebelum menggunakan Input Suara Doubao, Anda perlu:

1. **Dapatkan Akun Platform Terbuka Doubao**
   - Kunjungi [Konsol Pengenalan Ucapan Doubao](https://console.volcengine.com/speech/service/10038)
   - Daftar atau masuk ke akun Anda

2. **Buat Aplikasi dan Dapatkan Kredensial**
   - Buat aplikasi pengenalan suara di platform
   - Dapatkan milikmu `APP ID` dan `Access Token`

3. **Pastikan Koneksi Jaringan**
   - Layanan pengenalan ucapan memerlukan koneksi jaringan
   - Pastikan perangkat Anda dapat mengakses layanan Doubao API

### Alur Penggunaan Dasar

Berikut adalah langkah-langkah dasar untuk menggunakan Doubao Voice Input:

1. Dapatkan `APP ID` dan `Access Token`
2. Konfigurasikan pengenalan suara pada produk
3. Uji kunci API
4. Temukan kotak masukan suara
5. Klik tombol mikrofon
6. Berikan izin mikrofon
7. Mulai merekam
8. Lihat hasil pengenalan secara real time
9. Klik untuk berhenti merekam
10. Konfirmasikan teks telah dimasukkan

## Persyaratan Teknis

### Lingkungan HTTP

Doubao Voice Input perlu dijalankan di lingkungan HTTP/HTTPS.

| Lingkungan Hidup | Status Dukungan | Deskripsi |
|-------------|----------------|-------------|
| **host lokal** | ✓ | Lingkungan pengembangan lokal (http://localhost atau http://127.0.0.1) |
| **Server Jarak Jauh HTTPS** | ✓ | Aplikasi web disebarkan ke jaringan publik dengan HTTPS |
| **Server Jarak Jauh HTTP** | ✗ | Aplikasi web disebarkan ke jaringan publik dengan HTTP |
| **berkas:// protokol** | ✗ | Membuka file HTML secara langsung tidak didukung |

:::hati-hati
Input Suara Doubao memiliki batasan berikut:
1. **HTTPS diperlukan untuk penerapan eksternal**: Karena pembatasan keamanan browser, akses mikrofon memerlukan HTTPS saat mengakses dari jaringan publik
2. **Alamat lokal tidak dibatasi**: HTTP dan HTTPS didukung saat menggunakan `localhost` atau `127.0.0.1`
3. **file:// protokol tidak didukung**: The `file://` lingkungan protokol tidak dapat menggunakan mikrofon dan koneksi WebSocket
:::

### Versi Desktop

Versi Desktop HagiCode memiliki lingkungan HTTP lokal bawaan dan sepenuhnya mendukung input suara. Anda dapat menggunakan pengenalan suara secara langsung tanpa konfigurasi tambahan.

### Modus Tuan Rumah

Mode host lokal mendukung input suara:
- Baik HTTP dan HTTPS didukung saat menggunakan `localhost` atau `127.0.0.1`
- HTTPS diperlukan saat diterapkan ke jaringan publik

## Langkah Konfigurasi

### Dapatkan ID APLIKASI

1. Kunjungi [Konsol Pengenalan Ucapan Doubao](https://console.volcengine.com/speech/service/10038)
2. Masuk atau daftarkan akun
3. Buka konsol dan buat aplikasi pengenalan suara baru
4. Di halaman detail aplikasi, temukan dan salin `APP ID`

### Dapatkan Token Akses

1. Di konsol Platform Terbuka Doubao
2. Buka aplikasi pengenalan suara Anda
3. Temukan area manajemen kunci API
4. Hasilkan atau salin `Access Token`

:::catatan
Tolong simpan milikmu `Access Token` aman dan jangan membaginya dengan orang lain. Jika Anda perlu menggantinya, Anda dapat membuatnya kembali di Platform Terbuka Doubao.
:::

### Konfigurasikan di Produk

1. Buka aplikasi HagiCode
2. Buka **Pengaturan** → **Pengaturan Pengenalan Suara**
3. Isi informasi berikut dalam formulir konfigurasi:
   - **Penyedia**: Pilih `doubao` (Doubao)
   - **ID APLIKASI**: Tempelkan ID APLIKASI yang Anda dapatkan dari platform Doubao
   - **Token Akses**: Tempelkan Token Akses yang Anda dapatkan dari platform Doubao
4. (Opsional) Sesuaikan parameter konfigurasi lainnya sesuai kebutuhan
5. Klik tombol **Uji Kunci API** untuk memverifikasi konfigurasi
6. Setelah verifikasi berhasil, konfigurasi secara otomatis disimpan ke penyimpanan lokal browser

### Uji Kunci API

Setelah konfigurasi, disarankan untuk menguji apakah Kunci API valid:

1. Klik tombol **Uji Kunci API**
2. Sistem akan memanggil antarmuka pengujian untuk memverifikasi konfigurasi Anda
3. Jika konfigurasi sudah benar maka akan muncul pesan sukses
4. Jika konfigurasi salah, pesan kesalahan akan ditampilkan. Silakan periksa:
   - Apakah ID APP dan Token Akses sudah benar
   - Apakah koneksi jaringan normal
   - Apakah Token Akses telah kedaluwarsa

:::tip
Jika pengujian gagal, periksa terlebih dahulu apakah koneksi jaringan normal, lalu konfirmasikan bahwa kunci API dari Platform Terbuka Doubao valid dan tidak kedaluwarsa.
:::

## Penggunaan

### Temukan Lokasi Masukan Suara

Di HagiCode, kotak input teks dengan ikon mikrofon mendukung input suara. Komponen-komponen ini biasanya disebut `VoiceTextArea`.

Lokasi masukan suara yang umum meliputi:
- Kotak masukan pesan
- Area pengeditan teks
- Berbagai bidang formulir yang memerlukan input teks

### Mulai Pengenalan Suara

1. Temukan kotak masukan dengan ikon mikrofon
2. Klik tombol mikrofon
3. Browser akan meminta izin mikrofon
4. Klik **Izinkan** untuk mengotorisasi akses mikrofon

:::catatan
Jika sebelumnya Anda telah mengotorisasi mikrofon, browser mungkin tidak meminta izin lagi. Untuk memeriksa atau mengubah pengaturan izin, Anda dapat melihatnya di ikon gembok di sebelah kiri bilah alamat browser.
:::

### Deskripsi Status Perekaman

Setelah otorisasi berhasil, pengenalan suara akan dimulai secara otomatis, dan Anda dapat melihat:

- **Animasi Bentuk Gelombang**: Bentuk gelombang suara dinamis akan ditampilkan di dalam tombol mikrofon, menandakan perekaman sedang berlangsung
- **Tampilan Durasi**: Durasi perekaman saat ini akan ditampilkan di bawah tombol
- **Pengenalan Waktu Nyata**: Teks yang dikenali akan ditampilkan sementara pada posisi kursor

### Tampilan Hasil Pengenalan Real-time

Selama perekaman, mesin pengenalan suara akan mengubah ucapan Anda menjadi teks secara real-time:

- Teks yang dikenali untuk sementara akan ditampilkan dalam warna abu-abu di kotak masukan
- Hasil pengakuan akan terus diperbarui saat Anda berbicara
- Mendukung pengenalan bahasa Mandarin dengan akurasi tinggi

:::tip
Untuk hasil pengenalan yang lebih baik, disarankan untuk:
- Gunakan di lingkungan yang tenang
- Bicaralah dengan jelas dan dengan kecepatan sedang
- Hindari gangguan kebisingan latar belakang
:::

### Hentikan Pengakuan

Untuk menghentikan pengenalan suara, Anda dapat:

1. **Klik tombol mikrofon**: Klik tombol lagi untuk berhenti merekam
2. **Klik kotak masukan**: Klik area lain di kotak masukan juga akan berhenti merekam

Setelah berhenti, teks terakhir yang dikenali akan dimasukkan secara resmi ke dalam kotak masukan, dan Anda dapat melanjutkan pengeditan atau pengiriman.
