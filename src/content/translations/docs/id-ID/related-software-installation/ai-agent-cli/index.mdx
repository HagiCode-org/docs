Halaman ini adalah entri instalasi CLI Agen AI lokal terpadu `repos/docs`. HagiCode saat ini mendukung 11 Agen CLI aktif, mengikuti backend `PrimaryProfessions` pesanan: Claude Code, Codex, GitHub Copilot, OpenCode, Hermes, QoderCLI, Kiro, Kimi, Gemini, DeepAgents, dan Codebuddy.

:::tip[Cara menggunakan halaman ini]
Periksa tabel untuk mengetahui status dukungan dan mode pengaturan saat ini terlebih dahulu, lalu lanjutkan dengan dokumen instalasi, autentikasi, ACP, izin, dan pemecahan masalah yang dikelola vendor. Cakupan dukungan dan mode penyiapan merupakan masalah terpisah.
:::

## Glosarium negara

- **Saat ini didukung** berarti CLI termasuk dalam rentang dukungan HagiCode yang aktif saat ini.
- **Instalasi sekali klik** berarti HagiCode menyediakan titik masuk instalasi yang dipandu; beberapa CLI masih memerlukan login vendor atau langkah autentikasi setelah instalasi.
- **Instalasi/autentikasi manual** berarti CLI sekarang didukung, tetapi Anda harus menyelesaikan instalasi lokal atau alur autentikasi sebelum HagiCode dapat mendeteksi dan menggunakannya.
- **Hanya kompatibilitas lama** berarti nilai yang dihentikan seperti `IFlowCli` masih ada untuk kompatibilitas data historis, namun bukan bagian dari rentang dukungan aktif saat ini.

## Rekomendasi langkah selanjutnya

- Lanjutkan dengan [Instal Node.js](/en/related-software-installation/nodejs/installation) dan [Instal OpenSpec](/en/related-software-installation/openspec/setup-openspec) jika Anda masih memerlukan prasyarat.
- Jika Anda menginginkan satu gateway lokal untuk beberapa alat yang kompatibel dengan OpenAI, lanjutkan [Instalasi dan Penggunaan OmniRoute](/en/related-software-installation/omniroute/).
- Kembali ke [Panduan Instalasi](/en/installation) untuk jalur pengaturan yang lebih luas.
- Jika Anda tiba di sini dari postingan blog bersejarah, perlakukan halaman ini dan ikhtisar instalasi sebagai sumber kebenaran terkini.
