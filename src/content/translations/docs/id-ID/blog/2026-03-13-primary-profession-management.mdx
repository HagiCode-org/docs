## Manajemen profesi utama dalam pengaturan pahlawan

- Pengaturan Hero kini menyertakan tab Profesi Utama khusus untuk mengubah ketersediaan.
- Pemberdayaan dipertahankan di tingkat sistem dan menjamin ketersediaan pahlawan dalam pemilihan ruang bawah tanah dan pemeriksaan status.
- Deteksi CLI menunjukkan ketersediaan dan versi; tombol pengaktifan tetap terkunci hingga CLI terdeteksi.
