Sesi percakapan adalah titik masuk paling ringan di Hagicode. Mulailah dengan dialog, pahami repositori, dan putuskan nanti apakah pengeditan diperlukan. UI saat ini membuka percakapan baru dalam mode baca-saja secara default, sehingga Anda dapat memeriksa basis kode sebelum meminta AI untuk mengubah file.

:::catatan[Tangkapan layar yang dibagikan]
Kumpulan tangkapan layar saat ini menunjukkan UI berbahasa Mandarin. Panduan bahasa Inggris sengaja menggunakan kembali tangkapan layar terkelola yang sama dan menjelaskan setiap kontrol dalam bahasa Inggris di samping gambar.
:::

## Prasyarat

Sebelum Anda memulai sesi percakapan, pastikan Anda memiliki:

- selesai [Instalasi Desktop](/id-ID/installation/desktop)
- selesai [Pengaturan Penyihir](/id-ID/quick-start/wizard-setup) dan membuat setidaknya satu proyek
- memasuki ruang kerja Hagicode untuk proyek itu

## Untuk apa sesi percakapan?

Sesi percakapan berfungsi paling baik bila Anda ingin memahami terlebih dahulu dan mengeditnya nanti. Penggunaan umum meliputi:

- dengan cepat menganalisis repositori yang tidak dikenal
- mengganti preset pahlawan untuk nada, penyedia, atau konteks
- tetap dalam mode baca-saja sampai Anda yakin bahwa pengeditan layak dilakukan
- melampirkan gambar, petunjuk cepat, atau konteks riwayat sebelum mengirim pesan berikutnya

Jika tugas Anda telah berpindah dari “memahami masalah” menjadi “merencanakan, meninjau, dan mengarsipkan perubahan terstruktur,” lanjutkan dengan [Buat Sesi Proposal](/id-ID/quick-start/proposal-session). Halaman ini hanya mencakup sesi percakapan.

## Ikhtisar alur kerja

Jalur utama dirangkum di bawah ini: membuat percakapan dari daftar sesi, secara opsional memilih preset pahlawan, mengajukan pertanyaan pertama dalam mode baca-saja, dan beralih ke mode edit hanya setelah Anda mengonfirmasi bahwa diperlukan perubahan file langsung.

| Panggung | Tindakan pengguna | Hasil sistem |
| --- | --- | --- |
| Masuk ke sesi | Buat percakapan dari kontrol sesi teratas | Sesi dibuka dengan entri pahlawan default |
| Pilihan pahlawan opsional | Buka panel Preset dan pilih pahlawan | Sesi ini menggunakan peran, penyedia, atau konteks tersebut |
| Pertanyaan hanya baca | Tanyakan sesuatu seperti `分析这个项目` | AI membaca repositori dan mengembalikan analisis |
| Putuskan eksekusi | Tetap dalam mode baca-saja atau aktifkan mode edit | Analisis aman dilanjutkan atau eksekusi yang dapat diedit dimulai |
| Tambahkan lebih banyak konteks | Gunakan alat gambar, prompt cepat, atau riwayat | Pesan berikutnya membawa konteks yang lebih kaya |

Contoh read-only pertama hanyalah rangkaian interaksi singkat. Aturannya sederhana: AI dapat memeriksa dan menjelaskan, tetapi AI tidak akan mengedit file sampai Anda beralih mode.

1. **Masukkan perintah**: pengguna mengetik `分析这个项目`
2. **Kirim dalam mode baca-saja**: UI mengirimkan permintaan dengan izin aman default
3. **Baca konteks repositori**: AI memeriksa folder, konfigurasi, dan file kunci
4. **Analisis pengembalian**: AI merangkum struktur, modul, dan risiko
5. **Pilih langkah berikutnya**: pengguna melanjutkan chat atau mengaktifkan mode edit

## Langkah 1: Buat percakapan dari daftar sesi

Entri saat ini untuk percakapan normal ada di kontrol sesi teratas. Anda bisa langsung memulai dengan hero default, atau membuka dropdown avatar di sebelah kanan sebelum memasuki sesi.

![Entri percakapan saat ini di sesi teratas mengontrol dengan status pahlawan default](../../img/screenshots/quick-start/create-conversation-session/01-create-with-default-hero/original.png)

> Gambar: kontrol sesi teratas adalah titik masuk percakapan saat ini; dropdown avatar membuka jalur preset pahlawan opsional.

Mulailah dengan dua detail:

- **Entri berada di kontrol atas**, bukan di kontrol lama `Add Chat` aliran tombol.
- **Pemilihan pahlawan bersifat opsional**. Jika Anda baru ingin mulai menganalisis repositori, lanjutkan saja.

## Langkah 2: Secara opsional pilih preset pahlawan

Jika Anda menginginkan nada, penyedia model, atau konteks preset yang berbeda, buka panel Presets dan pilih hero sebelum Anda melanjutkan.

![Panel Preset yang diperluas dengan beberapa preset pahlawan tersedia untuk sesi percakapan](../../img/screenshots/quick-start/create-conversation-session/02-create-with-selected-hero/original.png)

> Gambar: panel Preset mencantumkan preset pahlawan yang dapat digunakan kembali; memilih salah satu adalah opsional dan tidak mengubah alur percakapan dasar.

Memilih hero terlebih dahulu berguna ketika:

- Anda sudah tahu penyedia atau gaya mana yang Anda inginkan
- Anda mempertahankan peran sesi yang berbeda untuk tugas yang berbeda
- Anda ingin membandingkan beberapa sudut pandang pahlawan pada proyek yang sama

## Langkah 3: Analisis proyek dalam mode read-only default

Setelah sesi dibuka, mode baca-saja adalah mode keamanan default. Di tangkapan layar, perintah pertama adalah `分析这个项目`, dan AI merespons dengan membaca repositori dan merangkum proyek.

![Contoh percakapan hanya-baca menggunakan perintah 分析这个项目 untuk memeriksa repositori](../../img/screenshots/quick-start/create-conversation-session/03-chat-view-readonly/original.png)

> Gambar: sesi percakapan dimulai dalam mode baca-saja, yang ideal untuk analisis repositori, tinjauan struktur, dan pengumpulan konteks.

Dalam mode read-only, AI biasanya dapat:

- membaca file dan struktur direktori
- merangkum tujuan proyek, modul utama, dan ketergantungan
- jelaskan apa yang menjadi tanggung jawab suatu fitur, file, atau modul

Dalam mode read-only, AI tidak secara langsung:

- memodifikasi file repositori
- jalankan perubahan berorientasi edit sebelum Anda beralih mode

Jika tujuan Anda saat ini adalah memahami basis kode, memperjelas masalah, atau mempersiapkan pekerjaan selanjutnya, mode baca-saja biasanya sudah cukup.

## Langkah 4: Aktifkan mode edit dan pelajari toolbar

Saat Anda siap untuk perubahan langsung, alihkan sesi ke mode edit. UI saat ini menempatkan alat pengalih mode dan alat pembantu di baris toolbar di atas tombol kirim.

![Mode edit beralih dengan penyisipan gambar, prompt cepat, dan alat riwayat di komposer saat ini](../../img/screenshots/quick-start/create-conversation-session/04-edit-mode-controls/original.png)

> Gambar: baris toolbar menyertakan peralihan mode terlebih dahulu, lalu penyisipan gambar, prompt cepat, dan titik masuk riwayat.

Setiap kontrol memiliki tujuan berbeda:

- **Pengalih mode**: mengalihkan sesi dari mode baca-saja default ke mode edit sehingga AI dapat membuat perubahan nyata.
- **Tombol Gambar**: melampirkan konteks gambar sebelum pesan berikutnya dikirim.
- **Tombol prompt cepat**: menyisipkan pintasan cepat yang dapat digunakan kembali ke dalam percakapan.
- **Tombol riwayat**: membuka konteks sesi sebelumnya atau riwayat terkait.

Pola yang lebih aman adalah sederhana: konfirmasikan arah dalam mode baca-saja terlebih dahulu, kemudian aktifkan mode edit ketika Anda benar-benar menginginkan eksekusi.

## Langkah 5: Lampirkan gambar sebelum mengirim pesan berikutnya

Sesi percakapan mendukung lampiran gambar langsung di dalam komposer saat ini. Gambar ditambahkan sebagai konteks tambahan di atas area input, dan komposer yang sama tetap menyediakan mode pengalih dan alat pembantu.

![Komposer percakapan dengan pratinjau gambar terlampir dan toolbar mode edit yang sama masih terlihat](../../img/screenshots/quick-start/create-conversation-session/05-insert-image/original.png)

> Gambar: pratinjau gambar muncul di dalam area komposer terlebih dahulu, dan Anda masih dapat menggunakan alat pengalih dan pembantu sebelum mengirim.

Ini berguna ketika Anda ingin:

- minta AI untuk menjelaskan tangkapan layar antarmuka
- lampirkan tangkapan layar kesalahan atau desain mockup dan tambahkan instruksi tindak lanjut
- memperkaya prompt berikutnya dengan konteks visual sebelum beralih ke eksekusi

## Kasus penggunaan dan perkembangan yang umum

Dari lima tangkapan layar, alur yang disarankan adalah:

1. **Masuk sesi dengan cepat** dari kontrol atas.
2. **Pilih pahlawan hanya saat diperlukan** melalui panel Preset.
3. **Mulai dengan analisis hanya-baca** menggunakan perintah seperti `分析这个项目`.
4. **Tingkatkan ke mode edit dengan sengaja** ketika perubahan file langsung diperlukan.
5. **Tambahkan lebih banyak konteks** melalui gambar, petunjuk cepat, dan riwayat sebelum pesan berikutnya.

Jika tugas sekarang memerlukan perencanaan, peninjauan, dan disiplin pengarsipan, lanjutkan dengan panduan sesi proposal berikutnya.
