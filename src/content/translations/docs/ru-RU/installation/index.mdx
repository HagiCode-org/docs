HagiCode поддерживает несколько путей установки. Начните с метода развертывания, который соответствует вашему сценарию, а затем добавляйте вспомогательное программное обеспечение только тогда, когда оно вам понадобится.

Текущий диапазон активной поддержки CLI агента, наиболее подходящий для настройки: Claude Code, Codex, GitHub Copilot, OpenCode, Hermes, QoderCLI, Kiro, Kimi, Gemini, DeepAgents и Codebuddy. Hermes, Gemini и DeepAgents в настоящее время используют ручную установку или аутентификацию; другие интерфейсы командной строки предоставляют точки входа для установки одним щелчком мыши или управляемые потоки установки.

## Рекомендуемые следующие шаги

- Если вы новичок в HagiCode, начните с [Установка на рабочем столе](/ru-RU/installation/desktop).
- Если вы выполняете развертывание для серверной или групповой среды, используйте [Установка Docker Compose](/ru-RU/installation/docker-compose).
- Если вам нужно подтвердить состояние поддержки, режим установки или исторические примечания по совместимости для CLI, откройте [Установка CLI агента AI](/ru-RU/related-software-installation/ai-agent-cli/).
