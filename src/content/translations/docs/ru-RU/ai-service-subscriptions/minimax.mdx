## Позиционирование сервиса

MiniMax — один из рекомендуемых пунктов подписки для Claude Code в текущем preset документации. Единая модель, простая настройка — подходит для команд, которым важны плавность интеграции и низкие затраты на обслуживание.

## Для кого подходит

- Тем, кто хочет использовать одно и то же стандартное сопоставление для всех трёх уровней моделей
- Тем, для кого важны простота настройки и стабильность подключения
- Тем, кто хочет быстро запустить рабочий процесс Claude Code

## На что обратить внимание

- В текущем preset MiniMax отмечен как рекомендуемый поставщик.
- Единое использование `MiniMax-M2.5` в качестве модели по умолчанию снижает затраты на понимание конфигурации.
- Подходит для команд, которые хотят сначала запустить рабочий процесс, а затем уточнять стратегию моделей.

## Ключевая информация для подключения

| Пункт | Содержание |
|-------|-----------|
| Пункт подписки | [Перейти на страницу подписки](https://platform.minimaxi.com/subscribe/coding-plan?code=8wNck4etDM&source=link) |
| Официальная документация / консоль | [Посетить официальный сайт](https://www.minimaxi.com) |
| Anthropic-совместимый Base URL | `https://api.minimaxi.com/anthropic` |
| Рекомендуемая переменная аутентификации | `ANTHROPIC_AUTH_TOKEN` |

## Стандартные сопоставления моделей

| Уровень Claude | Модель по умолчанию |
|---------------|---------------------|
| Sonnet | `MiniMax-M2.5` |
| Opus | `MiniMax-M2.5` |
| Haiku | `MiniMax-M2.5` |

:::tip
Тарифы, скидки и права могут изменяться с течением времени. Если вас интересуют актуальные цены, перейдите непосредственно на страницу подписки поставщика.
:::

## Рекомендации по настройке

1. Сначала определитесь с тарифом или способом получения API Key.
2. Затем откройте раздел Claude Code на странице AI Agent CLI и перейдите к официальной документации Anthropic по установке.
3. После завершения настройку можно проверить, выполнив `claude --version`, проверив конфигурационные файлы или сделав тестовый запрос.
