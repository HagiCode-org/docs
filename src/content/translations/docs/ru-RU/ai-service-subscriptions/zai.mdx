## Позиционирование сервиса

Zhipu AI — один из рекомендуемых пунктов подписки для Claude Code в текущем preset документации. Тариф GLM Coding с мульти-модельным сопоставлением подходит разработчикам, которые используют Claude Code как основной ежедневный инструмент.

## Для кого подходит

- Тем, кому нужен тариф, специализированный для сценариев программирования
- Тем, кто нуждается в достаточно полном сопоставлении моделей уровня Claude
- Тем, кто хочет охватить Claude Code, Cline и другие инструменты через один сервис

## На что обратить внимание

- В текущем preset Zhipu AI отмечен как рекомендуемый поставщик.
- Предоставляет Anthropic-совместимый API для прямого подключения к Claude Code.
- В preset включены стандартные сопоставления моделей: `glm-4.7`, `glm-5` и `glm-4.5-air`.

## Ключевая информация для подключения

| Пункт | Содержание |
|-------|-----------|
| Пункт подписки | [Перейти на страницу подписки](https://www.bigmodel.cn/claude-code?ic=14BY54APZA) |
| Официальная документация / консоль | [Посетить официальный сайт](https://open.bigmodel.cn/dev/api) |
| Anthropic-совместимый Base URL | `https://open.bigmodel.cn/api/anthropic` |
| Рекомендуемая переменная аутентификации | `ANTHROPIC_AUTH_TOKEN` |

## Стандартные сопоставления моделей

| Уровень Claude | Модель по умолчанию |
|---------------|---------------------|
| Sonnet | `glm-4.7` |
| Opus | `glm-5` |
| Haiku | `glm-4.5-air` |

:::tip
Тарифы, скидки и права могут изменяться с течением времени. Если вас интересуют актуальные цены, перейдите непосредственно на страницу подписки поставщика.
:::

## Рекомендации по настройке

1. Сначала определитесь с тарифом или способом получения API Key.
2. Затем откройте раздел Claude Code на странице AI Agent CLI и перейдите к официальной документации Anthropic по установке.
3. После завершения настройку можно проверить, выполнив `claude --version`, проверив конфигурационные файлы или сделав тестовый запрос.
