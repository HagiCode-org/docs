## Позиционирование сервиса

Alibaba Cloud DashScope — один из рекомендуемых пунктов подписки для Claude Code в текущем preset документации. Подходит для команд, уже работающих в экосистеме Alibaba Cloud, которые хотят объединить подписку, консоль и управление ключами в единой облачной платформе.

## Для кого подходит

- Уже используют консоль Alibaba Cloud или облачные ресурсы
- Хотят управлять сервисами и API Key из единой облачной платформы
- Хотят быстро подключить Claude Code с фиксированным сопоставлением моделей

## На что обратить внимание

- В текущем preset Alibaba Cloud DashScope отмечен как рекомендуемый поставщик.
- В качестве модели по умолчанию используется `glm-4.7` — путь настройки понятен.
- Если у команды уже есть учётная запись Alibaba Cloud, подключение и управление правами будут удобнее.

## Ключевая информация для подключения

| Пункт | Содержание |
|-------|-----------|
| Пункт подписки | [Перейти на страницу подписки](https://www.aliyun.com/benefit/ai/aistar?userCode=vmx5szbq&clubBiz=subTask..12384055..10263..) |
| Официальная документация / консоль | [Посетить официальный сайт](https://dashscope.aliyun.com) |
| Anthropic-совместимый Base URL | `https://coding.dashscope.aliyuncs.com/apps/anthropic` |
| Рекомендуемая переменная аутентификации | `ANTHROPIC_AUTH_TOKEN` |

## Стандартные сопоставления моделей

| Уровень Claude | Модель по умолчанию |
|---------------|---------------------|
| Sonnet | `glm-4.7` |
| Opus | `glm-4.7` |
| Haiku | `glm-4.7` |

:::tip
Тарифы, скидки и права могут изменяться с течением времени. Если вас интересуют актуальные цены, перейдите непосредственно на страницу подписки поставщика.
:::

## Рекомендации по настройке

1. Сначала определитесь с тарифом или способом получения API Key.
2. Затем откройте раздел Claude Code на странице AI Agent CLI и перейдите к официальной документации Anthropic по установке.
3. После завершения настройку можно проверить, выполнив `claude --version`, проверив конфигурационные файлы или сделав тестовый запрос.
