Este capítulo recopila los sitios web de suscripción de servicios de IA actualmente marcados como recomendados en `repos/docs/public/presets/claude-code/providers/*.json`. Primero elija un proveedor aquí y luego continúe con la configuración de Claude Code.

A partir del **9 de marzo de 2026**, este capítulo cubre: **ZAI, MiniMax, Alibaba Cloud DashScope y Volcengine Coding Plan**.

:::tip[Orden de lectura sugerido]
Si aún no ha elegido un proveedor, comience con esta página para comparar. Si ya sabe qué proveedor desea, vaya primero a su página de detalles y luego continúe con la entrada del Código Claude en el centro CLI unificado de AI Agent.
:::

## Proveedores recomendados actualmente

| Proveedor | Fuerza | Buen ajuste para | Página de detalles |
|----------|----------|--------------|-------------|
| ZAI | Planes centrados en codificación con mapeo de niveles | Desarrolladores que quieren Claude Code como conductor diario | [Ver detalles](./zai/) |
| minimax | Un modelo predeterminado para todos los niveles, incorporación sencilla | Equipos optimizando para una configuración de baja fricción | [Ver detalles](./minimax/) |
| DashScope de nube de Alibaba | Alineación de la consola en la nube y gestión unificada | Equipos que ya trabajan en Alibaba Cloud | [Ver detalles](./aliyun/) |
| Plan de codificación Volcengine | Fuerte extensión del ecosistema y señales de plataforma multimodelo | Desarrolladores que quieran alinear el ecosistema Volcengine | [Ver detalles](./volcengine/) |

:::nota
`Anthropic Official` sigue siendo utilizable, pero no está incluido en el primer lote de páginas dedicadas porque los metadatos preestablecidos actuales no lo marcan como recomendado.
:::

## Consejos de selección rápida

- **Necesita planes centrados en la codificación y una asignación de niveles más clara**: comience con [ZAI](./zai/)
- **Primero necesita la ruta de configuración más sencilla**: comience con [minimax](./minimax/)
- **Ya se está ejecutando dentro de Alibaba Cloud**: comience con [DashScope de nube de Alibaba](./aliyun/)
- **Quiere alineación con el ecosistema Volcengine o alcance de plataforma multimodelo**: comience con [Plan de codificación Volcengine](./volcengine/)
