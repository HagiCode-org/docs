## Posicionamiento

Alibaba Cloud DashScope es uno de los proveedores recomendados actualmente por el conjunto preestablecido de documentos para Claude Code. Una opción práctica para equipos que ya operan en el ecosistema de Alibaba Cloud y desean una administración de consola unificada.

## Buen ajuste para

- Ya utilizas cuentas o recursos de Alibaba Cloud
- Quiere que la suscripción y la administración de claves permanezcan en una consola en la nube
- Prefiere un modelo mapeado fijo para una incorporación rápida

## Cosas clave a tener en cuenta

- Los metadatos preestablecidos actuales marcan Alibaba Cloud DashScope como recomendado.
- Asigna todos los niveles de Claude a `glm-4.7` para una ruta de instalación sencilla.
- Es especialmente conveniente si su equipo ya trabaja dentro de los flujos de trabajo de Alibaba Cloud.

## Instantánea de integración

| Artículo | Valor |
|------|-------|
| Entrada de suscripción | [Abrir página de suscripción](https://www.aliyun.com/benefit/ai/aistar?userCode=vmx5szbq&clubBiz=subTask..12384055..10263..) |
| Documentos oficiales/consola | [Abrir página oficial](https://dashscope.aliyun.com) |
| URL base compatible con antrópicos | `https://coding.dashscope.aliyuncs.com/apps/anthropic` |
| Var de entorno de autenticación recomendada | `ANTHROPIC_AUTH_TOKEN` |

## Mapeo de modelo predeterminado

| nivel claude | Modelo predeterminado |
|-------------|---------------|
| soneto | `glm-4.7` |
| obra | `glm-4.7` |
| haikus | `glm-4.7` |

:::consejo
Los precios, las promociones y los detalles del paquete pueden cambiar con el tiempo. Si el costo es su principal preocupación, confirme la información más reciente directamente en la página del proveedor.
:::

## Próximos pasos sugeridos

1. Confirme qué nivel de suscripción o ruta de activación de API desea utilizar.
2. Abra la entrada Claude Code en el centro CLI unificado de AI Agent y continúe con los documentos de referencia y de instalación oficiales de Anthropic.
3. Valide la conexión localmente después de la configuración.
