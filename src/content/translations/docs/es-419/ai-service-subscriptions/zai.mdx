## Posicionamiento

ZAI es uno de los proveedores recomendados actualmente por el conjunto preestablecido de documentos para Claude Code. Planes de codificación GLM con mapeo de niveles para desarrolladores que desean Claude Code como controlador diario.

## Buen ajuste para

- Quieres un plan de suscripción centrado en la codificación
- Necesita valores predeterminados de Sonnet/Opus/Haiku mapeados
- Espera que un proveedor trabaje con Claude Code y herramientas de codificación similares.

## Cosas clave a tener en cuenta

- Los metadatos preestablecidos actuales marcan a ZAI como recomendado.
- Expone un punto final compatible con Anthropic que Claude Code puede utilizar directamente.
- El ajuste preestablecido incluye asignaciones predeterminadas para `glm-4.7`, `glm-5`, y `glm-4.5-air`.

## Instantánea de integración

| Artículo | Valor |
|------|-------|
| Entrada de suscripción | [Abrir página de suscripción](https://www.bigmodel.cn/claude-code?ic=14BY54APZA) |
| Documentos oficiales/consola | [Abrir página oficial](https://open.bigmodel.cn/dev/api) |
| URL base compatible con antrópicos | `https://open.bigmodel.cn/api/anthropic` |
| Var de entorno de autenticación recomendada | `ANTHROPIC_AUTH_TOKEN` |

## Mapeo de modelo predeterminado

| nivel claude | Modelo predeterminado |
|-------------|---------------|
| soneto | `glm-4.7` |
| obra | `glm-5` |
| haikus | `glm-4.5-air` |

:::consejo
Los precios, las promociones y los detalles del paquete pueden cambiar con el tiempo. Si el costo es su principal preocupación, confirme la información más reciente directamente en la página del proveedor.
:::

## Próximos pasos sugeridos

1. Confirme qué nivel de suscripción o ruta de activación de API desea utilizar.
2. Abra la entrada Claude Code en el centro CLI unificado de AI Agent y continúe con los documentos de referencia y de instalación oficiales de Anthropic.
3. Valide la conexión localmente después de la configuración.
