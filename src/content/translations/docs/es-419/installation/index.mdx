HagiCode admite múltiples rutas de instalación. Comience con el método de implementación que se ajuste a su escenario y luego agregue software de soporte solo cuando lo necesite.

El rango de compatibilidad de CLI del agente activo actual más relevante para la configuración es: Claude Code, Codex, GitHub Copilot, OpenCode, Hermes, QoderCLI, Kiro, Kimi, Gemini, DeepAgents y Codebuddy. Hermes, Gemini y DeepAgents utilizan actualmente flujos de autenticación o instalación manual; las otras CLI exponen puntos de entrada de instalación con un solo clic o flujos de instalación guiados.

## Próximos pasos recomendados

- Si planeas comprar e instalar HagiCode a través de Steam, comienza con [Instalación de vapor](/es-419/installation/steam).
- Si eres nuevo en HagiCode, comienza con [Instalación de escritorio](/es-419/installation/desktop).
- Si está implementando para un servidor o entorno de equipo, use [Instalación de Docker Compose](/es-419/installation/docker-compose).
- Si necesita confirmar el estado de soporte, el modo de configuración o las notas de compatibilidad histórica para una CLI, abra [Instalación de la CLI del Agente AI](/es-419/related-software-installation/ai-agent-cli/).
