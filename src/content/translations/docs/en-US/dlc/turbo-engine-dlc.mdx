Turbo Engine is the core enhancement add-on for HagiCode Desktop, designed for users who have entered the high-frequency usage stage. It solves one thing: **pushing HagiCode from basic mode to high-intensity working mode.**

<SteamProductArtwork product={steamProducts['turbo-engine']} />

## How to Purchase

Turbo Engine is a one-time permanent license in the Microsoft Store (Store ID: `9NSD809W18Z6`), effective forever after purchase. You can purchase and refresh the license status directly from the subscription workspace in HagiCode Desktop.

## Unlocked Features

After purchasing Turbo Engine, the following capabilities are permanently unlocked:

| Feature | Description |
| --- | --- |
| **Proposal concurrency 3 → 32** | Generating, executing, and archiving proposals share this cap. 32 concurrency lets you advance multiple task lines in parallel. |
| **Extended document themes** | Switch document themes in the Adventure Party interface, previewing terminology and copy under different themes in real time. |
| **Custom Git commit signatures** | Set a default `Co-Authored-By` pattern, or override it temporarily for individual AI Compose Commit runs. |
| **Brand customization** | Upload a header logo, set the application title, browser title, and brand link target. |
| **Custom avatar uploads** | Upload a custom avatar for the current Hero, or reuse from the shared custom avatar library in create, edit, and batch-create flows. |
| **DLC avatar resources** | Turbo Engine avatar packs integrate into Hero Avatar Gallery and batch-create flows, directly participating in workflow selection. |

## Who Needs It

- Users who frequently advance multiple proposals or tasks at the same time.
- Heavy users for whom the default 3-concurrency cap is no longer enough.
- Users who need custom Git signatures, document themes, avatars, and brand identity.
