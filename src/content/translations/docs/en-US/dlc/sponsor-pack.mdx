`HagiCode - Sponsor Plan` is a Microsoft Store subscription that currently does not unlock features. Its essential difference from Turbo Engine DLC and All Beauties Pack is this: the latter two are respectively about functional upgrades and cosmetic additions, while the Sponsor Plan is about the act itself — "you are willing to directly support HagiCode's continued development."

## Who it's for

The Sponsor Plan is for users who:

- Already understand where HagiCode is heading and want the project to keep moving forward.
- Do not expect every act of support to immediately convert into a feature parameter bump. They find value in the idea that maintenance and iteration can keep happening.
- Want this support to have a clear, visible identity response inside the application — not silently forgotten.

If your main goal is concurrency expansion, document themes, brand customization, or avatar uploads, read [Turbo Engine DLC](../turbo-engine-dlc/) first.

## What's currently delivered

The Sponsor Plan is intentionally restrained. There are only two reliably confirmable entitlements:

**Sponsor Identity Badge (sponsorBadge)**

When sponsor entitlements are active on the current device, Desktop provides clear sponsor identity display. It is not a one-time popup or confirmation page — it is a recognition marker that remains visible throughout long-term use, letting you feel that "this support has been remembered by the system."

**Future Sponsor Feature Gate (premiumFeatureGate)**

This is a reserved permission entry point. It does not itself unlock any currently available feature, but it provides a stable integration point for future sponsor-only capabilities. When future sponsor rewards (such as more refined themes, additional visual polish, etc.) are ready, this framework allows them to ship smoothly without redesigning the entitlement model.
