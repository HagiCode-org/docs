## Positionering

ZAI is een van de providers die momenteel wordt aanbevolen door de documentvoorinstellingen voor Claude Code. GLM Coding-abonnementen met tier mapping voor ontwikkelaars die Claude Code als dagelijkse driver willen.

## Goede pasvorm voor

- U wilt een op code gericht abonnement
- U hebt de standaardinstellingen voor Sonnet/Opus/Haiku nodig
- Je verwacht van één provider dat hij met Claude Code en vergelijkbare codeertools werkt

## Belangrijke dingen om op te merken

- De huidige vooraf ingestelde metagegevens markeren ZAI als aanbevolen.
- Het onthult een Anthropic-compatibel eindpunt dat rechtstreeks door Claude Code kan worden gebruikt.
- De voorinstelling levert standaardtoewijzingen voor `glm-4.7`, `glm-5`, en `glm-4.5-air`.

## Momentopname van de integratie

| Artikel | Waarde |
|------|-------|
| Abonnementsinvoer | [Open de abonnementspagina](https://www.bigmodel.cn/claude-code?ic=14BY54APZA) |
| Officiële documenten/console | [Officiële pagina openen](https://open.bigmodel.cn/dev/api) |
| Antropisch compatibele basis-URL | `https://open.bigmodel.cn/api/anthropic` |
| Aanbevolen auth-env var | `ANTHROPIC_AUTH_TOKEN` |

## Standaard modeltoewijzing

| Claude-niveau | Standaardmodel |
|-------------|---------------|
| Sonnet | `glm-4.7` |
| Opus | `glm-5` |
| Haiku | `glm-4.5-air` |

:::tip
Prijzen, promoties en pakketgegevens kunnen in de loop van de tijd veranderen. Als de kosten uw grootste zorg zijn, bevestig dan de laatste informatie rechtstreeks op de providerpagina.
:::

## Voorgestelde volgende stappen

1. Bevestig welk abonnementsniveau of API-activeringspad u wilt gebruiken.
2. Open de Claude Code-vermelding in de uniforme AI Agent CLI-hub en ga verder met de officiële installatie- en referentiedocumenten van Anthropic.
3. Valideer de verbinding lokaal na de installatie.
