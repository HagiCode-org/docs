## Επισκόπηση

### Τι είναι η φωνητική είσοδος Doubao

Το Doubao Voice Input είναι η ενσωματωμένη δυνατότητα αναγνώρισης ομιλίας της HagiCode, που υποστηρίζεται από την Open Platform ByteDance Doubao. Μιλάτε και το σύστημα μετατρέπει την ομιλία σε κείμενο. Το HagiCode χρησιμοποιεί επίσης το τρέχον πλαίσιο έργου για να βοηθήσει στην πιο αξιόπιστη αναγνώριση του λεξιλογίου και των τεχνικών όρων του τομέα.

### Βασικές περιπτώσεις χρήσης

Στο HagiCode, η φωνητική είσοδος λειτουργεί καλύτερα για:

- **Γρήγορη εισαγωγή προτάσεων**: Δημιουργήστε και υποβάλετε προτάσεις με απλή ομιλία, βελτιώνοντας σημαντικά την αποτελεσματικότητα χωρίς πληκτρολόγηση
- **Παροχή σχολίων**: Προσθέστε σχόλια και σχόλια κατά τη διάρκεια ελέγχων κώδικα ή ελέγχου εγγράφων με φωνητική εισαγωγή
- **Απάντηση σε μηνύματα**: Απαντήστε γρήγορα σε μηνύματα σε συνομιλίες και συζητήσεις χωρίς να πληκτρολογείτε
- **Δημιουργία περιεχομένου μεγάλης μορφής**: Δημιουργήστε γρήγορα τεκμηρίωση έργου, τεχνικές προδιαγραφές, σημειώσεις συσκέψεων και άλλο περιεχόμενο μεγάλης μορφής

### Πρακτικά πλεονεκτήματα

- **Βαθιά ενοποίηση με Context Awareness**: Το HagiCode ενσωματώνεται σε βάθος με την αναγνώριση ομιλίας Doubao, αξιοποιώντας αυτόματα το τρέχον περιβάλλον του έργου (όπως δομή κώδικα, στοίβα τεχνολογίας, ορολογία τομέα) για να παρέχει ακριβή αναγνώριση λεξιλογίου τομέα, βελτιώνοντας σημαντικά την ακρίβεια αναγνώρισης τεχνικών όρων
- **Δωρεάν ώρες δοκιμής**: Η πλατφόρμα Doubao παρέχει 20 ώρες δωρεάν χρόνο αναγνώρισης για τους νέους χρήστες να βιώσουν φωνητική είσοδο με μηδενικό εμπόδιο
- **Εξαιρετικά γρήγορο**: Αναγνώριση σε πραγματικό χρόνο, δείτε κείμενο καθώς μιλάτε χωρίς αναμονή
- **Υψηλή Ακρίβεια**: Υποστηρίζεται από τα προηγμένα μοντέλα αναγνώρισης ομιλίας της Doubao σε συνδυασμό με το πλαίσιο του έργου για ακριβή και αξιόπιστα αποτελέσματα
- **Απρόσκοπτη ενσωμάτωση**: Άμεσα ενσωματωμένο στο πλαίσιο εισαγωγής μηνυμάτων του HagiCode, δεν χρειάζεται να αλλάξετε εφαρμογές
- **Εύκολο στη χρήση**: Απλώς κάντε κλικ στο μικρόφωνο για να ξεκινήσετε, διαισθητικό και απλό

## Γρήγορη εκκίνηση

### Προαπαιτούμενα

Πριν χρησιμοποιήσετε τη φωνητική είσοδο Doubao, πρέπει:

1. **Αποκτήστε έναν λογαριασμό ανοιχτής πλατφόρμας Doubao**
   - Επίσκεψη [Κονσόλα αναγνώρισης ομιλίας Doubao](https://console.volcengine.com/speech/service/10038)
   - Εγγραφείτε ή συνδεθείτε στον λογαριασμό σας

2. **Δημιουργήστε μια εφαρμογή και λάβετε διαπιστευτήρια**
   - Δημιουργήστε μια εφαρμογή αναγνώρισης ομιλίας στην πλατφόρμα
   - Πάρε το δικό σου `APP ID` και `Access Token`

3. **Εξασφαλίστε τη σύνδεση δικτύου**
   - Η υπηρεσία αναγνώρισης ομιλίας απαιτεί σύνδεση δικτύου
   - Βεβαιωθείτε ότι η συσκευή σας έχει πρόσβαση στην υπηρεσία Doubao API

### Ροή βασικής χρήσης

Ακολουθούν τα βασικά βήματα για τη χρήση της φωνητικής εισόδου Doubao:

1. Αποκτήστε `APP ID` και `Access Token`
2. Διαμορφώστε τη φωνητική αναγνώριση στο προϊόν
3. Δοκιμάστε το κλειδί API
4. Βρείτε το πλαίσιο φωνητικής εισαγωγής
5. Κάντε κλικ στο κουμπί του μικροφώνου
6. Εκχώρηση άδειας μικροφώνου
7. Ξεκινήστε την εγγραφή
8. Δείτε τα αποτελέσματα αναγνώρισης σε πραγματικό χρόνο
9. Κάντε κλικ για να σταματήσετε την εγγραφή
10. Επιβεβαιώστε ότι έχει εισαχθεί το κείμενο

## Τεχνικές Απαιτήσεις

### Περιβάλλον HTTP

Η φωνητική είσοδος Doubao απαιτεί εκτέλεση σε περιβάλλον HTTP/HTTPS.

| Περιβάλλον | Κατάσταση υποστήριξης | Περιγραφή |
|-------------|----------------|-------------|
| **localhost** | ✓ | Τοπικό περιβάλλον ανάπτυξης (http://localhost ή http://127.0.0.1) |
| **Απομακρυσμένος διακομιστής HTTPS** | ✓ | Εφαρμογές Ιστού που αναπτύσσονται σε δημόσια δίκτυα με HTTPS |
| **Απομακρυσμένος διακομιστής HTTP** | ✗ | Εφαρμογές Ιστού που αναπτύσσονται σε δημόσια δίκτυα με HTTP |
| **αρχείο:// πρωτόκολλο** | ✗ | Το άμεσο άνοιγμα αρχείων HTML δεν υποστηρίζεται |

:::προσοχή
Η φωνητική είσοδος Doubao έχει τους ακόλουθους περιορισμούς:
1. **Απαιτείται HTTPS για εξωτερική ανάπτυξη**: Λόγω περιορισμών ασφαλείας του προγράμματος περιήγησης, η πρόσβαση στο μικρόφωνο απαιτεί HTTPS κατά την πρόσβαση από δημόσια δίκτυα
2. **Τοπικές διευθύνσεις απεριόριστες**: Τόσο το HTTP όσο και το HTTPS υποστηρίζονται κατά τη χρήση `localhost` ή `127.0.0.1`
3. **Το πρωτόκολλο file:// δεν υποστηρίζεται**: Το `file://` Το περιβάλλον πρωτοκόλλου δεν μπορεί να χρησιμοποιήσει συνδέσεις μικροφώνου και WebSocket
:::

### Έκδοση για επιτραπέζιους υπολογιστές

Η έκδοση Desktop του HagiCode έχει ενσωματωμένο τοπικό περιβάλλον HTTP και υποστηρίζει πλήρως τη φωνητική είσοδο. Μπορείτε να χρησιμοποιήσετε την αναγνώριση ομιλίας απευθείας χωρίς πρόσθετη διαμόρφωση.

### Λειτουργία κεντρικού υπολογιστή

Η λειτουργία τοπικού κεντρικού υπολογιστή υποστηρίζει φωνητική είσοδο:
- Τόσο το HTTP όσο και το HTTPS υποστηρίζονται κατά τη χρήση `localhost` ή `127.0.0.1`
- Το HTTPS απαιτείται όταν αναπτύσσεται σε δημόσια δίκτυα

## Βήματα διαμόρφωσης

### Λήψη αναγνωριστικού εφαρμογής

1. Επίσκεψη [Κονσόλα αναγνώρισης ομιλίας Doubao](https://console.volcengine.com/speech/service/10038)
2. Συνδεθείτε ή εγγραφείτε λογαριασμού
3. Μεταβείτε στην κονσόλα και δημιουργήστε μια νέα εφαρμογή αναγνώρισης ομιλίας
4. Στη σελίδα λεπτομερειών εφαρμογής, βρείτε και αντιγράψτε το `APP ID`

### Λήψη διακριτικού πρόσβασης

1. Στην κονσόλα Doubao Open Platform
2. Μεταβείτε στην εφαρμογή αναγνώρισης ομιλίας σας
3. Βρείτε την περιοχή διαχείρισης κλειδιών API
4. Δημιουργήστε ή αντιγράψτε το `Access Token`

:::σημείωση
Παρακαλώ κρατήστε το δικό σας `Access Token` ασφαλές και μην το μοιράζεστε με άλλους. Εάν πρέπει να το αντικαταστήσετε, μπορείτε να το αναδημιουργήσετε στην Ανοιχτή Πλατφόρμα Doubao.
:::

### Διαμόρφωση στο προϊόν

1. Ανοίξτε την εφαρμογή HagiCode
2. Μεταβείτε στις **Ρυθμίσεις** → **Ρυθμίσεις αναγνώρισης φωνής**
3. Συμπληρώστε τις παρακάτω πληροφορίες στη φόρμα διαμόρφωσης:
   - **Παροχέας**: Επιλέξτε `doubao` (Ντουμπάο)
   - **APP ID**: Επικολλήστε το APP ID που λάβατε από την πλατφόρμα Doubao
   - **Access Token**: Επικολλήστε το Access Token που πήρατε από την πλατφόρμα Doubao
4. (Προαιρετικό) Προσαρμόστε άλλες παραμέτρους διαμόρφωσης όπως απαιτείται
5. Κάντε κλικ στο κουμπί **Δοκιμή κλειδιού API** για να επαληθεύσετε τη διαμόρφωση
6. Μετά την επιτυχή επαλήθευση, η διαμόρφωση αποθηκεύεται αυτόματα στην τοπική αποθήκευση του προγράμματος περιήγησης

### Δοκιμή κλειδιού API

Μετά τη διαμόρφωση, συνιστάται να ελέγξετε εάν το κλειδί API είναι έγκυρο:

1. Κάντε κλικ στο κουμπί **Δοκιμή κλειδιού API**
2. Το σύστημα θα καλέσει τη διεπαφή δοκιμής για να επαληθεύσει τη διαμόρφωσή σας
3. Εάν η διαμόρφωση είναι σωστή, θα εμφανιστεί ένα μήνυμα επιτυχίας
4. Εάν η διαμόρφωση είναι λανθασμένη, θα εμφανιστεί ένα μήνυμα σφάλματος. Ελέγξτε:
   - Αν το APP ID και το Access Token είναι σωστά
   - Εάν η σύνδεση δικτύου είναι κανονική
   - Εάν το Access Token έχει λήξει

:::συμβουλή
Εάν η δοκιμή αποτύχει, ελέγξτε πρώτα εάν η σύνδεση δικτύου είναι κανονική και, στη συνέχεια, επιβεβαιώστε ότι το κλειδί API από την Open Platform του Doubao είναι έγκυρο και δεν έχει λήξει.
:::

## Χρήση

### Εύρεση θέσης φωνητικής εισαγωγής

Στο HagiCode, τα πλαίσια εισαγωγής κειμένου με εικονίδιο μικροφώνου υποστηρίζουν φωνητική εισαγωγή. Αυτά τα στοιχεία συνήθως ονομάζονται `VoiceTextArea`.

Οι κοινές τοποθεσίες φωνητικής εισαγωγής περιλαμβάνουν:
- Πλαίσια εισαγωγής μηνυμάτων
- Περιοχές επεξεργασίας κειμένου
- Διάφορα πεδία φόρμας που απαιτούν εισαγωγή κειμένου

### Ξεκινήστε την Αναγνώριση φωνής

1. Βρείτε το πλαίσιο εισόδου με ένα εικονίδιο μικροφώνου
2. Κάντε κλικ στο κουμπί του μικροφώνου
3. Το πρόγραμμα περιήγησης θα ζητήσει άδεια μικροφώνου
4. Κάντε κλικ στο **Να επιτρέπεται** για να εξουσιοδοτήσετε την πρόσβαση στο μικρόφωνο

:::σημείωση
Εάν έχετε εξουσιοδοτήσει προηγουμένως το μικρόφωνο, το πρόγραμμα περιήγησης ενδέχεται να μην ζητήσει ξανά άδεια. Για να ελέγξετε ή να αλλάξετε τις ρυθμίσεις αδειών, μπορείτε να τις προβάλετε στο εικονίδιο κλειδώματος στα αριστερά της γραμμής διευθύνσεων του προγράμματος περιήγησης.
:::

### Περιγραφή κατάστασης εγγραφής

Μετά την επιτυχή εξουσιοδότηση, η αναγνώριση φωνής θα ξεκινήσει αυτόματα και μπορείτε να δείτε:

- **Κινούμενη μορφή κυματομορφής**: Η κυματομορφή δυναμικού ήχου θα εμφανιστεί μέσα στο κουμπί του μικροφώνου, υποδεικνύοντας ότι η εγγραφή βρίσκεται σε εξέλιξη
- **Εμφάνιση διάρκειας**: Η τρέχουσα διάρκεια εγγραφής θα εμφανιστεί κάτω από το κουμπί
- **Αναγνώριση σε πραγματικό χρόνο**: Το αναγνωρισμένο κείμενο θα εμφανίζεται προσωρινά στη θέση του δρομέα

### Εμφάνιση αποτελεσμάτων αναγνώρισης σε πραγματικό χρόνο

Κατά την εγγραφή, η μηχανή αναγνώρισης ομιλίας θα μετατρέψει την ομιλία σας σε κείμενο σε πραγματικό χρόνο:

- Το αναγνωρισμένο κείμενο θα εμφανίζεται προσωρινά με γκρι χρώμα στο πλαίσιο εισαγωγής
- Τα αποτελέσματα της αναγνώρισης θα ενημερώνονται συνεχώς καθώς μιλάτε
- Υποστηρίζει την αναγνώριση κινεζικής γλώσσας Mandarin με υψηλή ακρίβεια

:::συμβουλή
Για καλύτερα αποτελέσματα αναγνώρισης, συνιστάται:
- Χρήση σε ήσυχο περιβάλλον
- Μιλήστε καθαρά και με μέτριο ρυθμό
- Αποφύγετε την παρεμβολή θορύβου περιβάλλοντος
:::

### Διακοπή Αναγνώρισης

Για να διακόψετε τη φωνητική αναγνώριση, μπορείτε:

1. **Κάντε κλικ στο κουμπί του μικροφώνου**: Κάντε ξανά κλικ στο κουμπί για να σταματήσετε την εγγραφή
2. **Κάντε κλικ στο πλαίσιο εισαγωγής**: Κάντε κλικ σε άλλες περιοχές του πλαισίου εισαγωγής θα σταματήσει επίσης η εγγραφή

Μετά τη διακοπή, το τελικό αναγνωρισμένο κείμενο θα εισαχθεί επίσημα στο πλαίσιο εισαγωγής και μπορείτε να συνεχίσετε την επεξεργασία ή την αποστολή.
