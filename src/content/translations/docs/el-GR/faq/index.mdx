## Τι καλύπτει αυτή η ενότητα

Η ενότητα Συχνές Ερωτήσεις είναι το μέρος για:

- προειδοποιήσεις εκκίνησης και χρόνου εκτέλεσης
- σημειώσεις συμβατότητας που συνδέονται με πλατφόρμες, κανάλια διανομής, συσκευασίες ή συνδυασμένους χρόνους εκτέλεσης
- αντιμετώπιση προβλημάτων για θέματα που αξίζουν μια σταθερή αυτόνομη σελίδα επεξήγησης

Εάν πρέπει να επιβεβαιώσετε ποιον Node Desktop χρησιμοποιεί ή γιατί αναφέρουν τα διαγνωστικά `portable-fixed/toolchain`, ανοιχτό [Περιβάλλον κόμβου επιφάνειας εργασίας](/el-GR/faq/desktop-node-environment).

Αν θέλετε να μάθετε γιατί το HagiCode δεν είναι στο Steam, ανοίξτε το [Γιατί το HagiCode δεν είναι διαθέσιμο στο Steam](/el-GR/faq/steam-distribution-status).
