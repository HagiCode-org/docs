Το HagiCode υποστηρίζει πολλαπλές διαδρομές εγκατάστασης. Ξεκινήστε με τη μέθοδο ανάπτυξης που ταιριάζει στο σενάριο σας και, στη συνέχεια, προσθέστε λογισμικό υποστήριξης μόνο όταν το χρειάζεστε.

Το τρέχον εύρος υποστήριξης του ενεργού Agent CLI που σχετίζεται περισσότερο με τη ρύθμιση είναι: Claude Code, Codex, GitHub Copilot, OpenCode, Hermes, QoderCLI, Kiro, Kimi, Gemini, DeepAgents και Codebuddy. Οι Hermes, Gemini και DeepAgents χρησιμοποιούν αυτήν τη στιγμή ροές μη αυτόματης εγκατάστασης ή ελέγχου ταυτότητας. Τα άλλα CLI εκθέτουν σημεία εισόδου εγκατάστασης με ένα κλικ ή καθοδηγούμενες ροές εγκατάστασης.

## Προτεινόμενα επόμενα βήματα

- Εάν σκοπεύετε να αγοράσετε και να εγκαταστήσετε το HagiCode μέσω του Steam, ξεκινήστε με [Εγκατάσταση Steam](/el-GR/installation/steam).
- Εάν είστε νέοι στο HagiCode, ξεκινήστε με [Εγκατάσταση στην επιφάνεια εργασίας](/el-GR/installation/desktop).
- Εάν κάνετε ανάπτυξη για περιβάλλον διακομιστή ή ομάδας, χρησιμοποιήστε το [Εγκατάσταση Docker Compose](/el-GR/installation/docker-compose).
- Εάν πρέπει να επιβεβαιώσετε σημειώσεις κατάστασης υποστήριξης, λειτουργίας ρύθμισης ή ιστορικής συμβατότητας για ένα CLI, ανοίξτε [Εγκατάσταση AI Agent CLI](/el-GR/related-software-installation/ai-agent-cli/).
