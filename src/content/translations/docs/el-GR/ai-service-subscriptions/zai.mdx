## Τοποθέτηση

Το ZAI είναι ένας από τους παρόχους που προτείνονται επί του παρόντος από το προκαθορισμένο σύνολο εγγράφων για τον Κώδικα Claude. Σχέδια κωδικοποίησης GLM με αντιστοίχιση επιπέδων για προγραμματιστές που θέλουν τον Claude Code ως καθημερινό πρόγραμμα οδήγησης.

## Καλή εφαρμογή για

- Θέλετε ένα πρόγραμμα συνδρομής εστιασμένο στην κωδικοποίηση
- Χρειάζεστε αντιστοιχισμένες προεπιλογές Sonnet/Opus/Haiku
- Αναμένετε από έναν πάροχο να εργάζεται σε Claude Code και παρόμοια εργαλεία κωδικοποίησης

## Βασικά πράγματα που πρέπει να προσέξετε

- Τα τρέχοντα προκαθορισμένα μεταδεδομένα επισημαίνουν το ZAI ως συνιστάται.
- Εκθέτει ένα τελικό σημείο συμβατό με Anthropic που μπορεί να χρησιμοποιηθεί απευθείας από τον Claude Code.
- Η προεπιλογή αποστέλλει προεπιλεγμένες αντιστοιχίσεις για `glm-4.7`, `glm-5`, και `glm-4.5-air`.

## Στιγμιότυπο ενσωμάτωσης

| Στοιχείο | Αξία |
|------|-------|
| Καταχώρηση συνδρομής | [Ανοίξτε τη σελίδα συνδρομής](https://www.bigmodel.cn/claude-code?ic=14BY54APZA) |
| Επίσημα έγγραφα / κονσόλα | [Ανοίξτε την επίσημη σελίδα](https://open.bigmodel.cn/dev/api) |
| Διεύθυνση URL βάσης συμβατή με Anthropic | `https://open.bigmodel.cn/api/anthropic` |
| Συνιστώμενη auth env var | `ANTHROPIC_AUTH_TOKEN` |

## Προεπιλεγμένη αντιστοίχιση μοντέλου

| βαθμίδα Claude | Προεπιλεγμένο μοντέλο |
|-------------|---------------|
| Σονέτο | `glm-4.7` |
| Opus | `glm-5` |
| Χαϊκού | `glm-4.5-air` |

:::συμβουλή
Οι τιμές, οι προσφορές και οι λεπτομέρειες του πακέτου μπορούν να αλλάξουν με την πάροδο του χρόνου. Εάν το κόστος είναι το κύριο μέλημά σας, επιβεβαιώστε τις πιο πρόσφατες πληροφορίες απευθείας στη σελίδα του παρόχου.
:::

## Προτεινόμενα επόμενα βήματα

1. Επιβεβαιώστε ποιο επίπεδο συνδρομής ή διαδρομή ενεργοποίησης API θέλετε να χρησιμοποιήσετε.
2. Ανοίξτε την καταχώρηση Claude Code στον ενοποιημένο κόμβο AI Agent CLI και συνεχίστε με τα επίσημα έγγραφα εγκατάστασης και αναφοράς της Anthropic.
3. Επικυρώστε τη σύνδεση τοπικά μετά τη ρύθμιση.
