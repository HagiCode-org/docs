## Κρατικό γλωσσάρι

- **Υποστηρίζεται αυτήν τη στιγμή** σημαίνει ότι το CLI ανήκει στην τρέχουσα ενεργή περιοχή υποστήριξης HagiCode.
- **Εγκατάσταση με ένα κλικ** σημαίνει ότι το HagiCode παρέχει ένα κατευθυνόμενο σημείο εισόδου εγκατάστασης. Ορισμένα CLI εξακολουθούν να απαιτούν σύνδεση προμηθευτή ή βήμα εξουσιοδότησης μετά την εγκατάσταση.
- **Μη αυτόματη εγκατάσταση / έλεγχος ταυτότητας** σημαίνει ότι το CLI υποστηρίζεται τώρα, αλλά πρέπει να ολοκληρώσετε την τοπική εγκατάσταση ή τη ροή εξουσιοδότησης προτού το HagiCode μπορέσει να το εντοπίσει και να το χρησιμοποιήσει.
- **Μόνο συμβατότητα παλαιού τύπου** σημαίνει μια τιμή που έχει αποσυρθεί όπως π.χ `IFlowCli` εξακολουθεί να υπάρχει για συμβατότητα ιστορικών δεδομένων, αλλά δεν αποτελεί μέρος του τρέχοντος εύρους ενεργής υποστήριξης.
