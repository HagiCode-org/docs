Esta página é a entrada de instalação local unificada da CLI do AI Agent em `repos/docs`. HagiCode atualmente suporta 11 Agent CLIs ativos, seguindo o backend `PrimaryProfessions` pedido: Claude Code, Codex, GitHub Copilot, OpenCode, Hermes, QoderCLI, Kiro, Kimi, Gemini, DeepAgents e Codebuddy.

:::dica[Como usar esta página]
Verifique primeiro a tabela para obter o estado de suporte atual e o modo de configuração e, em seguida, continue com os documentos de instalação, autenticação, ACP, permissão e solução de problemas mantidos pelo fornecedor. O escopo do suporte e o modo de configuração são preocupações distintas.
:::

## Glossário estadual

- **Atualmente suportado** significa que a CLI pertence à faixa de suporte ativa atual do HagiCode.
- **Instalação com um clique** significa que o HagiCode fornece um ponto de entrada de instalação guiada; algumas CLIs ainda exigem login do fornecedor ou etapa de autenticação após a instalação.
- **Instalação/autenticação manual** significa que a CLI é suportada agora, mas você deve concluir a instalação local ou o fluxo de autenticação antes que o HagiCode possa detectá-la e usá-la.
- **Somente compatibilidade herdada** significa um valor retirado, como `IFlowCli` ainda existe para compatibilidade de dados históricos, mas não faz parte da faixa de suporte ativo atual.
