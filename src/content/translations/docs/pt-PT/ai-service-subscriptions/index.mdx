Este capítulo coleta os sites de assinatura de serviços de IA atualmente marcados como recomendados em `repos/docs/public/presets/claude-code/providers/*.json`. Escolha um provedor aqui primeiro e depois prossiga para a configuração do Claude Code.

A partir de **9 de março de 2026**, este capítulo abrange: **ZAI, MiniMax, Alibaba Cloud DashScope e plano de codificação Volcengine**.

:::dica[Ordem de leitura sugerida]
Se você ainda não escolheu um fornecedor, comece com esta página para comparação. Se você já sabe qual provedor deseja, vá primeiro para a página de detalhes e depois prossiga para a entrada do Código Claude no hub unificado da CLI do AI Agent.
:::

## Provedores recomendados atuais

| Provedor | Força | Bom ajuste para | Página de detalhes |
|----------|----------|--------------|-------------|
| ZAI | Planos focados em codificação com mapeamento de camadas | Desenvolvedores que desejam Claude Code como motorista diário | [Ver detalhes](./zai/) |
| MiniMax | Um modelo padrão em todas as camadas, integração simples | Equipes otimizando para configuração de baixo atrito | [Ver detalhes](./minimax/) |
| Alibaba Cloud DashScope | Alinhamento do console em nuvem e gerenciamento unificado | Equipes já trabalhando no Alibaba Cloud | [Ver detalhes](./aliyun/) |
| Plano de codificação Volcengine | Forte extensão do ecossistema e sinais de plataforma multimodelo | Desenvolvedores que desejam alinhamento do ecossistema Volcengine | [Ver detalhes](./volcengine/) |

:::nota
`Anthropic Official` permanece utilizável, mas não está incluído no primeiro lote de páginas dedicadas porque os metadados predefinidos atuais não o marcam como recomendado.
:::

## Dicas de seleção rápida

- **Precisa de planos focados em codificação e mapeamento de níveis mais claro**: comece com [ZAI](./zai/)
- **Precisa primeiro do caminho de configuração mais fácil**: comece com [MiniMax](./minimax/)
- **Já executado dentro do Alibaba Cloud**: comece com [Alibaba Cloud DashScope](./aliyun/)
- **Deseja alinhamento do ecossistema Volcengine ou alcance de plataforma multimodelo**: comece com [Plano de codificação Volcengine](./volcengine/)
