## Mitä tämä osio kattaa

UKK-osio on paikka:

- käynnistys- ja ajonaikaiset varoitukset
- alustoihin, jakelukanaviin, pakkauksiin tai niputettuihin suoritusaikoihin sidotut yhteensopivuushuomautukset
- vianetsintäaiheita, jotka ansaitsevat vakaan erillisen selityssivun

Jos sinun on vahvistettava, mitä Node Desktop käyttää tai miksi diagnostiikka mainitsee `portable-fixed/toolchain`, auki [Työpöytäsolmuympäristö](/fi-FI/faq/desktop-node-environment).
