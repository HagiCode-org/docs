## Paikannus

ZAI on yksi Claude Coden dokumenttien esiasetuksen tällä hetkellä suosittelemista tarjoajista. GLM-koodaussuunnitelmat tasokartoituksella kehittäjille, jotka haluavat Claude Coden päivittäiseksi kuljettajaksi.

## Sopii hyvin

- Haluat koodaukseen keskittyvän tilaussuunnitelman
- Tarvitset kartoitetut Sonnet/Opus/Haiku-oletukset
- Odotat yhden palveluntarjoajan toimivan Claude Coden ja vastaavien koodaustyökalujen välillä

## Keskeisiä huomioitavia asioita

- Nykyiset esiasetetut metatiedot merkitsevät ZAI:ta suositelluksi.
- Se paljastaa Anthropic-yhteensopivan päätepisteen, jota Claude Code voi käyttää suoraan.
- Esiasetus toimittaa oletuskartoitukset kohteelle `glm-4.7`, `glm-5`, ja `glm-4.5-air`.

## Integraation tilannekuva

| Tuote | Arvo |
|------|-------|
| Tilausmerkintä | [Avaa tilaussivu](https://www.bigmodel.cn/claude-code?ic=14BY54APZA) |
| Viralliset asiakirjat / konsoli | [Avaa virallinen sivu](https://open.bigmodel.cn/dev/api) |
| Ihmisten kanssa yhteensopiva perus-URL-osoite | `https://open.bigmodel.cn/api/anthropic` |
| Suositeltu auth env var | `ANTHROPIC_AUTH_TOKEN` |

## Mallin oletuskartoitus

| Claude taso | Oletusmalli |
|-------------|---------------|
| Sonetti | `glm-4.7` |
| Opus | `glm-5` |
| Haiku | `glm-4.5-air` |

:::vinkki
Hinnat, tarjoukset ja paketin tiedot voivat muuttua ajan myötä. Jos hinta on tärkein huolenaihe, vahvista viimeisimmät tiedot suoraan palveluntarjoajan sivulta.
:::

## Ehdotetut seuraavat vaiheet

1. Vahvista, mitä tilaustasoa tai API-aktivointipolkua haluat käyttää.
2. Avaa Claude Code -merkintä yhdistetyssä AI Agent CLI -keskittimessä ja jatka Anthropicin virallisilla asennus- ja viiteasiakirjoilla.
3. Vahvista yhteys paikallisesti asennuksen jälkeen.
