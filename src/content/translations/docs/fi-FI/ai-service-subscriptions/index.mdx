Tämä luku kerää AI-palvelun tilaussivustot, jotka on tällä hetkellä merkitty suositelluiksi `repos/docs/public/presets/claude-code/providers/*.json`. Valitse ensin palveluntarjoaja tästä ja jatka sitten Claude Coden määritykseen.

Tämä luku kattaa **9. maaliskuuta 2026** alkaen: **ZAI, MiniMax, Alibaba Cloud DashScope ja Volcengine Coding Plan**.

:::vinkki[Suositeltu lukujärjestys]
Jos et ole vielä valinnut palveluntarjoajaa, aloita tältä sivulta vertailua varten. Jos tiedät jo haluamasi palveluntarjoajan, siirry ensin sen tietosivulle ja jatka sitten Claude Code -merkintään yhdistetyssä AI Agent CLI -keskittimessä.
:::

## Nykyiset suositellut palveluntarjoajat

| Palveluntarjoaja | Vahvuus | Sopii hyvin | Yksityiskohtaiset sivut |
|----------|----------|--------------|-------------|
| ZAI | Koodaukseen keskittyvät suunnitelmat tasokartoituksella | Kehittäjät, jotka haluavat Claude Coden päivittäiseksi kuljettajaksi | [Näytä tiedot](./zai/) |
| MiniMax | Yksi oletusmalli tasoilla, yksinkertainen käyttöönotto | Tiimit optimoivat alhaisen kitkan asennukseen | [Näytä tiedot](./minimax/) |
| Alibaba Cloud DashScope | Pilvikonsoli-kohdistus ja yhtenäinen hallinta | Alibaba Cloudissa jo työskentelevät tiimit | [Näytä tiedot](./aliyun/) |
| Volsengin koodaussuunnitelma | Vahvat ekosysteemin laajennus- ja usean mallin alustan signaalit | Kehittäjät, jotka haluavat Volcengine-ekosysteemin yhdenmukaistamisen | [Näytä tiedot](./volcengine/) |

:::huomautus
`Anthropic Official` pysyy käyttökelpoisena, mutta sitä ei sisällytetä ensimmäiseen omistettujen sivujen joukkoon, koska nykyiset esiasetetut metatiedot eivät merkitse sitä suositelluksi.
:::
