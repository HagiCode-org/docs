HagiCode tukee useita asennuspolkuja. Aloita skenaarioosi sopivalla käyttöönottomenetelmällä ja lisää sitten tukiohjelmisto vain silloin, kun tarvitset sitä.

Nykyinen aktiivinen Agent CLI -tukialue, joka on olennaisin asennuksen kannalta, on: Claude Code, Codex, GitHub Copilot, OpenCode, Hermes, QoderCLI, Kiro, Kimi, Gemini, DeepAgents ja Codebuddy. Hermes, Gemini ja DeepAgents käyttävät tällä hetkellä manuaalisia asennus- tai todennuskulkuja; muut CLI:t paljastavat yhden napsautuksen asennuksen aloituspisteet tai ohjatut asennustyöt.

## Suositellut seuraavat vaiheet

- Jos aiot ostaa ja asentaa HagiCoden Steamin kautta, aloita [Steamin asennus](/fi-FI/installation/steam).
- Jos olet uusi HagiCodessa, aloita [Työpöytäasennus](/fi-FI/installation/desktop).
- Jos otat käyttöön palvelin- tai ryhmäympäristön, käytä [Docker Compose -asennus](/fi-FI/installation/docker-compose).
- Jos sinun on vahvistettava CLI:n tukitila, asennustila tai historialliset yhteensopivuushuomautukset, avaa [AI Agent CLI -asennus](/fi-FI/related-software-installation/ai-agent-cli/).
