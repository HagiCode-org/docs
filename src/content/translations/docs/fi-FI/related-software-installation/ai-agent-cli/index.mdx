Tämä sivu on yhtenäinen paikallinen AI Agent CLI -asennusmerkintä `repos/docs`. HagiCode tukee tällä hetkellä 11 aktiivista agentin CLI:tä taustajärjestelmän jälkeen `PrimaryProfessions` järjestys: Claude Code, Codex, GitHub Copilot, OpenCode, Hermes, QoderCLI, Kiro, Kimi, Gemini, DeepAgents ja Codebuddy.

:::vinkki [Kuinka tätä sivua käytetään]
Tarkista ensin nykyinen tukitila ja asennustila taulukosta ja jatka sitten toimittajan ylläpitämillä asennus-, todennus-, ACP-, käyttöoikeus- ja vianmääritysasiakirjoilla. Tuen laajuus ja asennustila ovat erillisiä huolenaiheita.
:::

## Valtion sanasto

- **Tällä hetkellä tuettu** tarkoittaa, että CLI kuuluu nykyiseen aktiiviseen HagiCode-tukialueeseen.
- **Asenna yhdellä napsautuksella** tarkoittaa, että HagiCode tarjoaa ohjatun asennuksen aloituskohdan; jotkin CLI:t vaativat edelleen toimittajan kirjautumisen tai todennusvaiheen asennuksen jälkeen.
- **Manuaalinen asennus/todennus** tarkoittaa, että CLI:tä tuetaan nyt, mutta sinun on suoritettava paikallinen asennus tai todennus, ennen kuin HagiCode voi havaita ja käyttää sitä.
- **Vain vanha yhteensopivuus** tarkoittaa vanhentunutta arvoa, kuten `IFlowCli` on edelleen olemassa historiallisten tietojen yhteensopivuuden vuoksi, mutta se ei ole osa nykyistä aktiivista tukialuetta.

- Jatka kanssa [Asenna Node.js](/en/related-software-installation/nodejs/installation) ja [Asenna OpenSpec](/en/related-software-installation/openspec/setup-openspec) jos tarvitset vielä esitietoja.
- Jos haluat yhden paikallisen yhdyskäytävän useille OpenAI-yhteensopiville työkaluille, jatka [OmniRouten asennus ja käyttö](/en/related-software-installation/omniroute/).
- Palaa kohtaan [Asennusopas](/en/installation) laajempaa asennuspolkua varten.
- Jos saavuit tänne historiallisesta blogikirjoituksesta, pidä tätä sivua ja asennuskuvaa nykyisenä ikivihreänä totuuden lähteenä.
