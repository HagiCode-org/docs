## 서비스 포지셔닝

알리윈 DashScope는 현재 문서 preset에서 추천하는 Claude Code 구독 진입점 중 하나입니다. 이미 알리윈 생태계에서 작업 중인 팀에 적합하며, 구독, 콘솔, 키 관리를 클라우드 플랫폼에서 통합적으로 관리할 수 있습니다.

## 적합 대상

- 이미 알리윈 콘솔이나 클라우드 리소스를 사용 중인 사용자
- 통합 클라우드 플랫폼에서 서비스 및 API Key를 관리하고 싶은 사용자
- 고정 모델 매핑으로 Claude Code에 빠르게 연동하고 싶은 사용자

## 먼저 확인해야 할 사항

- 현재 preset에서 알리윈 DashScope를 추천 서비스 제공자로 표시합니다.
- `glm-4.7`을 기본 매핑 모델로 사용하여 설정 경로가 명확합니다.
- 팀에 이미 알리윈 계정 체계가 있다면 연동 및 권한 관리가 더 원활합니다.

## 핵심 연동 정보

| 항목 | 내용 |
|------|------|
| 구독 진입점 | [구독 페이지로 이동](https://www.aliyun.com/benefit/ai/aistar?userCode=vmx5szbq&clubBiz=subTask..12384055..10263..) |
| 공식 문서 / 콘솔 | [공식 페이지 보기](https://dashscope.aliyun.com) |
| Anthropic 호환 Base URL | `https://coding.dashscope.aliyuncs.com/apps/anthropic` |
| 권장 인증 환경 변수 | `ANTHROPIC_AUTH_TOKEN` |

## 기본 모델 매핑

| Claude 계층 | 기본 모델 |
|-------------|----------|
| Sonnet | `glm-4.7` |
| Opus | `glm-4.7` |
| Haiku | `glm-4.7` |

:::tip
플랜, 할인 및 혜택은 시간에 따라 변동될 수 있습니다. 실시간 가격 정보가 필요하면 서비스 제공자의 구독 페이지를 직접 확인하세요.
:::

## 설정 권장 사항

1. 먼저 사용할 플랜 또는 API Key 개통 방식을 확인합니다.
2. 그다음 통합 AI Agent CLI 페이지의 Claude Code 항목을 열어 Anthropic 공식 설치 및 참고 문서로 이동합니다.
3. 완료 후 로컬에서 `claude --version`, 설정 파일 확인 또는 실제 요청을 실행하여 연동이 성공했는지 검증합니다.
