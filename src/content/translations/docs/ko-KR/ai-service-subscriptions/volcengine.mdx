## 서비스 포지셔닝

화산엔진 Coding Plan은 현재 문서 preset에서 추천하는 Claude Code 구독 진입점 중 하나입니다. 플랫폼 프로모션에 관심이 있고 화산엔진 생태계에서 다중 모델 기능을 사용하려는 개발자에게 적합합니다.

## 적합 대상

- 화산엔진 생태계에서 AI 도구 체인을 계속 확장하고 싶은 사용자
- 다중 모델 플랫폼 기능과 프로모션 진입점에 관심 있는 사용자
- Anthropic 호환 진입점으로 Claude Code를 연동해야 하는 사용자

## 먼저 확인해야 할 사항

- 현재 preset에서 화산엔진 Coding Plan을 추천 서비스 제공자로 표시합니다.
- preset 설명에 Doubao, GLM, DeepSeek, Kimi 등 모델 생태계를 지원한다고 명시되어 있습니다.
- 프로모션 및 가격은 빠르게 변동될 수 있으므로, 구체적인 혜택은 서비스 제공자의 구독 페이지 실시간 안내를 기준으로 확인하세요.

## 설정 권장 사항

1. 먼저 사용할 플랜 또는 API Key 개통 방식을 확인합니다.
2. 그다음 통합 AI Agent CLI 페이지의 Claude Code 항목을 열어 Anthropic 공식 설치 및 참고 문서로 이동합니다.
3. 완료 후 로컬에서 `claude --version`, 설정 파일 확인 또는 실제 요청을 실행하여 연동이 성공했는지 검증합니다.
