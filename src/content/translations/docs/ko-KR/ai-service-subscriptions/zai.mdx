## 서비스 포지셔닝

Zhipu AI는 현재 문서 preset에서 추천하는 Claude Code 구독 진입점 중 하나입니다. GLM Coding 플랜과 다중 모델 매핑을 제공하며, Claude Code를 일상 주력 도구로 사용하려는 개발자에게 적합합니다.

## 적합 대상

- 코딩 시나리오에 특화된 플랜을 원하는 사용자
- 비교적 완전한 Claude 계층 모델 매핑이 필요한 사용자
- 하나의 서비스로 Claude Code, Cline 등 여러 도구를 커버하고 싶은 사용자

## 먼저 확인해야 할 사항

- 현재 preset에서 Zhipu AI를 추천 서비스 제공자로 표시합니다.
- Anthropic 호환 인터페이스를 제공하여 Claude Code와 직접 연동이 가능합니다.
- preset에 `glm-4.7`, `glm-5`, `glm-4.5-air` 등의 기본 모델 매핑이 포함되어 있습니다.

## 핵심 연동 정보

| 항목 | 내용 |
|------|------|
| 구독 진입점 | [구독 페이지로 이동](https://www.bigmodel.cn/claude-code?ic=14BY54APZA) |
| 공식 문서 / 콘솔 | [공식 페이지 보기](https://open.bigmodel.cn/dev/api) |
| Anthropic 호환 Base URL | `https://open.bigmodel.cn/api/anthropic` |
| 권장 인증 환경 변수 | `ANTHROPIC_AUTH_TOKEN` |

## 기본 모델 매핑

| Claude 계층 | 기본 모델 |
|-------------|----------|
| Sonnet | `glm-4.7` |
| Opus | `glm-5` |
| Haiku | `glm-4.5-air` |

:::tip
플랜, 할인 및 혜택은 시간에 따라 변동될 수 있습니다. 실시간 가격 정보가 필요하면 서비스 제공자의 구독 페이지를 직접 확인하세요.
:::

## 설정 권장 사항

1. 먼저 사용할 플랜 또는 API Key 개통 방식을 확인합니다.
2. 그다음 통합 AI Agent CLI 페이지의 Claude Code 항목을 열어 Anthropic 공식 설치 및 참고 문서로 이동합니다.
3. 완료 후 로컬에서 `claude --version`, 설정 파일 확인 또는 실제 요청을 실행하여 연동이 성공했는지 검증합니다.
