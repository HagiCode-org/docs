`HagiCode: Turbo Engine DLC`는 세 가지 DLC 중 "효율 업그레이드"에 가장 가까운 제품이며, 헤비 유저가 가장 큰 차이를 느낄 수 있는 DLC입니다. 외형 보충과 달리 기본 기능이 더 높은 빈도와 복잡성의 워크플로우를 충분히 지원할 수 있는지를 해결합니다.

## 우선 구매를 추천하는 이유

무료 버전과 Steam 본체 버전은 기본적으로 3개의 제안 동시 실행 상한을 포함합니다. 생성, 실행, 아카이브 상태의 제안이 이 상한을 공동으로 차지합니다.

Turbo Engine DLC의 핵심 역할은 이 기본 상한을 32로 확장하여, 더 집중적인 병렬 워크플로우에서 여러 제안을 동시에 진행할 수 있게 하는 것입니다. 이 향상은 단순히 수치상으로 좋아 보이는 것이 아니라, 여러 요구사항, 여러 구현 방향, 여러 정리 작업을 원활하게 병렬로 진행할 수 있는지에 직접적인 영향을 미칩니다.

이미 "가끔 HagiCode를 사용해보는" 단계에서 "매일 실제 작업을 진행하는" 단계로 넘어왔다면, Turbo Engine DLC가 가장 우선적으로 구매할 만한 DLC일 것입니다.

## 핵심 업그레이드 포인트

동시 실행 확장 외에 Turbo Engine DLC는 현재 다음과 같이 구현된 기능도 명확하게 해금합니다:

- 확장 문서 테마 (기본 Agent CLI 베이스라인 테마에만 국한되지 않음)
- AI Compose Commit의 `Co-Authored-By` 기본 전략 및 단일 실행 오버라이드
- Hero 워크플로우의 공식 DLC 아바타 리소스 및 커스텀 아바타 업로드
- Hagiode Plus 헤더 브랜드 커스터마이징 (로고, 앱 타이틀, 브라우저 타이틀 포함)

이러한 향상들이 함께 제공되는 것은 두 가지 측면의 업그레이드를 의미합니다:

- 워크플로우 측면에서 더 강력해져, 더 높은 밀도의 제안 병렬 처리를 감당할 수 있습니다.
- 표현 측면에서 더 자유로워져, Steam 버전 워크벤치를 팀, 프로젝트 또는 개인 취향에 더 맞게 커스터마이징할 수 있습니다.

이것이 Turbo Engine DLC가 단순한 "기능 추가"가 아니라, Steam 버전을 기본 상태에서 고강도 사용 상태로 바로 끌어올리는 이유입니다.

## 현재 확인 가능한 지원 기능

"지금까지 어디까지 지원되는지"가 더 궁금하다면, 현재 구현된 다음 기능 그룹으로 이해할 수 있습니다:

- **제안 동시 실행 상한이 3에서 32로 확장.** 여전히 Turbo Engine DLC의 가장 핵심적이고 체감이 큰 업그레이드입니다. 생성, 실행, 아카이브 중인 제안이 동시 실행 슬롯을 공동으로 차지하므로, 높은 상한은 여러 작업 라인을 안정적으로 병렬 진행할 수 있는지에 직접적인 영향을 미칩니다.
- **확장 문서 테마.** 단순한 "문구 전환"이 아니라, 모험단 관리 관련 인터페이스에서 확장 Document Theme을 전환하고 워크벤치, 히스토리 영역, 기본 편성이 다른 테마에서 어떻게 표시되는지 실시간으로 미리볼 수 있습니다.
- **AI Compose Commit의 `Co-Authored-By` 커스터마이징.** Git 설정에서 기본 `Co-Authored-By` 패턴을 저장할 수 있고, 단일 AI Compose 실행에서 임시로 커스텀 공동 작성자 이름과 이메일로 변경하거나 이번 실행에서는 해당 트레일러를 추가하지 않을 수도 있습니다.
- **브랜드 커스터마이징.** Turbo Engine DLC가 활성화되면 헤더 로고를 업로드하고, 앱 타이틀, 브라우저 타이틀, 브랜드 클릭 이동 링크를 설정할 수 있습니다. 즉, 단순히 왼쪽 상단 아이콘을 변경하는 것이 아니라 워크벤치 헤더 브랜드 전체를 커스터마이징할 수 있습니다.
- **커스텀 아바타 업로드 및 공유 재사용.** 현재 Hero의 이미지를 변경하는 것에 그치지 않습니다. 현재 Hero의 커스텀 아바타를 업로드할 수 있고, 공유 커스텀 아바타 라이브러리를 새로 생성, 편집, 일괄 생성 프로세스에서 재사용할 수 있습니다.
- **공식 DLC 아바타 리소스 런타임 갤러리 연동.** Turbo Engine DLC의 아바타 리소스는 Hero Avatar Gallery 및 일괄 생성 등 프로세스에 참여하므로, 프로모션 페이지에만 있는 소재 팩이 아니라 실제 워크플로우에 연동된 아바타 에셋입니다.

즉, Turbo Engine DLC는 현재 "더 높은 동시 실행 + 더 강력한 Git 워크플로우 + 더 깊은 개인화 설정"의 업그레이드 체인을 커버하며, 단순히 동시 실행 숫자가 더 큰 것만은 아닙니다.

## 추천 대상

- 여러 제안, 여러 구현 브랜치 또는 여러 문서 작업을 동시에 진행하는 사용자.
- 기본 3 동시 실행보다 더 높은 상한이 필요한 헤비 사용자.
- Steam 워크벤치에서 확장 문서 테마, Git `Co-Authored-By` 커스터마이징, 아바타 확장, 커스텀 아바타 및 브랜드 커스터마이징을 활성화하려는 사용자.

이미 3 동시 실행이 부족하다고 느끼거나, 인터페이스에 더 강한 커스터마이징 표현을 원한다면 이 DLC는 필수적 업그레이드입니다.
