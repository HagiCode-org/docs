Turbo Engine은 HagiCode Desktop의 핵심 확장 애드온으로, 고빈도 사용 단계에 접어든 사용자를 위한 것입니다. **HagiCode를 기본 모드에서 고강도 작업 모드로 전환**하는 한 가지 목표를 해결합니다.

## 구매 방법

Turbo Engine은 Microsoft Store의 일회성 영구 라이선스(스토어 ID: `9NSD809W18Z6`)로, 구매 후 영구적으로 적용됩니다. HagiCode Desktop의 구독 작업 영역에서 직접 구매 및 상태 새로고침을 완료할 수 있습니다.

## 해금되는 기능

Turbo Engine 구매 후 다음 기능이 영구적으로 해금됩니다:

| 기능 | 설명 |
| --- | --- |
| **제안 동시 실행 3 → 32** | 생성, 실행, 아카이브 중인 제안이 이 상한을 공유합니다. 32 동시 실행으로 여러 작업 라인을 안정적으로 병렬 진행할 수 있습니다. |
| **확장 문서 테마** | 모험단 관리 인터페이스에서 문서 테마를 전환하고, 다양한 테마에서의 용어와 텍스트 표현을 실시간으로 미리 볼 수 있습니다. |
| **사용자 지정 Git 커밋 서명** | 기본 `Co-Authored-By` 패턴을 설정하거나, AI Compose Commit의 단일 실행 시 임시로 재정의할 수 있습니다. |
| **브랜드 사용자 지정** | 헤더 로고 업로드, 애플리케이션 제목, 브라우저 제목 및 브랜드 링크 대상을 설정할 수 있습니다. |
| **사용자 지정 아바타 업로드** | 현재 Hero에 사용자 지정 아바타를 업로드하거나, 공유 사용자 지정 아바타 라이브러리에서 생성, 편집 및 일괄 생성 흐름에 재사용할 수 있습니다. |
| **DLC 아바타 리소스** | Turbo Engine 전용 아바타 팩이 Hero Avatar Gallery 및 일괄 생성 흐름에 통합되어 워크플로 선택에 직접 참여합니다. |

## 필요한 사용자

- 여러 제안이나 작업을 자주 동시에 진행하는 사용자.
- 기본 3 동시 실행 상한으로는 더 이상 충분하지 않은 헤비 사용자.
- 사용자 지정 Git 서명, 문서 테마, 아바타 및 브랜드 아이덴티티가 필요한 사용자.
