## Mit takar ez a szakasz

A GYIK szekció a következőknek való:

- indítási és futási figyelmeztetések
- platformokhoz, terjesztési csatornákhoz, csomagoláshoz vagy kötegelt futtatókörnyezetekhez kötött kompatibilitási megjegyzések
- hibaelhárítási témák, amelyek megérdemelnek egy stabil önálló magyarázó oldalt

Ha meg kell erősítenie, hogy melyik Node Desktopot használja, vagy miért említi meg a diagnosztika `portable-fixed/toolchain`, nyitott [Asztali csomópont környezet](/hu-HU/faq/desktop-node-environment).
