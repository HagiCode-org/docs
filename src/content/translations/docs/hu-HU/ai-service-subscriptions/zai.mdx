## Elhelyezés

A ZAI egyike azoknak a szolgáltatóknak, amelyeket jelenleg a Claude Code-hoz előre beállított dokumentumok ajánlanak. GLM Coding tervek rétegleképezéssel azon fejlesztők számára, akik Claude Code-ot napi vezetőként szeretnék használni.

## Jó illeszkedés

- Kódolásra összpontosító előfizetési tervet szeretne
- Leképezett Sonnet/Opus/Haiku alapértelmezésekre van szüksége
- Elvárja, hogy egy szolgáltató dolgozzon a Claude Code és hasonló kódoló eszközökön keresztül

## A legfontosabb dolgok, amelyeket észre kell venni

- Az aktuális előre beállított metaadatok a ZAI-t ajánlottnak jelölik.
- Feltár egy Anthropic-kompatibilis végpontot, amelyet a Claude Code közvetlenül használhat.
- Az előre beállított alapértelmezett leképezéseket szállítja a következőhöz `glm-4.7`, `glm-5`, és `glm-4.5-air`.

## Integrációs pillanatkép

| Tétel | Érték |
|------|-------|
| Előfizetéses bejegyzés | [Nyissa meg az előfizetési oldalt](https://www.bigmodel.cn/claude-code?ic=14BY54APZA) |
| Hivatalos dokumentumok / konzol | [Nyissa meg a hivatalos oldalt](https://open.bigmodel.cn/dev/api) |
| Antropikus kompatibilis alap URL | `https://open.bigmodel.cn/api/anthropic` |
| Ajánlott auth env var | `ANTHROPIC_AUTH_TOKEN` |

## Alapértelmezett modellleképezés

| Claude szint | Alapértelmezett modell |
|-------------|---------------|
| Szonett | `glm-4.7` |
| Opus | `glm-5` |
| Haiku | `glm-4.5-air` |

:::tipp
Az árak, az akciók és a csomag részletei idővel változhatnak. Ha a költségek a fő gondja, erősítse meg a legfrissebb információkat közvetlenül a szolgáltató oldalán.
:::

## Javasolt következő lépések

1. Erősítse meg, hogy melyik előfizetési szintet vagy API aktiválási útvonalat szeretné használni.
2. Nyissa meg a Claude Code bejegyzést az egyesített AI Agent CLI hubban, és folytassa az Anthropic hivatalos telepítési és referenciadokumentumaival.
3. A beállítást követően helyileg ellenőrizze a kapcsolatot.
