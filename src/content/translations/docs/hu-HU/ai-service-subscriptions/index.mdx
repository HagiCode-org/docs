Ez a fejezet összegyűjti azokat a mesterséges intelligenciaszolgáltatás-előfizetéses webhelyeket, amelyek jelenleg ajánlottként vannak megjelölve `repos/docs/public/presets/claude-code/providers/*.json`. Először itt válasszon szolgáltatót, majd folytassa a Claude Code konfigurációjával.

**2026. március 9-én** ez a fejezet a következőkre terjed ki: **ZAI, MiniMax, Alibaba Cloud DashScope és Volcengine Coding Plan**.

:::tipp[Javasolt olvasási sorrend]
Ha még nem választott szolgáltatót, kezdje ezzel az oldallal az összehasonlításhoz. Ha már tudja, melyik szolgáltatót szeretné, először ugorjon a részletes oldalra, majd folytassa a Claude Code bejegyzéssel az egyesített AI Agent CLI hubban.
:::

## Jelenlegi ajánlott szolgáltatók

| Szolgáltató | Erő | Jó illeszkedés | Részletes oldal |
|----------|----------|--------------|-------------|
| ZAI | Kódolásra összpontosító tervek szintleképezéssel | Fejlesztők, akik Claude Code-ot napi sofőrnek szeretnék | [Részletek megtekintése](./zai/) |
| MiniMax | Egy alapértelmezett modell a szinteken keresztül, egyszerű beépítés | Alacsony súrlódású beállításra optimalizáló csapatok | [Részletek megtekintése](./minimax/) |
| Alibaba Cloud DashScope | Felhőkonzol összehangolás és egységes kezelés | Az Alibaba Cloudban már működő csapatok | [Részletek megtekintése](./aliyun/) |
| Volcengin kódolási terv | Erős ökoszisztéma-kiterjesztés és többmodell platform jelek | Fejlesztők, akik szeretnék a Volcengine ökoszisztéma összehangolását | [Részletek megtekintése](./volcengine/) |

::: Megjegyzés
`Anthropic Official` használható marad, de nem szerepel a dedikált oldalak első kötegében, mert az aktuális előre beállított metaadatok nem jelölik meg ajánlottként.
:::
