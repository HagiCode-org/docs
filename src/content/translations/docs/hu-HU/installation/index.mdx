A HagiCode több telepítési utat is támogat. Kezdje a forgatókönyvnek megfelelő telepítési módszerrel, majd csak akkor adjon hozzá támogató szoftvert, amikor szüksége van rá.

A jelenlegi aktív Agent CLI támogatási tartomány a legrelevánsabb a beállításhoz: Claude Code, Codex, GitHub Copilot, OpenCode, Hermes, QoderCLI, Kiro, Kimi, Gemini, DeepAgents és Codebuddy. A Hermes, a Gemini és a DeepAgents jelenleg manuális telepítési vagy hitelesítési folyamatokat használ; a többi CLI egy kattintással elérhető telepítési belépési pontokat vagy irányított telepítési folyamatokat tesz lehetővé.

## Javasolt következő lépések

- Ha a HagiCode megvásárlását és telepítését tervezi a Steamen keresztül, kezdje ezzel [Steam telepítés](/hu-HU/installation/steam).
- Ha még nem ismeri a HagiCode-ot, kezdje ezzel [Asztali telepítés](/hu-HU/installation/desktop).
- Ha kiszolgáló- vagy csoportkörnyezetre telepíti, használja [Docker Compose telepítése](/hu-HU/installation/docker-compose).
- Ha meg kell erősítenie a CLI támogatási állapotát, beállítási módját vagy korábbi kompatibilitási megjegyzéseit, nyissa meg [AI Agent CLI telepítése](/hu-HU/related-software-installation/ai-agent-cli/).
