## Hva denne delen dekker

FAQ-delen er stedet for:

- advarsler om oppstart og kjøretid
- kompatibilitetsnotater knyttet til plattformer, distribusjonskanaler, emballasje eller medfølgende kjøretider
- feilsøking av emner som fortjener en stabil frittstående forklaringsside

Hvis du trenger å bekrefte hvilken Node Desktop bruker eller hvorfor diagnostikk nevner `portable-fixed/toolchain`, åpen [Desktop Node Environment](/nb-NO/faq/desktop-node-environment).

Hvis du vil vite hvorfor HagiCode ikke er på Steam, åpne [Hvorfor HagiCode ikke er tilgjengelig på Steam](/nb-NO/faq/steam-distribution-status).
