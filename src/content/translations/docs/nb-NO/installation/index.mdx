HagiCode støtter flere installasjonsveier. Start med distribusjonsmetoden som passer ditt scenario, og legg deretter til støtteprogramvare bare når du trenger det.

Det gjeldende aktive Agent CLI-støtteområdet som er mest relevant for oppsett er: Claude Code, Codex, GitHub Copilot, OpenCode, Hermes, QoderCLI, Kiro, Kimi, Gemini, DeepAgents og Codebuddy. Hermes, Gemini og DeepAgents bruker for tiden manuell installasjon eller autentiseringsflyter; de andre CLI-ene viser installasjonsinngangspunkter med ett klikk eller veiledede installasjonsflyter.

## Anbefalte neste trinn

- Hvis du planlegger å kjøpe og installere HagiCode gjennom Steam, start med [Steam installasjon](/nb-NO/installation/steam).
- Hvis du er ny på HagiCode, start med [Desktop installasjon](/nb-NO/installation/desktop).
- Hvis du distribuerer for en server eller et teammiljø, bruk [Installasjon av Docker Compose](/nb-NO/installation/docker-compose).
- Hvis du trenger å bekrefte støttestatus, oppsettmodus eller historiske kompatibilitetsnotater for en CLI, åpne [AI Agent CLI installasjon](/nb-NO/related-software-installation/ai-agent-cli/).
