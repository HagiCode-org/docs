Denne personvernerklæringen forklarer kategoriene av data HagiCode kan behandle når den oppgir `hagicode.com` og alle dets underdomenenettsteder, veiledning for programvarenedlasting, støttekanaler og relaterte produktopplevelser, sammen med formålene, oppbevaringsmetoden og rettighetene som er tilgjengelige for deg.

## Informasjonskategorier vi samler inn

Avhengig av hvordan du samhandler med HagiCode, kan vi behandle:

- **Tilgangs- og enhetsinformasjon**, for eksempel nettlesertype, operativsystem, besøkte sider, henvisningslenker, tidsstempler, grov region og enhetsidentifikatorer;
- **Data om bruksatferd**, som sidevisninger, tid på siden, klikk, rulleaktivitet, øktregistreringer og samlet analyse som brukes for å forstå `hagicode.com` og alle dets underdomenesider sammen med relaterte produktsider;
- **Støtte- og kommunikasjonsinformasjon**, slik som navn, e-postadresse, problembeskrivelse, loggutdrag eller vedlegg som du frivillig sender inn via e-post, problemer, fellesskapskanaler eller skjemaer; og
- **Nedlastings- og konfigurasjonskontekst**, for eksempel installasjonsveiledninger, nedlastingsoppføringer, forhåndsinnstilte filer eller utgivelsesnotater du får tilgang til for å hjelpe oss med å forstå dokumentasjon og utgivelseskvalitet.

## Formål med bruk

Vi behandler denne informasjonen til:

- gi og vedlikeholde `hagicode.com` og alle dets underdomenenettsteder, nedlastingsinnganger og støttekanaler;
- forstå hvilke sider, oppsettstrinn eller produktforklaringer som må forbedres;
- feilsøke nettstedsfeil, nedlastingsproblemer, kompatibilitetsfeil og sikkerhetsrisikoer;
- svare på dine spørsmål, støtteforespørsler, tilbakemeldinger eller klager; og
- tilfredsstille gjeldende juridiske, regulatoriske, håndhevende eller plattformsikkerhetskrav når det er nødvendig.

## Berøringspunkter for nettsted og produktdata

Den nåværende HagiCode `hagicode.com` eiendom og alle dens underdomenenettsteder kan berøre relevante data gjennom:

- nettstedtilgangslogger og grunnleggende forespørselsmetadata;
- Microsoft Clarity eller sammenlignbar sideanalyse som brukes til å forstå leseflyt og interaksjonshotspots;
- 51LA eller sammenlignbar trafikkanalyse som brukes til å forstå trafikktrender og henvisninger; og
- støtteinformasjon du frivillig sender til `support@hagicode.com` eller GitHub-problemer.

Hvis HagiCode-skrivebordsklienten, backend-tjenestene eller tredjeparts AI-integrasjoner behandler tilleggsdata i separate produktkontekster, er disse detaljene dokumentert i det relevante produktmaterialet, innstillingene eller tredjepartsvilkårene.

## Lagring og oppbevaring

Vi oppbevarer data i rimelige perioder basert på operasjonelle behov, juridiske forpliktelser og sikkerhetskrav:

- analysedata holdes vanligvis i aggregert form eller under standard oppbevaringsreglene til analyseleverandøren;
- støtte-e-poster, problemrapporter og feilsøkingsposter beholdes mens de er nødvendige for å løse problemer, revidere kommunikasjon eller oppfylle juridiske forpliktelser; og
- noen logger og støtteposter kan oppbevares lenger når det er nødvendig for sikkerhet, forebygging av misbruk, oppbevaring av samsvar eller tvisteløsning.
- noen data kan være vert for, behandlet, overført, sikkerhetskopiert eller på annen måte håndtert gjennom Microsoft Azure, Alibaba Cloud eller relaterte infrastrukturfunksjoner, noe som betyr at de relevante dataene kan flytte mellom forskjellige land eller regioner.

Når data ikke lenger er nødvendig for disse formålene, sletter vi dem, anonymiserer dem eller lar dem utløpe under plattformens standard oppbevaringsregler.

## Tredjepartstjenester

HagiCode kan stole på tredjeparts infrastruktur, analyseleverandører, kodevertsplattformer, e-posttjenester eller AI-tjenesteleverandører. Disse partene behandler data kun etter behov for å levere tjenestene deres og forblir underlagt deres egne vilkår og personvernerklæringer.

I nåværende eller fremtidige produkt- og nettstedsoperasjoner kan vi få visse data behandlet eller på annen måte håndtert gjennom skyinfrastruktur som Microsoft Azure og Alibaba Cloud. Det betyr at data kan overføres på tvers av forskjellige land eller regioner for lagring, sikkerhetskopiering, logganalyse, feilsøking, innholdslevering eller tjenesteorkestrering.

Der det er rimelig praktisk, bruker vi relevante data i anonymisert, aggregert eller avidentifisert form for operasjonell analyse, tjenesteforbedring, stabilitetsarbeid og sikkerhetsbeskyttelse. Du forstår imidlertid og godtar at vi ikke fullt ut kan forhindre, kontrollere eller garantere at skyleverandører, infrastrukturveier eller overføringer på tvers av regioner alltid vil forbli innenfor et enkelt land eller en region.

Når du bruker tredjepartsintegrasjoner, bør du også gjennomgå de relevante leverandørretningslinjene for å forstå deres egen behandling og grenseoverskridende overføringspraksis.

Hvis du ikke kan akseptere denne skybaserte behandlingen, grenseoverskridende overføringen eller relatert anonymisert bruk av data, må du ikke laste ned, installere, få tilgang til eller fortsette å bruke HagiCode-produkter, -tjenester eller `hagicode.com` og noen av underdomenenettstedene.

## Brukerrettigheter

I den grad det er gitt av gjeldende lov, kan du be om å:

- vite om vi har personopplysninger knyttet til deg;
- korrigere unøyaktig eller ufullstendig informasjon;
- slette informasjon som ikke lenger er nødvendig eller som du på annen måte har krav på å få slettet;
- begrense eller protestere mot visse behandlingsaktiviteter; og
- få ytterligere forklaring angående en støttekommunikasjon eller -klage.

Der en forespørsel påvirker kontosikkerhet, juridiske forpliktelser, systemsikkerhet, misbruksforebygging eller grenser for tredjepartsplattformer, kan det hende vi må bekrefte identiteten først eller beholde begrensede data som tillatt i henhold til loven.

## Policyoppdateringer

Hvis HagiCode-nettstedets funksjoner, analyseverktøy, kontaktkanaler eller samsvarskrav endres, kan vi oppdatere denne personvernerklæringen. Oppdaterte versjoner vil bli publisert på denne siden med en revidert ikrafttredelsesdato.
