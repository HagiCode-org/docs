## Posisjonering

ZAI er en av leverandørene som for øyeblikket anbefales av forhåndsinnstillingen for dokumenter for Claude Code. GLM Coding-planer med tiermapping for utviklere som ønsker Claude Code som daglig sjåfør.

## God passform til

- Du vil ha en kodefokusert abonnementsplan
- Du trenger kartlagte Sonnet/Opus/Haiku-standarder
- Du forventer at én leverandør fungerer på tvers av Claude Code og lignende kodeverktøy

## Viktige ting å legge merke til

- Gjeldende forhåndsinnstilte metadata markerer ZAI som anbefalt.
- Den avslører et antropisk-kompatibelt endepunkt som kan brukes direkte av Claude Code.
- Forhåndsinnstillingen sender standardtilordninger for `glm-4.7`, `glm-5`, og `glm-4.5-air`.

## Øyeblikksbilde av integrering

| Vare | Verdi |
|------|-------|
| Abonnementsoppføring | [Åpne abonnementssiden](https://www.bigmodel.cn/claude-code?ic=14BY54APZA) |
| Offisielle dokumenter / konsoll | [Åpne offisiell side](https://open.bigmodel.cn/dev/api) |
| Antropisk-kompatibel base URL | `https://open.bigmodel.cn/api/anthropic` |
| Anbefalt auth env var | `ANTHROPIC_AUTH_TOKEN` |

## Standard modellkartlegging

| Claude tier | Standardmodell |
|-------------|---------------|
| Sonett | `glm-4.7` |
| Opus | `glm-5` |
| Haiku | `glm-4.5-air` |

:::tips
Priser, kampanjer og pakkedetaljer kan endres over tid. Hvis kostnaden er hovedproblemet ditt, bekrefter du den nyeste informasjonen direkte på leverandørsiden.
:::

## Foreslått neste trinn

1. Bekreft hvilket abonnementsnivå eller API-aktiveringssti du vil bruke.
2. Åpne Claude Code-oppføringen i den forente AI Agent CLI-huben og fortsett med Anthropics offisielle installasjons- og referansedokumenter.
3. Valider tilkoblingen lokalt etter oppsett.
