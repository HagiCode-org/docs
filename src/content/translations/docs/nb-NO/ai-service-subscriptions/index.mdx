Dette kapittelet samler AI-tjenesteabonnementnettstedene som for øyeblikket er merket som anbefalt i `repos/docs/public/presets/claude-code/providers/*.json`. Velg en leverandør her først, fortsett deretter til Claude Code-konfigurasjon.

Fra **9. mars 2026** dekker dette kapittelet: **ZAI, MiniMax, Alibaba Cloud DashScope og Volcengine Coding Plan**.

:::tips[Foreslått leserekkefølge]
Hvis du ikke har valgt en leverandør ennå, start med denne siden for sammenligning. Hvis du allerede vet hvilken leverandør du vil ha, hopp til detaljsiden først og fortsett deretter til Claude Code-oppføringen i den forente AI Agent CLI-huben.
:::

## Aktuelle anbefalte leverandører

| Leverandør | Styrke | God passform til | Detaljside |
|----------|----------|--------------|-------------|
| ZAI | Kodingfokuserte planer med lagkartlegging | Utviklere som ønsker Claude Code som daglig sjåfør | [Se detaljer](./zai/) |
| MiniMax | Én standardmodell på tvers av nivåer, enkel onboarding | Lag som optimaliserer for lavfriksjonsoppsett | [Se detaljer](./minimax/) |
| Alibaba Cloud DashScope | Skykonsolljustering og enhetlig administrasjon | Team som allerede jobber i Alibaba Cloud | [Se detaljer](./aliyun/) |
| Volcengine Kodeplan | Sterke økosystemutvidelser og multi-modell plattformsignaler | Utviklere som ønsker Volcengine-økosystemjustering | [Se detaljer](./volcengine/) |

:::merk
`Anthropic Official` forblir brukbar, men den er ikke inkludert i den første gruppen med dedikerte sider fordi gjeldende forhåndsinnstilte metadata ikke markerer den som anbefalt.
:::
