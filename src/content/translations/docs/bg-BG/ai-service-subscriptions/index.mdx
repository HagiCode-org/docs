Тази глава събира уебсайтовете за абонамент за AI услуги, които в момента са маркирани като препоръчани в `repos/docs/public/presets/claude-code/providers/*.json`. Първо изберете доставчик тук, след което продължете към конфигурацията на Claude Code.

Към **9 март 2026 г.** тази глава обхваща: **ZAI, MiniMax, Alibaba Cloud DashScope и Volcengine Coding Plan**.

:::съвет[Предложен ред на четене]
Ако все още не сте избрали доставчик, започнете с тази страница за сравнение. Ако вече знаете кой доставчик искате, първо преминете към неговата страница с подробности и след това продължете към записа на Claude Code в унифицирания AI Agent CLI център.
:::

## Текущи препоръчани доставчици

| Доставчик | Сила | Подходящ за | Страница с подробности |
|----------|----------|--------------|-------------|
| ZAI | Планове, фокусирани върху кодирането с картографиране на нива | Разработчици, които искат Claude Code като ежедневен драйвер | [Вижте подробности](./zai/) |
| МиниМакс | Един модел по подразбиране за всички нива, лесно включване | Екипи, оптимизиращи за настройка с ниско триене | [Вижте подробности](./minimax/) |
| Alibaba Cloud DashScope | Подравняване на облачна конзола и унифицирано управление | Екипи вече работят в Alibaba Cloud | [Вижте подробности](./aliyun/) |
| План за кодиране на Volcengine | Силно разширение на екосистемата и многомоделни платформени сигнали | Разработчици, които искат привеждане в съответствие с екосистемата на Volcengine | [Вижте подробности](./volcengine/) |

:::бележка
`Anthropic Official` остава използваем, но не е включен в първата партида от специални страници, тъй като текущите предварително зададени метаданни не го маркират като препоръчан.
:::

## Съвети за бърз избор

- **Нуждаете се от планове, фокусирани върху кодиране и по-ясно картографиране на нива**: започнете с [ZAI](./zai/)
- **Първо се нуждаете от най-лесния път за настройка**: започнете с [МиниМакс](./minimax/)
- **Вече работи в Alibaba Cloud**: започнете с [Alibaba Cloud DashScope](./aliyun/)
- **Искате привеждане в съответствие с екосистемата на Volcengine или обхват на мултимоделна платформа**: започнете с [План за кодиране на Volcengine](./volcengine/)
