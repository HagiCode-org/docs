## Позициониране

ZAI е един от доставчиците, които в момента се препоръчват от предварително зададените документи за Claude Code. Планове за кодиране на GLM с картографиране на нива за разработчици, които искат Claude Code като ежедневен драйвер.

## Подходящ за

- Искате абонаментен план, фокусиран върху кодирането
- Имате нужда от картографирани настройки по подразбиране за Sonnet/Opus/Haiku
- Очаквате един доставчик да работи с Claude Code и подобни инструменти за кодиране

## Ключови неща, които трябва да забележите

- Текущите предварително зададени метаданни маркират ZAI като препоръчан.
- Той разкрива крайна точка, съвместима с Anthropic, която може да се използва директно от Claude Code.
- Предварителната настройка доставя съпоставяния по подразбиране за `glm-4.7`, `glm-5`, и `glm-4.5-air`.

## Моментна снимка на интеграцията

| Артикул | Стойност |
|------|-------|
| Вписване за абонамент | [Отворете страницата за абонамент](https://www.bigmodel.cn/claude-code?ic=14BY54APZA) |
| Официални документи / конзола | [Отворете официалната страница](https://open.bigmodel.cn/dev/api) |
| Основен URL адрес, съвместим с антропно | `https://open.bigmodel.cn/api/anthropic` |
| Препоръчителна авт | `ANTHROPIC_AUTH_TOKEN` |

## Картографиране на модела по подразбиране

| Клод ниво | Модел по подразбиране |
|-------------|---------------|
| Сонет | `glm-4.7` |
| Опус | `glm-5` |
| Хайку | `glm-4.5-air` |

::: съвет
Цените, промоциите и подробностите за пакета могат да се променят с времето. Ако цената е основната ви грижа, потвърдете най-новата информация директно на страницата на доставчика.
:::

## Предложени следващи стъпки

1. Потвърдете кое ниво на абонамент или път за активиране на API искате да използвате.
2. Отворете записа на Claude Code в унифицирания CLI център на AI Agent и продължете с официалните документи за инсталиране и справка на Anthropic.
3. Проверете връзката локално след настройката.
