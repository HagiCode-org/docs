Tato příručka popisuje instalaci a používání Hagicode Desktop. Je to oficiální klient pro stolní počítače a zachovává výchozí cestu místního nastavení a denní ovládací prvky na jednom místě.

:::tip[Doporučená metoda]
Desktop je výchozí místní cestou pro jednotlivé uživatele a vývojáře, protože:
- instalační program zachová výchozí tok v jedné naváděné cestě
- závislosti a služby jsou spravovány v aplikaci
- uživatelské rozhraní usnadňuje kontrolu stavu a operací
- vestavěná správa verzí usnadňuje přepínání
- data zůstávají ve výchozím nastavení místní
:::

## Co je Hagicode Desktop?

Hagicode Desktop je oficiální desktopová aplikace Hagicode určená pro systémy Windows, macOS a Linux. Sdružuje základní funkce Hagicode a instalační program udržuje výchozí cestu nastavení v jednom řízeném toku.

### Hlavní výhody

- **Instalace jedním kliknutím**: Dokončete všechny kroky konfigurace automaticky pomocí průvodce instalací
- **Automatická příprava běhového prostředí**: Výchozí instalační program již obsahuje běhové prostředí, které potřebuje, takže nemusíte instalovat samostatný `.NET` nejdříve runtime
- **Správa služeb**: Vestavěná správa serveru Hagicode umožňuje snadné spouštění a zastavování služeb
- **Přepínání verzí**: Podpora pro správu více verzí, volně přepínat mezi různými verzemi
- **Místní provoz**: Všechna data jsou uložena lokálně, což chrání vaše soukromí
- **Automatické aktualizace**: Automaticky detekuje a stahuje nejnovější verzi

### Použitelné scénáře

Desktop se hodí, když:
- Jednotliví vývojáři, kteří chtějí používat Hagicode lokálně
- Uživatelé, kteří potřebují používat Hagicode offline
- Netechnickí uživatelé, kteří chtějí zjednodušit proces instalace
- Vývojáři, kteří potřebují často přepínat mezi různými verzemi

### Další metody stahování

Instalační program Hagicode Desktop můžete také získat z následujících kanálů:

1. **Oficiální stránka vydání**: Navštivte [Vydání Hagicode GitHub](https://github.com/Hagicode-org/releases) ke stažení nejnovější verze
2. **Oficiální webové stránky**: Navštivte [Web Hagicode](https://hagicode.com/) získat odkazy ke stažení

### Vyberte správnou verzi

Vyberte odpovídající instalační program pro váš operační systém:
- **Windows**: Stáhněte si soubor `.exe` instalátor
- **macOS**: Stáhněte si soubor `.dmg` instalátor
- **Linux**: Stáhněte si `.AppImage` nebo `.tar.gz` instalační balíček

:::note[Systémové požadavky]
- **Windows**: Windows 10 nebo vyšší verze (64bitová)
- **macOS**: macOS 10.15 nebo vyšší
- **Linux**: Mainstreamové distribuce Linuxu (Ubuntu 20.04+, Fedora 33+ atd.)
:::

:::tip[Výchozí předpoklady]
Pro nejnovější postup místní instalace HagiCode obvykle nemusíte připravovat samostatný `.NET` runtime. Desktop již sbalí běhové prostředí požadované pro výchozí tok a ve výchozím nastavení inicializuje podporovanou místní cestu SQLite.
:::

## Poznámky k instalaci a spuštění

:::upozornění[upozornění správce brány macOS]
Pokud jste již aplikaci nainstalovali do výchozí cesty `/Applications/Hagicode.app` ale macOS při prvním spuštění zobrazuje „aplikace je poškozená“, „nelze otevřít“ nebo podobné varování Gatekeeper, obvyklou příčinou je `com.apple.quarantine` atribut, který macOS přidává do aplikací stažených z internetu, nikoli selhání v samotném instalačním programu Hagicode Desktop.

Spusťte následující příkaz v Terminálu. Vyžaduje oprávnění správce a rekurzivně odstraňuje atribut karantény z aplikace ve výchozí instalační cestě:

```bash
sudo xattr -dr com.apple.quarantine /Applications/Hagicode.app
```

Po dokončení příkazu znovu spusťte Hagicode Desktop.
:::

## Spusťte Desktop

Po stažení otevřete instalační program nebo balíček aplikace a postupujte podle pokynů na obrazovce. Desktop připravuje běhové prostředí v aktuálně podporované místní cestě, spravuje Hagicode Server a zobrazuje průběh a stav, když je vyžadováno stahování ze sítě.

:::poznámka[Požadavky na síť]
Ujistěte se, že je dostupné připojení k síti. Desktop může potřebovat načíst informace o verzi nebo balíčky součástí z internetu; pokud se stahování nezdaří, zkontrolujte nastavení sítě, proxy nebo brány firewall a zkuste to znovu.
:::

## Správa verzí

Hagicode Desktop podporuje správu více verzí, což vám umožňuje volně přepínat mezi různými verzemi.

![Správa verzí](../../img/screenshots/installation/install-desktop/12-version-management/original.png)

### Přepnout verzi

1. Otevřete Hagicode Desktop
2. Přejděte na stránku "Správa verzí".
3. Vyberte verzi, kterou chcete přepnout
4. Klikněte na tlačítko "Přepnout verzi".
5. Počkejte na dokončení přepínání

:::note[Poznámky k přepínání verzí]
- Přepnutí verzí automaticky restartuje Hagicode Server
- Před přepnutím se ujistěte, že jste uložili veškerou práci
- Konfigurace projektu se mohou mezi verzemi lišit
:::

## Spuštění a zastavení služby

Hagicode Desktop poskytuje jednoduchý způsob správy služby Hagicode Server.

### Spusťte službu

Na domovské stránce aplikace Desktop klikněte na tlačítko "Spustit službu" pro spuštění serveru Hagicode.

![Spusťte službu](../../img/screenshots/installation/install-desktop/13-start-service-homepage/original.png)

### Zastavit službu

Chcete-li Hagicode Server zastavit, klikněte na tlačítko "Zastavit službu" na domovské stránce.

:::tip[Stav služby]
Aplikace Desktop zobrazuje stav běhu serveru Hagicode v reálném čase, včetně:
- Stav služby (běžící/zastaveno)
- Číslo servisního portu
- Informace o aktuální verzi
:::

## Odstraňování problémů

### Selhání instalace

Pokud se během instalace setkáte s problémy:

1. **Zkontrolujte připojení k síti**: Ujistěte se, že máte přístup k internetu
2. **Zkontrolujte místo na disku**: Ujistěte se, že máte na disku dostatek místa
3. **Spustit jako správce**: V systému Windows klikněte pravým tlačítkem na instalační program a vyberte „Spustit jako správce“
4. **Zobrazit soubory protokolu**: Instalační program vygeneruje podrobné soubory protokolu, které mohou pomoci diagnostikovat problémy

### Selhání instalace závislosti

Pokud se instalace závislosti nezdaří:

1. **Zkontrolujte kompatibilitu systému**: Ujistěte se, že váš operační systém splňuje minimální požadavky
2. **Zkontrolujte nastavení sítě a brány firewall**: Ujistěte se, že instalační program může dosáhnout požadovaných stažení a není blokován vaším firewallem
3. **Potvrďte oprávnění instalačního programu**: V systému Windows zkuste instalační program spustit jako správce, aby měl dostatečná oprávnění
4. **Restartujte aplikaci a opakujte**: Zavřete a znovu otevřete Hagicode Desktop a opakujte krok instalace
5. **Zobrazit protokoly**: Prohlédněte si podrobné protokoly chyb v aplikaci Desktop

### Selhání spuštění služby

Pokud se Hagicode Server nespustí:

1. **Zkontrolujte obsazenost portu**: Ujistěte se, že port 45000 není obsazen jinými programy
2. **Zkontrolujte nastavení brány firewall**: Ujistěte se, že brána firewall umožňuje přístup k síti serveru Hagicode
3. **Restartovat aplikaci**: Zkuste restartovat Hagicode Desktop
4. **Zobrazit protokoly**: Zobrazení podrobných protokolů chyb v aplikaci Desktop
