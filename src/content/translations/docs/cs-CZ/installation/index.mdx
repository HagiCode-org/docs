HagiCode podporuje více instalačních cest. Začněte s metodou nasazení, která vyhovuje vašemu scénáři, a poté přidejte podpůrný software, pouze když jej potřebujete.

Aktuální rozsah podpory aktivního agenta CLI, který je pro nastavení nejdůležitější, je: Claude Code, Codex, GitHub Copilot, OpenCode, Hermes, QoderCLI, Kiro, Kimi, Gemini, DeepAgents a Codebuddy. Hermes, Gemini a DeepAgents v současné době používají ruční instalaci nebo toky ověřování; ostatní CLI odhalují vstupní body instalace jedním kliknutím nebo řízené instalační toky.

## Doporučené další kroky

- Pokud plánujete koupit a nainstalovat HagiCode přes Steam, začněte s [Instalace Steam](/cs-CZ/installation/steam).
- Pokud jste v HagiCode noví, začněte s [Instalace na plochu](/cs-CZ/installation/desktop).
- Pokud nasazujete pro server nebo týmové prostředí, použijte [Instalace Docker Compose](/cs-CZ/installation/docker-compose).
- Pokud potřebujete potvrdit stav podpory, režim nastavení nebo poznámky o historické kompatibilitě pro CLI, otevřete [Instalace AI Agent CLI](/cs-CZ/related-software-installation/ai-agent-cli/).
