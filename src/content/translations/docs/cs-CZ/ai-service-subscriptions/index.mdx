Tato kapitola shromažďuje webové stránky předplatného služby AI, které jsou aktuálně označeny jako doporučené `repos/docs/public/presets/claude-code/providers/*.json`. Nejprve zde vyberte poskytovatele a poté pokračujte na konfiguraci Claude Code.

Od **9. března 2026** tato kapitola zahrnuje: **ZAI, MiniMax, Alibaba Cloud DashScope a Volcengine Coding Plan**.

:::tip[Doporučené pořadí čtení]
Pokud jste si ještě nevybrali poskytovatele, začněte na této stránce pro srovnání. Pokud již víte, kterého poskytovatele chcete, přejděte nejprve na jeho podrobnou stránku a poté pokračujte na položku Claude Code v unifikovaném centru CLI agenta AI.
:::

## Aktuální doporučení poskytovatelé

| Poskytovatel | Síla | Vhodné pro | Detailní stránka |
|----------|----------|--------------|-------------|
| ZAI | Plány zaměřené na kódování s mapováním vrstev | Vývojáři, kteří chtějí Claude Code jako každodenního řidiče | [Zobrazit podrobnosti](./zai/) |
| MiniMax | Jeden výchozí model napříč vrstvami, jednoduché začlenění | Týmy optimalizující nastavení s nízkým třením | [Zobrazit podrobnosti](./minimax/) |
| Alibaba Cloud DashScope | Vyrovnání cloudové konzole a jednotná správa | Týmy již pracují v Alibaba Cloud | [Zobrazit podrobnosti](./aliyun/) |
| Kódovací plán Volcengine | Silné rozšíření ekosystému a signály platformy pro více modelů | Vývojáři, kteří chtějí zarovnání ekosystému Volcengine | [Zobrazit podrobnosti](./volcengine/) |

:::pozn
`Anthropic Official` zůstává použitelná, ale není zahrnuta v první dávce vyhrazených stránek, protože aktuální přednastavená metadata ji neoznačují jako doporučenou.
:::
