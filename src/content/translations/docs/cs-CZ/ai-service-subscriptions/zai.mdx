## Polohování

ZAI je jedním z poskytovatelů aktuálně doporučených přednastavením dokumentů pro Claude Code. Plány GLM Coding s mapováním úrovní pro vývojáře, kteří chtějí Claude Code jako každodenního řidiče.

## Vhodné pro

- Chcete plán předplatného zaměřený na kódování
- Potřebujete namapované výchozí hodnoty Sonnet/Opus/Haiku
- Očekáváte, že jeden poskytovatel bude pracovat s Claude Code a podobnými kódovacími nástroji

## Klíčové věci, kterých si musíte všimnout

- Aktuální přednastavená metadata označují ZAI jako doporučené.
- Odhaluje koncový bod kompatibilní s Anthropic, který může Claude Code používat přímo.
- Předvolba dodává výchozí mapování pro `glm-4.7`, `glm-5`a `glm-4.5-air`.

## Snímek integrace

| Položka | Hodnota |
|------|-------|
| Vstup do předplatného | [Otevřete stránku předplatného](https://www.bigmodel.cn/claude-code?ic=14BY54APZA) |
| Oficiální dokumenty / konzole | [Otevřete oficiální stránku](https://open.bigmodel.cn/dev/api) |
| Základní URL kompatibilní s antropickým režimem | `https://open.bigmodel.cn/api/anthropic` |
| Doporučené auth env var | `ANTHROPIC_AUTH_TOKEN` |

## Výchozí mapování modelu

| Claude tier | Výchozí model |
|-------------|---------------|
| Sonet | `glm-4.7` |
| Opus | `glm-5` |
| haiku | `glm-4.5-air` |

:::tip
Ceny, akce a podrobnosti o balíčcích se mohou v průběhu času měnit. Pokud je vaším hlavním zájmem cena, ověřte si nejnovější informace přímo na stránce poskytovatele.
:::

## Doporučené další kroky

1. Potvrďte, kterou vrstvu předplatného nebo cestu aktivace rozhraní API chcete použít.
2. Otevřete položku Claude Code v unifikovaném centru CLI AI Agent a pokračujte oficiálními instalačními a referenčními dokumenty Anthropic.
3. Po nastavení ověřte připojení lokálně.
