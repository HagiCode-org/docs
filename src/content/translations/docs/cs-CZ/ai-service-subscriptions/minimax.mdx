## Polohování

MiniMax je jedním z poskytovatelů aktuálně doporučených přednastavením dokumentů pro Claude Code. Jediný výchozí model napříč vrstvami vhodný pro týmy, které oceňují jednoduchou konfiguraci a předvídatelné začlenění.

## Vhodné pro

- Chcete stejný mapovaný model na všech úrovních Claude
- Dáváte přednost nastavení a údržbě s nízkým třením
- Před vyladěním strategie modelu chcete rychle spustit Claude Code

## Klíčové věci, kterých si musíte všimnout

- Aktuální přednastavená metadata označují MiniMax jako doporučené.
- Používá `MiniMax-M2.5` jako výchozí mapovaný model pro všechny úrovně.
- To se hodí, když na provozní jednoduchosti záleží více než na ladění na úrovni jednotlivých vrstev.

| Položka | Hodnota |

## Výchozí mapování modelu

| Claude tier | Výchozí model |
|-------------|---------------|
| Sonet | `MiniMax-M2.5` |
| Opus | `MiniMax-M2.5` |
| haiku | `MiniMax-M2.5` |

:::tip
Ceny, akce a podrobnosti o balíčcích se mohou v průběhu času měnit. Pokud je vaším hlavním zájmem cena, ověřte si nejnovější informace přímo na stránce poskytovatele.
:::

## Doporučené další kroky

1. Potvrďte, kterou vrstvu předplatného nebo cestu aktivace rozhraní API chcete použít.
2. Otevřete položku Claude Code v unifikovaném centru CLI AI Agent a pokračujte oficiálními instalačními a referenčními dokumenty Anthropic.
3. Po nastavení ověřte připojení lokálně.
