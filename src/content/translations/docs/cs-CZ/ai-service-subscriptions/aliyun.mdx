## Polohování

Alibaba Cloud DashScope je jedním z poskytovatelů aktuálně doporučených přednastavenými dokumenty pro Claude Code. Praktická volba pro týmy, které již fungují v ekosystému Alibaba Cloud a chtějí jednotnou správu konzole.

## Vhodné pro

- Účty nebo zdroje Alibaba Cloud již používáte
- Chcete, aby předplatné a správa klíčů zůstaly v jedné cloudové konzoli
- Dáváte přednost pevně namapovanému modelu pro rychlé nalodění

## Klíčové věci, kterých si musíte všimnout

- Aktuální přednastavená metadata označují Alibaba Cloud DashScope jako doporučené.
- Mapuje všechny úrovně Claude `glm-4.7` pro přímou cestu nastavení.
- Je to zvláště výhodné, pokud váš tým již pracuje v rámci pracovních postupů Alibaba Cloud.
