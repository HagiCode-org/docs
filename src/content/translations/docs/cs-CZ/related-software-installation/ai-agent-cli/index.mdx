:::tip[Jak používat tuto stránku]
Nejprve zkontrolujte aktuální stav podpory a režim nastavení v tabulce a poté pokračujte v dokumentaci k instalaci, ověřování, ACP, oprávnění a odstraňování problémů, kterou udržuje dodavatel. Rozsah podpory a režim nastavení jsou samostatné záležitosti.
:::

## Státní glosář

- **Aktuálně podporováno** znamená, že CLI patří do aktuální aktivní řady podpory HagiCode.
- **Instalace jedním kliknutím** znamená, že HagiCode poskytuje vstupní bod instalace s průvodcem; některá rozhraní CLI po instalaci stále vyžadují přihlášení dodavatele nebo autentizační krok.
- **Manuální instalace / ověřování** znamená, že CLI je nyní podporováno, ale musíte dokončit místní instalaci nebo autentizační tok, než jej HagiCode může detekovat a používat.
- **Pouze starší kompatibilita** znamená vyřazenou hodnotu, jako je např `IFlowCli` stále existuje pro kompatibilitu historických dat, ale není součástí aktuálního rozsahu aktivní podpory.
