## Positionering

ZAI er en af de udbydere, der i øjeblikket anbefales af docs-forudindstillingen til Claude Code. GLM Coding planer med tier mapping for udviklere, der ønsker Claude Code som daglig driver.

## God pasform til

- Du ønsker en kodningsfokuseret abonnementsplan
- Du skal have kortlagte Sonnet/Opus/Haiku-standarder
- Du forventer, at én udbyder arbejder på tværs af Claude Code og lignende kodningsværktøjer

## Nøgle ting at lægge mærke til

- De nuværende forudindstillede metadata markerer ZAI som anbefalet.
- Den afslører et antropisk-kompatibelt endepunkt, der kan bruges direkte af Claude Code.
- Forudindstillingen sender standardtilknytninger til `glm-4.7`, `glm-5`, og `glm-4.5-air`.

## Snapshot af integration

| Vare | Værdi |
|------|-------|
| Tilmelding til abonnement | [Åbn abonnementssiden](https://www.bigmodel.cn/claude-code?ic=14BY54APZA) |
| Officielle dokumenter / konsol | [Åbn officiel side](https://open.bigmodel.cn/dev/api) |
| Antropisk-kompatibel base-URL | `https://open.bigmodel.cn/api/anthropic` |
| Anbefalet auth env var | `ANTHROPIC_AUTH_TOKEN` |

## Standardmodelkortlægning

| Claude tier | Standardmodel |
|-------------|---------------|
| Sonnet | `glm-4.7` |
| Opus | `glm-5` |
| Haiku | `glm-4.5-air` |

::: tip
Priser, kampagner og pakkedetaljer kan ændre sig over tid. Hvis prisen er din største bekymring, skal du bekræfte de seneste oplysninger direkte på udbydersiden.
:::

## Foreslået næste skridt

1. Bekræft, hvilket abonnementsniveau eller API-aktiveringssti du vil bruge.
2. Åbn Claude Code-indgangen i den forenede AI Agent CLI-hub, og fortsæt med Anthropics officielle installations- og referencedokumenter.
3. Valider forbindelsen lokalt efter opsætning.
