Dette kapitel samler de AI-tjenesteabonnementswebsteder, der i øjeblikket er markeret som anbefalet i `repos/docs/public/presets/claude-code/providers/*.json`. Vælg en udbyder her først, og fortsæt derefter til Claude Code-konfiguration.

Fra **9. marts 2026** dækker dette kapitel: **ZAI, MiniMax, Alibaba Cloud DashScope og Volcengine Coding Plan**.

:::tip[Foreslået læserækkefølge]
Hvis du ikke har valgt en udbyder endnu, start med denne side til sammenligning. Hvis du allerede ved, hvilken udbyder du ønsker, skal du først springe til dens detaljeside og derefter fortsætte til Claude Code-indgangen i den forenede AI Agent CLI-hub.
:::

## Aktuelle anbefalede udbydere

| Udbyder | Styrke | God pasform til | Detalje side |
|----------|----------|--------------|-------------|
| ZAI | Kodningsfokuserede planer med tiermapping | Udviklere, der ønsker Claude Code som daglig chauffør | [Se detaljer](./zai/) |
| MiniMax | Én standardmodel på tværs af niveauer, enkel onboarding | Hold optimerer til lavfriktionsopsætning | [Se detaljer](./minimax/) |
| Alibaba Cloud DashScope | Cloud-konsoltilpasning og samlet administration | Teams, der allerede arbejder i Alibaba Cloud | [Se detaljer](./aliyun/) |
| Volcengine kodningsplan | Stærk økosystemudvidelse og multi-model platform signaler | Udviklere, der ønsker Volcengine-økosystemtilpasning | [Se detaljer](./volcengine/) |

:::bemærk
`Anthropic Official` forbliver brugbart, men det er ikke inkluderet i den første batch af dedikerede sider, fordi de nuværende forudindstillede metadata ikke markerer det som anbefalet.
:::

## Hurtige valg tips

- **Har brug for kodningsfokuserede planer og klarere niveaukortlægning**: start med [ZAI](./zai/)
- **Har brug for den nemmeste opsætningssti først**: start med [MiniMax](./minimax/)
- **Kør allerede inde i Alibaba Cloud**: start med [Alibaba Cloud DashScope](./aliyun/)
- **Ønsker Volcengine økosystemtilpasning eller multi-model platform rækkevidde**: start med [Volcengine kodningsplan](./volcengine/)
