HagiCode understøtter flere installationsstier. Start med den implementeringsmetode, der passer til dit scenario, og tilføj derefter understøttende software, når du har brug for det.

Det aktuelle aktive Agent CLI-supportområde, der er mest relevant for opsætning, er: Claude Code, Codex, GitHub Copilot, OpenCode, Hermes, QoderCLI, Kiro, Kimi, Gemini, DeepAgents og Codebuddy. Hermes, Gemini og DeepAgents bruger i øjeblikket manuelle installations- eller godkendelsesflows; de andre CLI'er afslører installationsindgangspunkter med et enkelt klik eller guidede installationsflows.

## Anbefalede næste trin

- Hvis du planlægger at købe og installere HagiCode gennem Steam, så start med [Steam installation](/da-DK/installation/steam).
- Hvis du er ny til HagiCode, så start med [Desktop installation](/da-DK/installation/desktop).
- Hvis du implementerer til et server- eller teammiljø, skal du bruge [Installation af Docker Compose](/da-DK/installation/docker-compose).
- Hvis du har brug for at bekræfte supporttilstand, opsætningstilstand eller historiske kompatibilitetsbemærkninger for en CLI, skal du åbne [AI Agent CLI installation](/da-DK/related-software-installation/ai-agent-cli/).
