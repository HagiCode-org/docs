Turbo EngineはHagiCode Desktopのコア拡張アドオンで、高頻度利用段階に入ったユーザーを対象としています。**HagiCodeを基本モードから高強度ワークモードに引き上げる**という一点に集中しています。

## 購入方法

Turbo EngineはMicrosoft Storeの買い切り永久ライセンス（ストアID: `9NSD809W18Z6`）で、購入後永続的に有効になります。HagiCode Desktopのサブスクリプションワークスペースから直接購入とステータス更新が可能です。

## アンロックされる機能

Turbo Engineを購入すると、以下の機能が永続的にアンロックされます：

| 機能 | 説明 |
| --- | --- |
| **提案同時実行上限 3 → 32** | 生成中・実行中・アーカイブ中の提案がこの上限を共有します。32同時実行により複数のタスクラインを安定して並行進行できます。 |
| **拡張ドキュメントテーマ** | 冒険団管理画面でドキュメントテーマを切り替え、異なるテーマでの用語やテキスト表現をリアルタイムにプレビューできます。 |
| **カスタムGitコミット署名** | デフォルトの`Co-Authored-By`パターンを設定、またはAI Compose Commitの単回実行時に一時的に上書きできます。 |
| **ブランドカスタマイズ** | ヘッダーロゴのアップロード、アプリタイトル、ブラウザタイトル、ブランドリンク先を設定できます。 |
| **カスタムアバターアップロード** | 現在のHeroにカスタムアバターをアップロード、または共有カスタムアバターライブラリから新規作成・編集・一括作成フローで再利用できます。 |
| **DLCアバターリソース** | Turbo Engine専用アバターパックがHero Avatar Galleryと一括作成フローに統合され、ワークフロー選択に直接参加します。 |

## 必要なユーザー

- 複数の提案やタスクを頻繁に同時進行するユーザー。
- デフォルトの3同時実行上限では足りないヘビーユーザー。
- カスタムGit署名、ドキュメントテーマ、アバター、ブランド識別が必要なユーザー。
