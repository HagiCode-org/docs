このプライバシーポリシーでは、HagiCodeが提供する際に処理するデータのカテゴリーについて説明します。 `hagicode.com` およびそのすべてのサブドメイン サイト、ソフトウェア ダウンロード ガイダンス、サポート チャネル、関連製品エクスペリエンス、および目的、保持方法、利用可能な権利。

## 当社が収集する情報のカテゴリ

お客様がHagiCodeをどのように操作するかに応じて、以下の処理が行われる場合があります。

- **アクセスおよびデバイス情報** (ブラウザの種類、オペレーティング システム、訪問したページ、リファラー リンク、タイムスタンプ、大まかな領域、デバイス ID など)。
- **利用行動データ** (ページビュー、ページ滞在時間、クリック数、スクロールアクティビティ、セッション記録、把握するために使用される集計分析など) `hagicode.com` およびそのすべてのサブドメイン サイトと関連製品ページ。
- **サポートおよびコミュニケーション情報**。お客様の名前、電子メール アドレス、問題の説明、ログの抜粋、または電子メール、問題、コミュニティ チャネル、またはフォームを通じて自発的に送信した添付ファイルなど。そして
- **ダウンロードおよび構成コンテキスト**。ドキュメントやリリースの品質を理解するためにアクセスするインストール ガイド、ダウンロード エントリ、プリセット ファイル、リリース ノートなど。

## 利用目的

その情報は次の目的で処理されます。

- 提供し、維持する `hagicode.com` およびそのすべてのサブドメイン サイト、ダウンロード エントリ ポイント、およびサポート チャネル。
- どのページ、セットアップ手順、製品説明に改善が必要かを理解します。
- サイトエラー、ダウンロードの問題、互換性障害、セキュリティリスクのトラブルシューティング。
- 質問、サポートリクエスト、フィードバック、苦情に対応するため。そして
- 必要に応じて、適用される法的、規制、施行、またはプラットフォームのセキュリティ要件を満たします。

## サイトと製品データのタッチポイント

現在のHagiCode `hagicode.com` プロパティとそのすべてのサブドメイン サイトは、以下を通じて関連データにアクセスする可能性があります。

- サイトアクセスログと基本的なリクエストメタデータ。
- Microsoft Clarity または同等のページ分析を使用して、読書フローとインタラクション ホットスポットを理解します。
- トラフィックの傾向とリファラーを理解するために使用される 51LA または同等のトラフィック分析。そして
- あなたが自発的に送信するサポート情報 `support@hagicode.com` または GitHub の問題。

HagiCode デスクトップ クライアント、バックエンド サービス、またはサードパーティの AI 統合が別の製品コンテキストで追加データを処理する場合、それらの詳細は関連する製品資料、設定、またはサードパーティの規約に文書化されます。

## 保管と保持

当社は、運用上の必要性、法的義務、セキュリティ要件に基づいて、合理的な期間データを保持します。

- 分析データは通常、集計形式で保存されるか、分析プロバイダーのデフォルトの保持ルールに従って保存されます。
- サポート電子メール、問題レポート、およびトラブルシューティング記録は、問題の解決、コミュニケーションの監査、または法的義務の履行に必要な間保持されます。そして
- 一部のログとサポート記録は、セキュリティ、悪用防止、コンプライアンスの保持、または紛争解決のために必要な場合、より長く保存される場合があります。
- 一部のデータは、Microsoft Azure、Alibaba Cloud、または関連するインフラストラクチャ機能を通じてホスト、処理、転送、バックアップ、またはその他の方法で処理される場合があります。つまり、関連するデータが異なる国または地域間で移動する可能性があります。

これらの目的でデータが不要になった場合、プラットフォームのデフォルトの保持ルールに基づいてデータを削除、匿名化するか、期限切れにします。

## サードパーティのサービス

HagiCode は、サードパーティのインフラストラクチャ、分析プロバイダー、コード ホスティング プラットフォーム、電子メール サービス、または AI サービス ベンダーに依存する場合があります。これらの当事者は、サービスを提供するために必要な場合にのみデータを処理し、独自の条件とプライバシー通知に従うことになります。

現在または将来の製品およびサイトの運営において、当社は特定のデータを Microsoft Azure や Alibaba Cloud などのクラウド インフラストラクチャを通じて処理または処理する場合があります。つまり、ストレージ、バックアップ、ログ分析、トラブルシューティング、コンテンツ配信、またはサービス オーケストレーションのために、データがさまざまな国や地域に転送される可能性があります。

合理的に実用的である場合、当社は運用分析、サービス改善、安定化作業、およびセキュリティ保護のために関連データを匿名化、集約、または匿名化された形式で使用します。ただし、クラウド プロバイダー、インフラストラクチャ パス、またはリージョン間の転送が常に 1 つの国または地域内にとどまることを完全に防止、制御、または保証することはできないことを理解し、これに同意するものとします。

サードパーティ統合を使用する場合は、関連するプロバイダーのポリシーも確認して、独自の処理と国境を越えた転送の慣行を理解する必要があります。

このクラウドベースの処理、国境を越えた転送、または関連するデータの匿名化された使用を受け入れることができない場合は、HagiCode 製品、サービス、またはサービスをダウンロード、インストール、アクセス、または使用を継続しないでください。 `hagicode.com` およびそのサブドメイン サイトのいずれか。

## ユーザーの権利

適用される法律で定められている範囲で、お客様は次のことを要求できます。

- 当社がお客様に関連する個人情報を保持しているかどうかを知る。
- 不正確または不完全な情報を修正する。
- 不要になった情報、または消去する権利のある情報を削除する。
- 特定の処理活動を制限または拒否する。そして
- サポートに関する連絡や苦情に関するさらなる説明を得る。

リクエストがアカウントのセキュリティ、法的義務、システムのセキュリティ、悪用防止、またはサードパーティのプラットフォームの制限に影響を与える場合、最初に身元を確認するか、法律で許可されている限定的なデータを保持する必要がある場合があります。
