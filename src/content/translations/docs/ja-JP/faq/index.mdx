## このセクションの内容

FAQ セクションは次の場所です。

- 起動時と実行時の注意事項
- プラットフォーム、流通チャネル、パッケージ化、またはバンドルされたランタイムに関連する互換性に関する注意事項
- 安定したスタンドアロンの説明ページが必要なトラブルシューティングのトピック

Desktop がどの Node を使用するのか、または診断で `portable-fixed/toolchain` が言及される理由を確認したい場合は、[デスクトップノード環境](/ja-JP/faq/desktop-node-environment) を開いてください。

HagiCode が Steam にない理由を知りたい場合は、[HagiCode が Steam で提供されていない理由](/ja-JP/faq/steam-distribution-status) を開いてください。
