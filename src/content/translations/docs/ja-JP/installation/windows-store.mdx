Windows ユーザー向けには、Windows Store が現在の公開デスクトップ導入口です。このページでは Store 経由の導入だけを扱います。通常の Desktop 配布、版管理、トラブルシュートは [Desktop Installation](/ja-JP/installation/desktop) を参照してください。

Steam のメインアプリ登録はすでに退役しているため、現在の主要な Windows 配布チャネルは Windows Store です。背景は [Why HagiCode is not available on Steam](/ja-JP/faq/steam-distribution-status/) を参照してください。

:::tip[この方法が向いているケース]
- Windows 10 または Windows 11 を利用していて、システムストアから直接導入したい
- インストーラーファイルを手動で扱うより、Store 経由でアプリ本体を受け取りたい
- 個人向けのローカル環境だけが必要で、Docker Compose 配備は予定していない
:::

## 事前確認

- 端末で [HagiCode for Windows の Windows Store ページ](https://apps.microsoft.com/detail/9N3PM0N3SVDW) を開けること
- 初回起動時に追加コンポーネントを準備する可能性があるため、ネットワークを利用できること
- デバイス、地域、組織ポリシーによっては Microsoft アカウントのサインインを求められることがある
- Store が利用できない環境では [Desktop Installation](/ja-JP/installation/desktop) に切り替える

## Windows Store からインストールする

1. [HagiCode for Windows](https://apps.microsoft.com/detail/9N3PM0N3SVDW) を開きます。
2. `入手`、`インストール`、または同等のボタンを押します。
3. Windows Store によるダウンロードとインストールが完了するまで待ちます。
4. 完了後、スタートメニューまたは Store ページから HagiCode を起動します。

## 初回起動時の挙動

インストール後も、HagiCode は通常の Desktop 初期化フローを続けます。

- 初回起動はローカル実行環境の準備があるため、一般的なアプリより遅く見えることがある
- Windows Store / MSIX チャネルでは、一部の依存関係管理やランタイムパス設定が意図的にロックされる
- アプリ本体は独立した自己更新を行わず、配布は導入チャネルに依存する。Store 版のパッケージ配信は Windows Store チャネルに結び付いている

オンボーディングを続ける場合は [Wizard Setup](/ja-JP/quick-start/wizard-setup) を参照してください。

## 別の導入方法を選ぶべき場合

- Windows Store にアクセスできない、または組織ポリシーで Store インストールが禁止されている
- `.exe` や `.msix` を手動で取得し、インストーラーの取得元を自分で管理したい
- サーバー、NAS、チーム環境に展開するため、[Docker Compose Deployment](/ja-JP/installation/docker-compose) の方が適している

## よくある問題

### Windows Store ページが開かない

まずネットワーク、地域設定、Microsoft Store のサインイン状態を確認してください。ブラウザでリンクが開いても Store クライアントが使えない場合は、ローカルのシステムポリシーが原因であることが多く、その場合は [Desktop Installation](/ja-JP/installation/desktop) を利用してください。

### 一部の設定がロックされている

Store / MSIX インストールでは、依存関係管理やランタイムデータパスの一部設定が制限されます。これはインストール失敗ではなく、配布チャネルの仕様です。

### 初回起動でさらにダウンロードや準備が走る

Store パッケージが配布するのはデスクトップアプリ本体です。初回起動時には、HagiCode がバージョン情報の確認、ローカルコンポーネントの準備、既定の初期化を続けることがあります。ネットワークを維持したまま現在のステップが終わるまで待ってください。
