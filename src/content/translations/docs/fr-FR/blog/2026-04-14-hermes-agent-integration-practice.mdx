## Pratique d'intégration de Hermes Agent : du protocole à la production

> Partage de l'expérience complète d'HagiCode d'intégration de Hermes Agent, incluant les leçons fondamentales autour de l'adaptation du protocole ACP, la gestion du pool de sessions et la synchronisation des contrats frontend/backend.

## Contexte

Lors du développement de HagiCode, une plateforme de codage assistée par IA, notre équipe devait intégrer un framework Agent capable de fonctionner localement et de passer à l'échelle vers le cloud. Après recherche, Hermes Agent de Nous Research a été choisi comme moteur sous-jacent pour nos capacités d'Agent généraliste.

## À propos de HagiCode

L'approche partagée dans cet article provient de notre expérience pratique dans le projet [HagiCode](https://hagicode.com).

## Adaptation du protocole ACP

L'intégration de Hermes Agent nécessitait l'adaptation du protocole ACP (Agent Communication Protocol) pour assurer la communication fluide entre le frontend et le backend.

## Gestion du pool de sessions

La gestion efficace des sessions est cruciale pour une expérience utilisateur fluide. HagiCode utilise Orleans Grains pour gérer le cycle de vie des sessions.

## Conclusion

L'intégration de Hermes Agent dans HagiCode a été un processus d'apprentissage précieux. Les leçons apprises peuvent servir de référence pour d'autres projets cherchant à intégrer des frameworks Agent.
