## Ce que couvre cette section

La section FAQ est l'endroit idéal pour :

- mises en garde concernant le démarrage et l'exécution
- notes de compatibilité liées aux plates-formes, aux canaux de distribution, aux emballages ou aux environnements d'exécution groupés
- sujets de dépannage qui méritent une page d'explication autonome et stable

Si vous devez confirmer quel Node Desktop utilise ou pourquoi les diagnostics mentionnent `portable-fixed/toolchain`, ouvrez [Environnement de nœud de bureau](/fr-FR/faq/desktop-node-environment).

Si vous voulez savoir pourquoi HagiCode n’est pas sur Steam, ouvrez [Pourquoi HagiCode n’est pas disponible sur Steam](/fr-FR/faq/steam-distribution-status).
