Turbo Engine est l'extension essentielle pour HagiCode Desktop, conçue pour les utilisateurs passés en phase d'utilisation intensive. Elle résout une chose : **faire passer HagiCode du mode de base au mode de travail haute intensité.**

## Comment acheter

Turbo Engine est une licence permanente à achat unique dans le Microsoft Store (ID Store : `9NSD809W18Z6`), effective définitivement après l'achat. Vous pouvez acheter et actualiser le statut de la licence directement depuis l'espace de travail d'abonnement dans HagiCode Desktop.

## Fonctionnalités débloquées

Après l'achat de Turbo Engine, les fonctionnalités suivantes sont débloquées de façon permanente :

| Fonctionnalité | Description |
| --- | --- |
| **Concurrence des propositions 3 → 32** | Les propositions en génération, exécution et archivage partagent cette limite. 32 propositions simultanées vous permettent de faire avancer plusieurs lignes de tâches en parallèle. |
| **Thèmes de documents étendus** | Basculez entre les thèmes de documents dans l'interface du Groupe d'Aventure et prévisualisez la terminologie et les textes sous différents thèmes en temps réel. |
| **Signatures Git personnalisées** | Définissez un modèle `Co-Authored-By` par défaut, ou remplacez-le temporairement pour des exécutions individuelles d'AI Compose Commit. |
| **Personnalisation de la marque** | Téléchargez un logo d'en-tête et définissez le titre de l'application, le titre du navigateur et la cible du lien de marque. |
| **Téléchargement d'avatars personnalisés** | Téléchargez un avatar personnalisé pour le Héros actuel, ou réutilisez des entrées de la bibliothèque d'avatars partagée dans les flux de création, d'édition et de création par lots. |
| **Ressources d'avatars DLC** | Les packs d'avatars Turbo Engine sont intégrés à la Galerie d'Avatars de Héros et aux flux de création par lots, participant directement à la sélection des flux de travail. |

## Qui en a besoin

- Les utilisateurs qui font fréquemment avancer plusieurs propositions ou tâches simultanément.
- Les utilisateurs intensifs pour qui la limite par défaut de 3 propositions simultanées n'est plus suffisante.
- Les utilisateurs ayant besoin de signatures Git personnalisées, de thèmes de documents, d'avatars et d'identité de marque.
