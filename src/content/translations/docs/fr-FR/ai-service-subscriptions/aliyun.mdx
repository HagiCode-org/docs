## Positionnement du service

Alibaba Cloud DashScope est l'un des points d'abonnement Claude Code recommandes dans le preset de la documentation actuelle. Il convient aux equipes travaillant deja dans l'ecosysteme Alibaba Cloud qui souhaitent unifier les abonnements, la console et la gestion des cles au sein de la plateforme cloud.

## Pour qui

- Ceux qui utilisent deja la console Alibaba Cloud ou des ressources cloud
- Ceux qui souhaitent gerer les services et les cles API depuis une plateforme cloud unifiee
- Ceux qui souhaitent integrer rapidement Claude Code via un mappage de modeles fixe

## Points cles a retenir

- Le preset actuel marque Alibaba Cloud DashScope comme fournisseur recommande.
- Il utilise `glm-4.7` comme modele de mappage par defaut, avec un chemin de configuration clair.
- Si l'equipe dispose deja d'un systeme de comptes Alibaba Cloud, l'integration et la gestion des permissions seront plus fluides.

## Conseils de configuration

1. Confirmez d'abord le forfait ou la methode d'activation de cle API que vous souhaitez utiliser.
2. Accedez ensuite a la section Claude Code de la page unifiee des CLI d'agents IA pour poursuivre vers l'installation officielle et la documentation de reference d'Anthropic.
3. Une fois termine, vous pouvez verifier la reussite de l'integration en executant localement `claude --version`, en verifiant le fichier de configuration ou en effectuant une requete reelle.
