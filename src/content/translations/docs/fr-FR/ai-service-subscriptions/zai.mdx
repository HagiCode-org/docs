## Positionnement du service

Zhipu AI est l'un des points d'abonnement Claude Code recommandes dans le preset de la documentation actuelle. Le forfait GLM Coding et le mappage multi-modeles conviennent aux developpeurs qui utilisent Claude Code comme outil principal au quotidien.

## Pour qui

- Ceux qui souhaitent obtenir un forfait specialement concu pour les scenarios de developpement
- Ceux qui ont besoin d'un mappage complet des modeles de la gamme Claude
- Ceux qui souhaitent couvrir plusieurs outils comme Claude Code et Cline via un seul service

## Points cles a retenir

- Le preset actuel marque Zhipu AI comme fournisseur recommande.
- Il offre une interface compatible Anthropic, facilitant la connexion directe avec Claude Code.
- Le preset inclut des mappages de modeles par defaut tels que `glm-4.7`, `glm-5` et `glm-4.5-air`.

## Mappage de modeles par defaut

| Niveau Claude | Modele par defaut |
|---------------|-------------------|
| Sonnet | `glm-4.7` |
| Opus | `glm-5` |
| Haiku | `glm-4.5-air` |

:::tip
Les forfaits, les promotions et les avantages evoluent dans le temps ; si vous souhaitez connaitre les prix en temps reel, veuillez consulter directement la page d'abonnement du fournisseur.
:::

## Conseils de configuration

1. Confirmez d'abord le forfait ou la methode d'activation de cle API que vous souhaitez utiliser.
2. Accedez ensuite a la section Claude Code de la page unifiee des CLI d'agents IA pour poursuivre vers l'installation officielle et la documentation de reference d'Anthropic.
3. Une fois termine, vous pouvez verifier la reussite de l'integration en executant localement `claude --version`, en verifiant le fichier de configuration ou en effectuant une requete reelle.
