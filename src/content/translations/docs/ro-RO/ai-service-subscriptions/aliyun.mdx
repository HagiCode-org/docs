## Poziționare

Alibaba Cloud DashScope este unul dintre furnizorii recomandați în prezent de setările prestabilite de documente pentru Claude Code. O alegere practică pentru echipele care operează deja în ecosistemul Alibaba Cloud și care doresc un management unificat al consolei.

## Potrivit pentru

- Utilizați deja conturi sau resurse Alibaba Cloud
- Doriți ca abonamentul și gestionarea cheilor să rămână într-o singură consolă cloud
- Preferiți un model cu cartografiere fixă ​​pentru o integrare rapidă

## Lucruri cheie de observat

- Metadatele presetate actuale marchează Alibaba Cloud DashScope așa cum este recomandat.
- Mapează toate nivelurile Claude către `glm-4.7` pentru o cale de configurare simplă.
- Este deosebit de convenabil dacă echipa ta lucrează deja în fluxurile de lucru Alibaba Cloud.

## Instantaneu de integrare

| Articol | Valoare |
|------|-------|
| Intrare abonament | [Deschideți pagina de abonament](https://www.aliyun.com/benefit/ai/aistar?userCode=vmx5szbq&clubBiz=subTask..12384055..10263..) |
| Documente oficiale/consola | [Deschide pagina oficială](https://dashscope.aliyun.com) |
| URL de bază compatibil cu antropic | `https://coding.dashscope.aliyuncs.com/apps/anthropic` |
| Autentificare recomandată var | `ANTHROPIC_AUTH_TOKEN` |

## Mapare implicită a modelului

| Claude tier | Model implicit |
|-------------|---------------|
| Sonet | `glm-4.7` |
| Opus | `glm-4.7` |
| Haiku | `glm-4.7` |

## Pașii următori sugerați

1. Confirmați ce nivel de abonament sau calea de activare a API-ului doriți să utilizați.
2. Deschideți intrarea Claude Code în hub-ul unificat AI Agent CLI și continuați cu documentele oficiale de instalare și referință Anthropic.
3. Validați conexiunea la nivel local după configurare.
