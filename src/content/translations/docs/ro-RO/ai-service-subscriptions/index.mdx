Acest capitol colectează site-urile web cu abonament la servicii AI marcate în prezent ca fiind recomandate în `repos/docs/public/presets/claude-code/providers/*.json`. Alegeți mai întâi un furnizor aici, apoi continuați la configurarea Claude Code.

Începând cu **9 martie 2026**, acest capitol acoperă: **ZAI, MiniMax, Alibaba Cloud DashScope și Volcengine Coding Plan**.

:::sfat[Ordine de citire sugerată]
Dacă nu ați ales încă un furnizor, începeți cu această pagină pentru comparație. Dacă știți deja ce furnizor doriți, săriți mai întâi la pagina sa de detalii și apoi continuați la intrarea Claude Code din hub-ul unificat CLI AI Agent.
:::

## Furnizori actuali recomandați

| Furnizor | Rezistenţă | Potrivit pentru | Pagina de detalii |
|----------|----------|--------------|-------------|
| ZAI | Planuri axate pe codificare cu mapare a nivelurilor | Dezvoltatori care îl doresc pe Claude Code ca șofer zilnic | [Vezi detalii](./zai/) |
| MiniMax | Un model implicit pe mai multe niveluri, integrare simplă | Optimizarea echipelor pentru configurarea cu frecare redusă | [Vezi detalii](./minimax/) |
| Alibaba Cloud DashScope | Alinierea cloud-console și management unificat | Echipe care lucrează deja în Alibaba Cloud | [Vezi detalii](./aliyun/) |
| Planul de codificare Volcengine | Extensie puternică a ecosistemului și semnale de platformă multi-model | Dezvoltatori care doresc alinierea ecosistemului Volcengine | [Vezi detalii](./volcengine/) |

:::nota
`Anthropic Official` rămâne utilizabilă, dar nu este inclusă în primul lot de pagini dedicate deoarece metadatele presetate curente nu le marchează ca fiind recomandate.
:::
