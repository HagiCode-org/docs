## Poziționare

ZAI este unul dintre furnizorii recomandați în prezent de setările prestabilite de documente pentru Claude Code. Planuri de codare GLM cu mapare a nivelurilor pentru dezvoltatorii care doresc Claude Code ca driver zilnic.

## Potrivit pentru

- Doriți un plan de abonament axat pe codificare
- Aveți nevoie de valorile implicite mapate Sonnet/Opus/Haiku
- Vă așteptați ca un furnizor să lucreze cu Claude Code și instrumente de codare similare

## Lucruri cheie de observat

- Metadatele presetate curente marchează ZAI așa cum este recomandat.
- Expune un punct final compatibil cu antropic care poate fi utilizat direct de Claude Code.
- Presetarea oferă mapări implicite pentru `glm-4.7`, `glm-5`, și `glm-4.5-air`.

## Instantaneu de integrare

| Articol | Valoare |
|------|-------|
| Intrare abonament | [Deschideți pagina de abonament](https://www.bigmodel.cn/claude-code?ic=14BY54APZA) |
| Documente oficiale/consola | [Deschide pagina oficială](https://open.bigmodel.cn/dev/api) |
| URL de bază compatibil cu antropic | `https://open.bigmodel.cn/api/anthropic` |
| Autentificare recomandată var | `ANTHROPIC_AUTH_TOKEN` |

## Mapare implicită a modelului

| Claude tier | Model implicit |
|-------------|---------------|
| Sonet | `glm-4.7` |
| Opus | `glm-5` |
| Haiku | `glm-4.5-air` |

:::sfat
Prețurile, promoțiile și detaliile pachetului se pot schimba în timp. Dacă costul este principala dvs. preocupare, confirmați cele mai recente informații direct pe pagina furnizorului.
:::

## Pașii următori sugerați

1. Confirmați ce nivel de abonament sau calea de activare a API-ului doriți să utilizați.
2. Deschideți intrarea Claude Code în hub-ul unificat AI Agent CLI și continuați cu documentele oficiale de instalare și referință Anthropic.
3. Validați conexiunea la nivel local după configurare.
