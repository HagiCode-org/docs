HagiCode acceptă mai multe căi de instalare. Începeți cu metoda de implementare care se potrivește scenariului dvs., apoi adăugați software de asistență numai atunci când aveți nevoie de el.

Gama actuală de suport CLI agent activă cea mai relevantă pentru configurare este: Claude Code, Codex, GitHub Copilot, OpenCode, Hermes, QoderCLI, Kiro, Kimi, Gemini, DeepAgents și Codebuddy. Hermes, Gemini și DeepAgents folosesc în prezent fluxuri de instalare manuală sau de autentificare; celelalte CLI-uri expun puncte de intrare la instalare cu un singur clic sau fluxuri de instalare ghidate.

## Următorii pași recomandați

- Dacă intenționați să cumpărați și să instalați HagiCode prin Steam, începeți cu [Instalare cu abur](/ro-RO/installation/steam).
- Dacă sunteți nou în HagiCode, începeți cu [Instalare desktop](/ro-RO/installation/desktop).
- Dacă implementați pentru un server sau un mediu de echipă, utilizați [Instalare Docker Compose](/ro-RO/installation/docker-compose).
- Dacă trebuie să confirmați starea suportului, modul de configurare sau notele de compatibilitate istorice pentru un CLI, deschideți [Instalare CLI Agent AI](/ro-RO/related-software-installation/ai-agent-cli/).
