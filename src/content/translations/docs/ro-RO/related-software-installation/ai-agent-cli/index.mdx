Această pagină este intrarea locală unificată de instalare AI Agent CLI în `repos/docs`. HagiCode acceptă în prezent 11 CLI agent activ, după backend `PrimaryProfessions` comanda: Claude Code, Codex, GitHub Copilot, OpenCode, Hermes, QoderCLI, Kiro, Kimi, Gemini, DeepAgents și Codebuddy.

:::sfat[Cum se utilizează această pagină]
Verificați mai întâi tabelul pentru starea actuală de asistență și modul de configurare, apoi continuați cu documentele de instalare, autentificare, ACP, permisiuni și depanare menținute de furnizor. Domeniul de aplicare și modul de configurare sunt preocupări separate.
:::

## Glosar de stat

- **Suportat în prezent** înseamnă că CLI aparține intervalului de suport HagiCode activ curent.
- **Instalarea cu un singur clic** înseamnă că HagiCode oferă un punct de intrare ghidat pentru instalare; unele CLI necesită încă un pas de conectare la furnizor sau de autentificare după instalare.
- **Instalarea manuală/autentificarea** înseamnă că CLI este acceptat acum, dar trebuie să finalizați instalarea locală sau fluxul de autentificare înainte ca HagiCode să îl poată detecta și utiliza.
- **Numai compatibilitatea moștenită** înseamnă o valoare retrasă, cum ar fi `IFlowCli` există încă pentru compatibilitatea datelor istorice, dar nu face parte din gama actuală de suport activ.
