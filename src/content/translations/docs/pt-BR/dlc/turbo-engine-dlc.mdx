O Turbo Engine é o complemento essencial para o HagiCode Desktop, projetado para usuários que já entraram na fase de uso intensivo. Ele resolve uma coisa: **levar o HagiCode do modo básico para o modo de trabalho de alta intensidade.**

## Como comprar

O Turbo Engine é uma licença permanente de compra única na Microsoft Store (ID da Loja: `9NSD809W18Z6`), válida permanentemente após a compra. Você pode comprar e atualizar o status da licença diretamente no espaço de trabalho de assinatura do HagiCode Desktop.

## Funcionalidades desbloqueadas

Após comprar o Turbo Engine, as seguintes capacidades são desbloqueadas permanentemente:

| Funcionalidade | Descrição |
| --- | --- |
| **Concorrência de propostas 3 → 32** | Propostas em geração, execução e arquivamento compartilham este limite. 32 propostas simultâneas permitem avançar várias linhas de tarefas em paralelo. |
| **Temas de documentos expandidos** | Alterne entre temas de documentos na interface do Grupo de Aventura, visualizando terminologia e textos sob diferentes temas em tempo real. |
| **Assinaturas Git personalizadas** | Defina um padrão `Co-Authored-By` padrão ou substitua-o temporariamente em execuções individuais do AI Compose Commit. |
| **Personalização de marca** | Carregue um logotipo de cabeçalho e defina o título do aplicativo, título do navegador e link de destino da marca. |
| **Upload de avatares personalizados** | Carregue um avatar personalizado para o Herói atual ou reutilize entradas da biblioteca compartilhada de avatares nos fluxos de criação, edição e criação em lote. |
| **Recursos de avatares DLC** | Os pacotes de avatares Turbo Engine são integrados à Galeria de Avatares de Herói e aos fluxos de criação em lote, participando diretamente da seleção do fluxo de trabalho. |

## Quem precisa

- Usuários que frequentemente avançam várias propostas ou tarefas ao mesmo tempo.
- Usuários avançados para quem o limite padrão de 3 propostas simultâneas já não é suficiente.
- Usuários que precisam de assinaturas Git personalizadas, temas de documentos, avatares e identidade de marca.
