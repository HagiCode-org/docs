У цьому розділі зібрано веб-сайти підписки на послуги ШІ, які зараз позначені як рекомендовані в `repos/docs/public/presets/claude-code/providers/*.json`. Спочатку виберіть тут постачальника, а потім перейдіть до налаштування Claude Code.

Станом на **9 березня 2026 року** цей розділ охоплює: **ZAI, MiniMax, Alibaba Cloud DashScope та Volcengine Coding Plan**.

:::tip[Пропонований порядок читання]
Якщо ви ще не вибрали постачальника, почніть із цієї сторінки для порівняння. Якщо ви вже знаєте, якого постачальника вам потрібно, спочатку перейдіть до його сторінки з деталями, а потім перейдіть до запису Claude Code в уніфікованому центрі CLI агента AI Agent.
:::

## Поточні рекомендовані постачальники

| Провайдер | Сила | Добре підходить для | Сторінка деталей |
|----------|----------|--------------|-------------|
| ЗАІ | Плани, орієнтовані на кодування, із відображенням рівнів | Розробники, які хочуть, щоб Клод Код був щоденним водієм | [Переглянути деталі](./zai/) |
| МініМакс | Одна модель за замовчуванням на всіх рівнях, проста адаптація | Команди оптимізують установку з низьким тертям | [Переглянути деталі](./minimax/) |
| Alibaba Cloud DashScope | Вирівнювання хмарної консолі та уніфіковане керування | Команди вже працюють в Alibaba Cloud | [Переглянути деталі](./aliyun/) |
| План кодування Volcengine | Сильне розширення екосистеми та мультимодельні сигнали платформи | Розробники, які хочуть узгодження екосистеми Volcengine | [Переглянути деталі](./volcengine/) |

:::примітка
`Anthropic Official` залишається придатним для використання, але його не включено до першого пакету виділених сторінок, оскільки поточні попередньо встановлені метадані не позначають його як рекомендований.
:::

## Швидкі поради щодо вибору

- **Потрібні плани, орієнтовані на кодування, і чіткіше відображення рівнів**: почніть із цього [ЗАІ](./zai/)
- **Спочатку потрібен найпростіший шлях налаштування**: почніть із [МініМакс](./minimax/)
- **Уже працює в Alibaba Cloud**: почніть з [Alibaba Cloud DashScope](./aliyun/)
- **Бажаєте вирівняти екосистему Volcengine або охопити багатомодельну платформу**: почніть із цього [План кодування Volcengine](./volcengine/)
