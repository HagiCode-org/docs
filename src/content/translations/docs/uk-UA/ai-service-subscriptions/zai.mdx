## Позиціонування

ZAI є одним із постачальників, рекомендованих наразі стандартними налаштуваннями документів для Claude Code. Плани GLM Coding із відображенням рівнів для розробників, які хочуть Claude Code як щоденний драйвер.

## Добре підходить для

- Вам потрібен план підписки, орієнтований на кодування
- Вам потрібні стандартні параметри Sonnet/Opus/Haiku
- Ви очікуєте, що один постачальник працюватиме з Claude Code та подібними інструментами кодування

## Ключові речі, на які варто звернути увагу

- Поточні попередньо встановлені метадані позначають ZAI як рекомендований.
- Він відкриває кінцеву точку, сумісну з Anthropic, яку може використовувати безпосередньо Клод Код.
- Попередній набір надає зіставлення за замовчуванням для `glm-4.7`, `glm-5`, і `glm-4.5-air`.

| Пункт | Значення |

| Рекомендована автентифікація env var | `ANTHROPIC_AUTH_TOKEN` |

## Відображення моделі за замовчуванням

| Ярус Клода | Модель за замовчуванням |
|-------------|---------------|
| Сонет | `glm-4.7` |
| Опус | `glm-5` |
| Хайку | `glm-4.5-air` |

:::підказка
Ціни, акції та деталі пакету можуть змінюватися з часом. Якщо вартість — це ваша головна проблема, перевірте останню інформацію безпосередньо на сторінці постачальника.
:::

## Запропоновані наступні кроки

1. Підтвердьте, який рівень підписки або шлях активації API ви хочете використовувати.
2. Відкрийте запис Claude Code в уніфікованому центрі CLI агента AI Agent і перейдіть до офіційної інсталяційної та довідкової документації Anthropic.
3. Перевірте підключення локально після налаштування.
