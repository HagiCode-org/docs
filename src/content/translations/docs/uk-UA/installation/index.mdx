HagiCode підтримує кілька шляхів встановлення. Почніть із методу розгортання, який відповідає вашому сценарію, а потім додавайте допоміжне програмне забезпечення лише тоді, коли воно вам потрібно.

Поточний активний діапазон підтримки CLI агента, найбільш відповідний для налаштування: Claude Code, Codex, GitHub Copilot, OpenCode, Hermes, QoderCLI, Kiro, Kimi, Gemini, DeepAgents і Codebuddy. Hermes, Gemini та DeepAgents наразі використовують ручну інсталяцію або автентифікацію; інші CLI відображають точки входу для встановлення одним клацанням миші або керовані потоки встановлення.

## Рекомендовані наступні дії

- Якщо ви плануєте купити та встановити HagiCode через Steam, почніть з [Установка Steam](/uk-UA/installation/steam).
- Якщо ви новачок у HagiCode, почніть із [Інсталяція на робочий стіл](/uk-UA/installation/desktop).
- Якщо ви розгортаєте сервер або командне середовище, використовуйте [Інсталяція Docker Compose](/uk-UA/installation/docker-compose).
- Якщо вам потрібно підтвердити стан підтримки, режим налаштування або історичні примітки щодо сумісності для CLI, відкрийте [Інсталяція агента AI Agent CLI](/uk-UA/related-software-installation/ai-agent-cli/).
