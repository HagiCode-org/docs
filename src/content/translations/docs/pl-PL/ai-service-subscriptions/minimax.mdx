## Pozycjonowanie

MiniMax jest jednym z dostawców obecnie zalecanych przez zestaw gotowych dokumentów dla Claude Code. Pojedynczy model domyślny na wszystkich poziomach, odpowiedni dla zespołów ceniących prostą konfigurację i przewidywalne wdrożenie.

## Dobre dopasowanie do

- Chcesz mieć ten sam zmapowany model na wszystkich poziomach Claude
- Preferujesz konfigurację i konserwację o niskim tarciu
- Chcesz szybko uruchomić Claude Code, zanim dostosujesz strategię modelu

## Najważniejsze rzeczy, na które warto zwrócić uwagę

- Bieżące, wstępnie ustawione metadane oznaczają MiniMax jako zalecany.
- Używa `MiniMax-M2.5` jako domyślny model mapowany dla wszystkich warstw.
- Jest to dobre rozwiązanie, gdy prostota obsługi jest ważniejsza niż dostrajanie specyficzne dla poziomu.

| Poziom Claude'a | Domyślny model |
|-------------|---------------|
| Sonet | `MiniMax-M2.5` |
| Opus | `MiniMax-M2.5` |
| Haiku | `MiniMax-M2.5` |

## Sugerowane dalsze kroki

1. Potwierdź, którego poziomu subskrypcji lub ścieżki aktywacji interfejsu API chcesz użyć.
2. Otwórz wpis Claude Code w ujednoliconym centrum CLI AI Agent i kontynuuj korzystanie z oficjalnej dokumentacji instalacyjnej i referencyjnej Anthropic.
3. Po zakończeniu konfiguracji sprawdź połączenie lokalnie.
