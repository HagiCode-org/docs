## Pozycjonowanie

Alibaba Cloud DashScope jest jednym z dostawców obecnie zalecanych przez zestaw ustawień dokumentów dla Claude Code. Praktyczny wybór dla zespołów działających już w ekosystemie Alibaba Cloud i chcących ujednoliconego zarządzania konsolą.

## Dobre dopasowanie do

- Korzystasz już z kont lub zasobów Alibaba Cloud
- Chcesz, aby zarządzanie subskrypcjami i kluczami odbywało się w jednej konsoli w chmurze
- Preferujesz model ze stałą mapą, umożliwiający szybkie wdrożenie

## Najważniejsze rzeczy, na które warto zwrócić uwagę

- Bieżące wstępnie ustawione metadane oznaczają Alibaba Cloud DashScope jako zalecane.
- Odwzorowuje wszystkie poziomy Claude'a na `glm-4.7` dla prostej ścieżki konfiguracji.
- Jest to szczególnie wygodne, jeśli Twój zespół pracuje już w przepływach pracy Alibaba Cloud.

## Sugerowane dalsze kroki

1. Potwierdź, którego poziomu subskrypcji lub ścieżki aktywacji interfejsu API chcesz użyć.
2. Otwórz wpis Claude Code w ujednoliconym centrum CLI AI Agent i kontynuuj korzystanie z oficjalnej dokumentacji instalacyjnej i referencyjnej Anthropic.
3. Po zakończeniu konfiguracji sprawdź połączenie lokalnie.
