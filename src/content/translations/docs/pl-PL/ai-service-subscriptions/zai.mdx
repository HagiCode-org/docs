## Pozycjonowanie

ZAI jest jednym z dostawców obecnie rekomendowanych przez zestaw gotowych dokumentów dla Claude Code. Plany GLM Coding z mapowaniem poziomów dla programistów, którzy chcą Claude Code jako codziennego sterownika.

## Dobre dopasowanie do

- Chcesz mieć plan subskrypcji skoncentrowany na kodowaniu
- Potrzebujesz zmapowanych ustawień domyślnych Sonnet/Opus/Haiku
- Oczekujesz, że jeden dostawca będzie obsługiwał Claude Code i podobne narzędzia do kodowania

## Najważniejsze rzeczy, na które warto zwrócić uwagę

- Obecne wstępnie ustawione metadane oznaczają ZAI jako zalecane.
- Udostępnia punkt końcowy zgodny z Anthropic, z którego może korzystać bezpośrednio Claude Code.
- To ustawienie wstępne zawiera domyślne mapowania dla `glm-4.7`, `glm-5`, i `glm-4.5-air`.

## Migawka integracji

| Przedmiot | Wartość |
|------|-------|
| Wpis subskrypcji | [Otwórz stronę subskrypcji](https://www.bigmodel.cn/claude-code?ic=14BY54APZA) |
| Oficjalne dokumenty/konsola | [Otwórz oficjalną stronę](https://open.bigmodel.cn/dev/api) |
| Podstawowy adres URL zgodny z antropią | `https://open.bigmodel.cn/api/anthropic` |
| Zalecane uwierzytelnianie env var | `ANTHROPIC_AUTH_TOKEN` |

## Domyślne mapowanie modelu

| Poziom Claude'a | Domyślny model |
|-------------|---------------|
| Sonet | `glm-4.7` |
| Opus | `glm-5` |
| Haiku | `glm-4.5-air` |

:::wskazówka
Ceny, promocje i szczegóły pakietu mogą się zmieniać z biegiem czasu. Jeśli głównym problemem są dla Ciebie koszty, potwierdź najnowsze informacje bezpośrednio na stronie dostawcy.
:::

## Sugerowane dalsze kroki

1. Potwierdź, którego poziomu subskrypcji lub ścieżki aktywacji interfejsu API chcesz użyć.
2. Otwórz wpis Claude Code w ujednoliconym centrum CLI AI Agent i kontynuuj korzystanie z oficjalnej dokumentacji instalacyjnej i referencyjnej Anthropic.
3. Po zakończeniu konfiguracji sprawdź połączenie lokalnie.
