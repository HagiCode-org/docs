W tym rozdziale zebrano strony internetowe z subskrypcją usług AI, aktualnie oznaczone jako polecane `repos/docs/public/presets/claude-code/providers/*.json`. Najpierw wybierz dostawcę, a następnie przejdź do konfiguracji Claude Code.

Stan na **9 marca 2026 r.** ten rozdział obejmuje: **ZAI, MiniMax, Alibaba Cloud DashScope i plan kodowania Volcengine**.

:::tip[Sugerowana kolejność czytania]
Jeśli jeszcze nie wybrałeś dostawcy, zacznij od tej strony dla porównania. Jeśli już wiesz, jakiego dostawcy szukasz, przejdź najpierw do jego strony ze szczegółami, a następnie przejdź do wpisu Claude Code w zunifikowanym centrum CLI AI Agent.
:::

## Aktualni rekomendowani dostawcy

| Dostawca | Siła | Dobre dopasowanie do | Strona ze szczegółami |
|----------|----------|--------------|-------------|
| ZAI | Plany skoncentrowane na kodowaniu z mapowaniem poziomów | Deweloperzy, którzy chcą Claude Code jako codziennego sterownika | [Zobacz szczegóły](./zai/) |
| MiniMax | Jeden domyślny model na wszystkich poziomach, proste wdrożenie | Zespoły optymalizujące konfigurację pod kątem niskiego tarcia | [Zobacz szczegóły](./minimax/) |
| Alibaba Cloud DashScope | Dopasowanie konsoli w chmurze i ujednolicone zarządzanie | Zespoły już pracujące w Alibaba Cloud | [Zobacz szczegóły](./aliyun/) |
| Plan kodowania Volcengine | Silne rozszerzenie ekosystemu i sygnały dotyczące platform wielomodelowych | Deweloperzy, którzy chcą dostosowania ekosystemu Volcengine | [Zobacz szczegóły](./volcengine/) |

:::uwaga
`Anthropic Official` pozostaje użyteczny, ale nie jest uwzględniony w pierwszej partii dedykowanych stron, ponieważ obecne wstępnie ustawione metadane nie oznaczają go jako zalecanego.
:::
