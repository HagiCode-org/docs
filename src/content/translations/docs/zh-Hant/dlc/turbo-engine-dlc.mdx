Turbo Engine 是 HagiCode Desktop 的核心增強插件，面向已經進入高頻使用階段的使用者。它解決的是一件事：**把 HagiCode 從基礎態推向高強度工作態。**

## 購買方式

Turbo Engine 是 Microsoft Store 中的一次性買斷授權（Store ID: `9NSD809W18Z6`），購買後永久生效。你可以在 HagiCode Desktop 的訂閱工作區中直接完成購買和狀態重新整理。

## 解鎖特性

購買 Turbo Engine 後，以下能力會被永久解鎖：

| 特性 | 說明 |
| --- | --- |
| **提案並發上限 3 → 32** | 生成、執行、歸檔中的提案共享此上限。32 並發讓你得以穩定並行推進多條任務線。 |
| **擴充文件主題** | 在冒險團管理介面中切換文件主題，即時預覽不同主題下的術語與文案表現。 |
| **自訂 Git 提交署名** | 設定預設 `Co-Authored-By` 模式，或在單次 AI Compose Commit 中暫時切換署名。 |
| **品牌自訂** | 上傳頭部 Logo，設定應用標題、瀏覽器標題和品牌點擊跳轉連結。 |
| **自訂頭像上傳** | 為當前 Hero 上傳自訂頭像，或從共享頭像庫中複用到新建和批次建立流程。 |
| **DLC 頭像資源** | Turbo Engine 專屬頭像包接入 Hero Avatar Gallery 與批次建立流程，直接參與工作流選擇。 |

## 誰需要它

- 經常同時推進多個提案或任務的使用者。
- 預設 3 並發已經不夠用的重度使用者。
- 需要自訂 Git 署名、文件主題、頭像和品牌標識的使用者。
